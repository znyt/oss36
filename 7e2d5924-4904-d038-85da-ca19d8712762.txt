　　华山脚下，
<br/>　　“叶帅！各大军区的精锐，已经在百里外驻扎，神龙卫也全员出动，只能您一声令下了！”

<br/>　　叶尘一声令下，神龙卫众人纷纷出动，开始去寻找山上的游客。

<br/>　　“叶帅，山上的游客全都不见了！”
<br/>　　叶尘听到这个消息，顿时眉头微微一皱，
<br/>　　“我原本以为，这些神界中人，自持身份，应该不会对世俗凡人出手，看来是想多了，他们是想用这些人，来逼迫我进入大阵之中！”

<br/>　　“叶帅，你可不能以身犯险，大不了咱们只围不攻，再也核武震慑，我还就不信吓不走他们！”
<br/>　　叶尘深吸了一口气，缓缓摇头道：
<br/>　　“老易，你太小瞧这些神界高手了！核武对于他们，根本构不成太大威胁，倘若我不出现，他们必然会将这些人质杀死，然后从容离去，军队也根本阻挡不了！”
<br/>　　“什么！”
<br/>　　听到叶尘这话，易山河等人顿时纷纷脸色大变。
<br/>　　倘若真是如此的话，这些人对于华夏国来说，将会是一场前所未有的巨大危机！
<br/>　　“可是，既然明知道是陷阱，您怎么能……”
<br/>　　易山河还想再劝，叶尘摆了摆手道：
<br/>　　“我既然是一国元帅，自然就有责任守护这一方土地！更何况，就凭他们这点伎俩，也未必杀得了我！”
<br/>　　说完这话，叶尘转而望了一旁萧若曦等人一眼，
<br/>　　“你们保护好我的朋友，我去会会他们！”
<br/>　　嗖！
<br/>　　声音落下的同时，众人还没有反应过来，叶尘的身影已经在原地消失不见。
<br/>　　……
<br/>　　另外一边，华山山巅之上，
<br/>　　“黛丽丝，你说过要替我报仇的，你们不去抓叶尘，把我们抓来做什么？”
<br/>　　整个华山上的游客已经全部被神界之人擒获，而那陶婉清自然也在其中，此时已经完全懵了。
<br/>　　“咯咯咯~~”
<br/>　　妖艳白人美女黛丽丝，顿时咯咯一阵娇笑，语气中透着冰冷道：
<br/>　　“小美人，你还好意思来质问我？我让你办的事情，你可曾做到？甚至我都怀疑，是不是你向那小子泄了密，否则他岂会发觉，我们精心设计的陷阱？”
<br/>　　“不！我没有！我恨不得要将他千刀万剐，又怎么可能会帮他！”
<br/>　　陶婉清立即否认。
<br/>　　黛丽丝两眼微微一转，咯咯笑道：
<br/>　　“好，本座姑且相信你说的都是真的，那我就再给你一次机会！”
<br/>　　陶婉清眉头一皱，
<br/>　　“你又想让我做什么？”
<br/>　　黛丽丝笑道：
<br/>　　“很简单，我想让你去找姓叶的小子，带我传个话过去，半个小时之内，倘若他再不来，我们便杀光这里的所有人！”
<br/>　　“什么！”
<br/>　　数百名游客，顿时全都纷纷脸色大变，
<br/>　　“你们凭什么这么做？”
<br/>　　“你们这是犯法的！”
<br/>　　“我不信你们还能把我们全都杀了！”
<br/>　　……
<br/>　　人群中炸开了锅，一些人已经开始蠢蠢欲动，准备反抗逃跑。

<br/>　　黛丽丝娇喝一声，同时随手一掌拍出，
<br/>　　彭！
<br/>　　远处一块高达近两米的硕大巨石，被黛丽丝这轻飘飘的一掌，瞬间拍的粉碎。

<br/>　　众人大多都是普通人，纵然如今世界已经进入了武修时代，可是他们何曾见过如此强悍的手段，顿时一个个纷纷倒吸一口凉气，吓得连连倒退，不敢再多说什么了。
<br/>　　见众人已经被震慑住，黛丽丝这才又恢复了仿佛的妖媚神态，伸手勾起陶婉清的下巴，咯咯娇笑道：
<br/>　　“小美人，如果你还想为你的情人报仇的话，就乖乖按照我说的去做，不管你用什么办法，只要你能把他引到这里，本座便饶了你和这些蝼蚁的性命，如何？”
<br/>　　陶婉清的脸色，顿时变得越发难看，
<br/>　　她又不傻，既然叶尘已经识破了她阴谋，又岂会再前来送死，自己此时再去找他，只怕非得被对方一掌拍死不可！
<br/>　　不过，面对黛丽丝的威胁，陶婉清根本别无选择。
<br/>　　就在她准备勉强答应下来的时候，忽然一道淡漠的声音，陡然在九天之上响起，
<br/>　　“不必这么麻烦，我来了！”

<br/>　　神界众高手听到这道声音，全都纷纷一震，所有人全都循着声音望去，

<br/>　　陶婉清望着空中那道身影，心情顿时五味杂陈，心里生出一种说不上来的感觉。
<br/>　　“哈哈哈哈！”

<br/>　　“好好！叶狂仙，看来传言不假，你倒是个重情重义之人！”

<br/>　　“你想多了！一群蝼蚁的死活，我叶狂仙又岂会在乎？你身为神界之主，用一群蝼蚁的性命来威胁我，未免有些上不了台面！”

<br/>　　叶尘淡淡瞥了一眼陶婉清，冷冷开口道。

<br/>　　“既然你们要战，那便战吧！”
