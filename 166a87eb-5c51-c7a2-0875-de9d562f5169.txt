　　听闻吕罡之言，全宗主微微颌首，旋即望向琥圣，道：“琥圣，苏莫之所以会杀你天虚宗的两名真传弟子，完全是那两名真传弟子咎由自取，怨不得苏莫！”
<br/>　　“哼，全宗主，你这分明是在袒护苏莫！”琥圣冷哼一声道，其实他也没指望全宗主会将苏莫交出来。
<br/>　　毕竟，苏莫这种妖孽天才，任何一个宗门都会袒护。
<br/>　　但是，他必须要将天虚宗的态度摆出来。
<br/>　　“琥圣，本宗主恩怨分明，若是苏莫错了，自然会受到应有的惩罚，但他并没有什么过错，这怎么能算是袒护？”全宗主摇头说道，无论苏莫对与错，道宗都会袒护苏莫。
<br/>　　“呵呵，你道宗弟子的一面之词，就是真理了？”
<br/>　　琥圣闻言不屑，他也懒得再与全宗主辩论对与错，冷笑着道：“全宗主，丑话说在前头，无论如何，苏莫杀我天虚宗两名真传弟子，我们绝对不会善罢甘休！”
<br/>　　“哦？琥圣你这是在威胁本宗吗？”全宗主眼眸微眯。
<br/>　　“谈不上威胁，只是苏莫犯下的罪孽，必须要付出代价！”琥圣面色淡淡的说道。
<br/>　　全宗主微微皱眉，若是天虚宗一心要杀苏莫的话，他们道宗的确是无法庇护。
<br/>　　毕竟，苏莫不可能一直待在问道山上，而一旦离开问道山，天虚宗派人暗杀的话，那就是防不胜防了。
<br/>　　略一沉吟，全宗主提醒道：“琥圣，你不要忘了，苏莫在会武中夺得了第一，被太阴族看重，普阳神长老更是亲自加以指点，你们若是对苏莫出手，太阴族那里也无法交代吧！”
<br/>　　琥圣闻言皱眉，这也是他比较担心的地方，所以他才会来到问道山，想要坐实了苏莫的罪名，到时候把苏莫击杀，太阴族那边也无法问责。
<br/>　　只不过，道宗袒护苏莫，想要坐实苏莫的罪名，却也不容易。
<br/>　　“全宗主，我天虚宗惩治十恶不赦的罪人，太阴族也无法说什么！”
<br/>　　琥圣冷笑一声，随即长身而起，道：“既然全宗主处事不公，偏袒门下弟子，那我们也没有什么好谈的了，告辞了！”
<br/>　　言罢，琥圣一挥袖袍，大步走出了大殿。
<br/>　　其他天虚宗的强者见此，纷纷长身而起，离开了大殿。
<br/>　　扬天成冷笑一声，此事天虚宗一定不会善罢甘休，苏莫必须要死。
<br/>　　见琥圣等天虚宗之人离去，全宗主无奈的摇了摇头，这个苏莫真是会惹事。
<br/>　　不过，此等小事他也没有时间多管，吩咐下面的人处理便可。
<br/>　　琥圣等人，离开问道山之后，便向天虚宗返回而去。
<br/>　　天空中，扬天成飞在琥圣的身边，问道：“圣长老，我们接下来怎么办，那苏莫罪不可赦，绝对不能放过！”
<br/>　　“这是自然，公然残杀我宗两名真传弟子，若是饶过他，我天虚宗脸面何存！”琥圣淡漠的说道。
<br/>　　“那我们接下来怎么办？”扬天成问道。
<br/>　　“先查出苏莫在何地，然后再做定夺！”琥圣沉声道。
<br/>　　“嗯！”扬天成点了点头，先找到苏莫在哪里，到时候擒拿了对方之后，再做处理。
<br/>　　……
<br/>　　九重浮屠塔，顶层的空间之内，苏莫完全沉浸在参悟之中，并不知道天虚宗之人已经踏上了问道山。
<br/>　　随着不断的参悟，他对于五行之力，了解的越发精深。
<br/>　　目前，他的属性融合，可以做到土金融合，木火融合，然后配合上水属性玄力，变成了三色的玄力。
<br/>　　而他的下一步，是将水属性和土金属性融合，或者是与木火属性融合。
<br/>　　随着不断的参悟，他发现，若想将水属性融合进土金属性之中，难度非常的大。
<br/>　　而将水属性融合进木火属性之中，难度则是相对较小几分。
<br/>　　因为土生金，金生水，而水与土既不相生也不相克，很难保证力量的平衡。
<br/>　　但是，水生木、木生火，而水火却是相克，木属性能中和两者的力量，让两者保持在一种微妙的平衡之中。
<br/>　　这或许就是所谓的生中有制，制中有生。
<br/>　　所以，在浮屠塔之中，苏莫一直在参悟水、木、火这三种属性，寻找其中的玄妙。
<br/>　　整整一个月的时间，他一直在参悟这三种属性。
<br/>　　半个月之后，他已经有了很深的感悟，便离开了九重浮屠塔，住进了太阴城之中属于道宗的府邸中。
<br/>　　玉镯空间之内，苏莫盘膝而坐，面带凝重之色。
<br/>　　“平衡！”苏莫低声自喃，他只要能做到水木火三种属性的平衡，便能融合成功。
<br/>　　但是，想做到这一点，却是非常的困难，需要不断的尝试与摸索。
<br/>　　少倾之后，苏莫抬起手掌，其中三根手指之上，分别缭绕着一缕发丝般的玄力。
<br/>　　这三缕玄力，分别为蓝色、青色和火红色，正是水属性玄力、木属性玄力和火属性玄力。
<br/>　　现在，苏莫首先要将水属性和木属性融合在一起，然后再融合火属性玄力。
<br/>　　因为他的火属性玄力和木属性玄力，早已能做到随意的融合，但水属性和木属性还无法融合。
<br/>　　等到水属性和木属性能随意融合之时，再融合进火属性，就比较容易了。
<br/>　　这一切，看似简单无比，实则非常的复杂。
<br/>　　可以这么说，苏莫完全是在摸索着修炼，没有任何前车之鉴，只能靠自己一步一步摸索着前进。
<br/>　　没办法，他选择的是一条独特的修炼之路，想要更加强大，就要不断的摸索。
<br/>　　少倾之后，苏莫手指微动，开始试着融合水木属性。
<br/>　　嘭！
<br/>　　水属性玄力和木属性玄力挤压在一起，没过一息时间，水属性玄力便崩溃开来。
<br/>　　这是因为水属性玄力较弱，不及木属性玄力强大。
<br/>　　苏莫无奈，只能先行修炼水属性功法，壮大水属性玄力。
<br/>　　随后，他便运转了《水仙录》，开始修炼水属性功法。
<br/>　　这一修炼，便是十余日的时间，期间他数次尝试融合，试探两种属性玄力的威力。
<br/>　　经过十余日的努力，他终于让两种属性玄力的威力，保持在了相等的地步，接下来就是融合了。
<br/>　　“马上就到一个月了！”
<br/>　　苏莫叹了口气，在太阴族只能待一个月时间，不然就要额外支付灵晶。
