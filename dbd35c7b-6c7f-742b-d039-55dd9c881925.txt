　　第254章
<br/>　　程大雷看着她，无奈的摇摇头，这个血统天赋还真是强，正所谓娘蠢蠢一个，爹蠢蠢一窝，有这样的父亲，不开窍也容易理解。
<br/>　　程大雷有些不想理她，开始研究脑海中的系统。现在还有一次抽奖机会，如今不知多少人要杀自己，多些东西就多些活下去的希望，所以程大雷立刻选择抽奖。
<br/>　　脑海中的扭蛋机出现，骨碌碌旋转，一物掉出来，程大雷也收到了系统提示。
<br/>　　嘟，获得战曲一支，热血技能双倍增益，体力、敏捷增益50%，恢复能力增益45%，持续时间一炷香。
<br/>　　呃……
<br/>　　程大雷有些无语，这战曲怎么用，难道以后要一边唱歌，一边和人打仗。
<br/>　　『妹妹你做船头喔』
<br/>　　“呔，狗贼吃我一斧！”
<br/>　　『大河向东流』
<br/>　　“你虽然本事不错，但我这直来直往剑你可未必认得。”
<br/>　　想起这一幕，程大雷就觉得有些魔性，他忙摇摇头，把画面驱散。
<br/>　　李婉儿看着程大雷坐在那里，一会笑得像个白痴，一会像吃了屎一样满脸便秘。
<br/>　　她摇摇头，道：“你能否送我到戎族王城，或者，帮我找到鱼龙卫。”
<br/>　　“你的那些护卫不行的，我已得到消息，现在不知多少山贼要劫你，他们人很多，也很强。”
<br/>　　“那你送我！”李婉儿开口有种不容置疑的意味。
<br/>　　“这不是扯么，我是来劫你的，不是护送你的。”
<br/>　　程大雷信口说着，忽然察觉到李婉儿看自己的眼神有些不对了，双目灼灼，令人不得不认真面对。
<br/>　　程大雷叹口气：“两国之争，实在轮不到一个女人去牺牲。一个需要女人牺牲才能保护的国家又算怎样的国家，而这样换来的和平又能持续多久？”
<br/>　　李婉儿没有被说服，她裹着锦袍，屈膝坐在地上，目光看着很远的地方。
<br/>　　“我是帝国公主，我有牺牲自己的责任，这是我的使命，我不能逃避。”
<br/>　　程大雷听到『使命』这个词就有些头疼，他捂着脑袋道：“你一定要去？”
<br/>　　“一定要去。”
<br/>　　“非去不可？”
<br/>　　“非去不可。”
<br/>　　目光坚定，如同燃烧着火焰，有股不容抗拒的味道。
<br/>　　“如果我非不让你去呢！”
<br/>　　程大雷也有些火气，身体前倾，如一座将要倾倒的山，紧紧盯着对方。
<br/>　　“我可是个山贼，难道你不怕我抢你回去做压寨夫人？”
<br/>　　“我会死给你看。”
<br/>　　李婉儿向前挺了挺身子，更靠近程大雷的脸，两只眼睛睁得很大。
<br/>　　双方僵持下来，紧紧注视着对方，如两支对峙的小兽。
<br/>　　“离这么近，一般……都是要接吻了……”半晌，程大雷口中吐出一句话。
<br/>　　李婉儿瞬间脸一红，正当程大雷以为她要退缩的时候，她突然更靠近一些，幽香呵在程大雷脸上。
<br/>　　程大雷败下阵来。
<br/>　　哎呀呀，这个女人怎么这么流·氓，俺老程的贞洁牌坊差点没保住。
<br/>　　嘟，触发任务，护送明玉公主抵达戎族王城，本任务难度极高，可拒绝。
<br/>　　嘟，触发隐藏任务抢夺压寨夫人，目标：明玉公主。因目标二身份特殊，任务难度提升，奖励提升，可拒绝。
<br/>　　咦，程大雷收回要和李婉儿斗气的心理，开始研究自己刚收到的两条任务提示。
<br/>　　又是二选一的任务，选择完成其中一个，第二个会自动消失。
<br/>　　关键在于抢夺压寨夫人的任务，程大雷在苏樱那里的任务线还没走完呢，一直也不知道触发下一环节的条件是什么。怎么又在李婉儿这里触发一条任务线。
<br/>　　或许，自己走入一个思维死角中，好像没谁规定，山贼的压寨夫人只能有一个吧。
<br/>　　任务一与任务二究竟该怎么选？
<br/>　　答案呼之欲出，可面前的李婉儿显然已经认同明帝的理念，决定牺牲一个人，幸福千千万。
<br/>　　半晌，程大雷伸出手揉揉她的脑顶，道：
<br/>　　“好吧，既然你想去看看，我便带你去看看，不就是戎族王城么，我们去。”
<br/>　　“你肯答应了。”
<br/>　　终于得到程大雷松口，可李婉儿却没有想象中的开心。类似于一支敢死队要炸敌方总部，将军说：不要去。敢死队：便要去。将军说：那就去吧……
<br/>　　没有理由，敢死队要欢呼雀跃。
<br/>　　“既然决定要去，就要想好接下来的路如何走，这一路怕是不会太平。”程大雷想了想道：“你先把衣服脱了吧。”
<br/>　　“你想干嘛？”李婉儿吓了一跳。
<br/>　　“现在不知多少人找我们，你穿成这样太招眼，虽然我dps很强，但你也不用往死里给我拉仇恨吧。”
<br/>　　李婉儿虽然听不懂迟到了的话，但大概意思是能明白的。她穿着帝国绣匠精心制作的红袍，上面环佩叮当，有各种金银玉珠，还是蛮重的，也太扎眼。
<br/>　　如今也没有衣服换，李婉儿把衣服上的金银珠宝摘下，整个人顿时灵便许多。程大雷把这些东西收起来，接下来的路上是可以当钱使的。
<br/>　　正此时，忽有说话声，二人顿时一惊，只见草坡的另一侧，有四个马贼正往这边过来。
<br/>　　“等我。”
<br/>　　程大雷道了一声，身体从黑暗中消失。草坡下就只剩下李婉儿一个人，她被抛在黑暗中，四周空寂寂，有虫鸣响动，她心里狠狠打了个寒颤。
<br/>　　好黑，好怕。
<br/>　　不知他能不能敌过对方，毕竟他只有一个人，又经过一场乱战，此时体力尚未恢复。
<br/>　　空气中一些凉意，忽觉得很冷，抱住肩膀，一动不敢动，怕程大雷回来后找不到自己。
<br/>　　有脚步声响动，李婉儿心中一激灵，不敢发生，只是看着黑暗处。
<br/>　　是程大雷回来了，他怀里还抱着东西。
<br/>　　“人杀了，想抢一匹马的，又怕你等得害怕。从他们身上搜了些肉干和酒，你肯定饿了，先吃些东西。”
<br/>　　来去之间，杀人如割草，匹夫剑仍在滴血。
<br/>　　李婉儿下意识抓住他衣角，眼睛紧紧盯着对方，怕他会突然消失。
<br/>　　“哈，吓住了。”
<br/>　　程大雷抓了一把草把剑上的血擦干，收剑入鞘，咕咚咚灌了一气马奶酒。
<br/>　　“别怕，有我呢。”
<br/>　　
<br/>(本章完)
