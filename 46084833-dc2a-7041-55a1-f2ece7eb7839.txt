　　一夜无话。
<br/>　　第二天早上六点多的时候，刘哔就被一阵敲门声叫起，眉头一皱，他退出了修炼状态。
<br/>　　打开房间的大门，旧二多福站在外面，两只眼睛完成了月牙形状，看起来似乎很开心的样子。
<br/>　　“刘君，今天可是有很重要的会议呢，七点之前，我们必须要赶到会议厅呢。”抬起了手，多福打了一声招呼，露出了雪白的手腕。
<br/>　　像是女孩子一般，纤细，细腻，很难想象这手腕的主人是一个男人。
<br/>　　刘哔点了点头，随手从门口的衣架上取下来衣服。
<br/>　　一套黑色的正式西装，这是白鸽常见的打扮，木嶋班的规定十分严格，上班的时候必须要穿戴一致的正装。
<br/>　　门外另外两名小组成员，坐在餐桌上吃着早饭，早餐是很简单的牛奶面包，桌子中央还放着一瓶被打开的沙拉酱。
<br/>　　看到早饭，刘哔的目光波动了几下，却也没说什么，抬脚走进了卫生间。
<br/>　　看着面前摆放在一起的几套牙具，他不知道哪个属于自己，无奈之下，只能随便洗了把脸，用手接了一捧水来漱口。
<br/>　　洗漱完毕，刘哔擦了擦手，走出了卫生间，旧二多福见他出来，招呼了一声。
<br/>　　虽然心中有些抗拒，但他还是走到了餐桌前面。
<br/>　　和上个世界不同，在这里如果他不吃人类食物的话，难免会引起一些怀疑。
<br/>　　面包一如既往的难吃，杯子中的牛奶都像是难喝的石灰水一般，刘哔强忍着吃了一些，脸上没露出任何表情。
<br/>　　胃里开始不断翻腾了起来，他感觉自己的肠胃都要纠缠到了一起，吃进胃中的食物无法消化，在体内有一种很鲜明的别扭感，让他有些难受。
<br/>　　得找个时间把胃部清理一下，但现在明显没有什么机会。
<br/>　　低垂着眼睛，刘哔艰难熬过了这漫长的早餐时间。
<br/>　　等到大家都用餐完毕，那位名叫松下立智的男人站起身来，带头向门外走去。
<br/>　　门口有两个银色的密码箱，那是搜查官盛放库克因的容具，但刘哔和旧二多福，都没有自己的库克因。
<br/>　　不过那种东西，也没有什么值得稀罕的。
<br/>　　门外的天色刚刚亮起，大部分的人现在也就刚刚起床，街上的行人并不算太多。
<br/>　　如果没有喰种，这个国家也还是十分安定的，几乎所有的上班族，都在过着朝九晚五的生活，现在这个时间赶去上班的，职业大都有些特殊。
<br/>　　CCG就属于特殊职业的一种，拿着政府发放的高额薪资，他们的付出自然要比一些其他的职业多出许多。
<br/>　　走了一段路程，一行四人先后来到了自家小组的办公室，木嶋式还没来，众人一边在办公室内等候着，一边做着一些自己的事情。
<br/>　　坐在自己办公桌上的刘哔，有些无聊的看着窗外，其他几人都提前入座了，最上方那个超大的办公桌自然是木嶋班长的，故此，他很轻易就找到了自己的桌子。
<br/>　　“刘，你不准备写报告吗？”同为新人，旧二多福也没有事情干，靠在自己位置的椅背上，他好奇的看了一眼刘哔。
<br/>　　之前这家伙还说，去调查一个很机密的事情，既然这么重要，他都不打算写一份报告吗？
<br/>　　当然，他之所以问出这句话，也只是因为无聊罢了。
<br/>　　看了一眼多福，刘哔笑了一声，正欲回答，办公室的推拉门却被人给打开了。
<br/>　　一个头戴圆边帽子的怪异家伙走了进来，之所以说怪异，是因为此人的面相实在是太过于狰狞。
<br/>　　面部的皮肤像是被缝补起来的一般，打着补丁，耳朵、鼻子已被削平，右腿的动作十分僵硬，大概是假肢，身材很是臃肿，被一个大号的风衣掩盖着。
<br/>　　与其说他是人类，倒更像是不知道从哪里冒出来的怪物，单凭这份相貌，就很难让人心中升起好感。
<br/>　　整个CCG有这幅造型的，也就独此一人，木嶋式这个名字，也让众多喰种恨得咬牙切齿。
<br/>　　“小伙子们，大家好啊，哟，刘君竟然回来了。”嘴角咧开一个狰狞无比的笑容，木嶋的声音有些沙哑，如同金属摩擦的那种声音一般，很是难听。
<br/>　　众人都从自己的办公桌上战起，来到木嶋式面前站成一排，脸上的笑容没有任何掩饰。
<br/>　　虽然班级中老人和新人的关系并不和睦，但大家对于木嶋式却是由衷的敬爱，他们都知道，在这一副让人心中生惧的皮囊下面，可是隐藏着一颗和善真诚的内心。
<br/>　　至于对待喰种的手段残忍一些，所有人都不觉得这是个问题，反正，那些怪物不就是应该杀戮的吗。
<br/>　　跟随着其他人的动作，刘哔也动了起来，站在旧二多福旁边，他突然听到木嶋式对他发问。
<br/>　　稍稍愣了一下，刘哔点了点头：“是的，我发现了一些东西，这件事情我需要单独向木嶋班长进行汇报。”
<br/>　　“恩，等开完会说吧，会议马上就要开始了。”闻言，木嶋式抬起了手腕，看了一眼时间，回答道。
<br/>　　目光微微闪烁了一下，他也没有继续纠缠，反正在CCG攻打青铜树之前，他能将这个消息传播出去就可以了。
<br/>　　“还有什么问题吗，如果没有的话，那就一起去会议室吧。”环视了一眼四周，木嶋看到大家都没说话，便挥了挥手，带头走向了会议室。
<br/>　　和一些小型的会议不同，这次大会似乎十分重要，几乎随处都能见到赶往会议室的搜查官，密密麻麻的人群，让过道都显得有些拥堵了。
<br/>　　有人带路，刘哔自然乐得清闲，跟随着自己的班长，很快走进了一间庞大的会议大厅。
<br/>　　大厅中，已经坐下了不少人群，木嶋式带领着众人，来到自己的位置坐下，简约的白色会议桌上，上面摆着一个牌子，上面标注着几人的姓名。
<br/>　　片刻，手腕上的时间定格在七点整，会议大厅的最前方，一位身材干瘦的搜查官拿过麦克风，轻轻咳嗽了一声。
<br/>　　场中嘈杂的气氛顿时安静了下来，大家都知道会议即将开始了。
<br/>(本章完)
