　　第539章
<br/>　　苏离拧起眉头，盯着刘发财：“你笑什么？”
<br/>　　“呃，我笑了么？”刘发财。
<br/>　　“你脸上没笑，但眼里已经笑了。眼里没笑，心里一定笑了。”苏离道：“你定是嘲笑我夫妻三人，好好好，我曾发誓，谁敢嘲笑我们，我必杀他。你既然敢笑定然不怕死了。来来来，到我剑下受死。”
<br/>　　刘发财眼睛睁得铜铃一般大：“我他妈没有笑！”
<br/>　　“哼，男子汉大丈夫，敢做就要敢当。你做了不敢当，算什么男人。你定是不愿做男人了，来来来，吃我一剑去死吧。”剑恨南。
<br/>　　刘发财用哀求的眼神看着程大雷。
<br/>　　程大雷拍拍他的肩膀：“唉，你这个人做了就要承认么，我也看见你笑了。”
<br/>　　“……”刘发财。
<br/>　　李越英双手抱拳：“程当家是个公道人。”
<br/>　　“认识我的人都这么说。”程大雷道：“伦理道德无非是圣人定下，拘禁人的枷锁。只有超脱枷锁，方为自在人，三位都是自在潇洒人。”
<br/>　　苏离三人同时抱拳，重重向程大雷行了一礼。
<br/>　　“若天下都是程当家这般的了人，我们不知少受多少苦楚。”
<br/>　　刘发财无语的看着这一幕，道：“既然你们这么有得聊，就干脆找个地方好好聊，我还有事，就先告辞了。”
<br/>　　“呔，站住！”
<br/>　　三人成三角形，将程大雷与刘发财包围在其中。
<br/>　　“东海苏离、剑恨南、李越英向程当家见礼。”三人同时道：“在这里恭候多时，借程大雷脑袋一用。”
<br/>　　刘发财松了口气，这才是正常的台词么。他也握住腰上的刀，看自己有没有机会出手。
<br/>　　他出手自然不是为了帮程大雷，而是看看有没有机会捅程大雷一刀。当然，杀不死程大雷，趁机脱身也是可以的。
<br/>　　程大雷轻轻咳嗽了一声，苏离三人瞬间如临大敌。毕竟，面对这魔头，三人不得不谨慎。
<br/>　　程大雷摆摆手：“不着急，我想先向三位打听一件事？”
<br/>　　苏离三人擎着剑，不断变幻方位，寻找最佳出手的机会。
<br/>　　“程当家请讲？”
<br/>　　“我想问三位，东海一地，可有一位姓华的医生，应该很有名气？”
<br/>　　三人同时摇摇头：“从未听说过，程当家问这件事做什么？”
<br/>　　“唉，既然没听说过那就算了吧，走好不送。”程大雷叹了口气显得颇为遗憾。
<br/>　　“什么意思？”苏离皱起眉头，没听清程大雷后面的话。
<br/>　　“黄泉路远，你们三个一起上路当不孤单。走好不送的意思是，我送你们下黄泉。”
<br/>　　声音落地，程大雷的身子从牛背上突然腾起。匹夫剑在空中出鞘，在空中一折，一剑向苏离斩去。
<br/>　　苏离急退，一退再退，程大雷出手的第一剑就给她带来巨大的压力。
<br/>　　剑恨南与李越英三人扑过来，攻击程大雷的后背，程大雷回身扫了一剑，铛铛声响，将两剑逼开。
<br/>　　苏离的剑已向他后脑扎来，而程大雷就像背后长着眼睛一般，一记铁板桥，险之又险的避开这一剑。
<br/>　　回剑在身周一荡，将三人逼退。
<br/>　　刘发财暗暗为程大雷叫好，盛名之下无虚士，程大雷这一身本事当然非同凡响。当然杀人夫妻也不是弱手，双方这一战，必定精彩纷呈。
<br/>　　刘发财很是想为程大雷鼓掌叫好，但他很明显没有为武道牺牲的觉悟。如今这光景，自己还是逃命要紧。
<br/>　　拨转牛头，撒丫子狂奔。
<br/>　　程大雷陷入苏离三人的包围中，一时间却也不好脱身。他看到这一幕，口中一声呼哨。那头墨玉黑月牛哞地一声，狂奔而来，直接将刘发财的座骑抵到在地。
<br/>　　刘发财在地上打了两个滚，这才稳住身子。抬起头看到黑牛盯着自己，鼻孔中呼哧呼哧喘气。
<br/>　　刘发财心底一激灵，这黑牛哪里是看着有些怕，它确实挺可怕。
<br/>　　程大雷一剑抖开，剑光粼粼。三才剑阵的确非同凡响，三人占了天地人位置，一人攻击，两人防御，或一人防御，两人攻击。三人合力，却像是一个人生了三头六臂，瞬间便给程大雷带来巨大压力。
<br/>　　程大雷来到东海的消息已散出去，想杀他的人是多的，各路诸侯都派出高手。而江湖上的草莽英雄也以杀了程大雷为荣，毕竟杀死程大雷不用有什么心理负担。
<br/>　　程大雷呼喝一声，一剑斩出，斩的是中央的苏离。剑恨南和李越英两柄剑同时回挡，而苏离的剑仍义无反顾的往前刺。
<br/>　　剑光忽地一变，一剑变三剑，阿飞快剑的第二式爆烟花。虽然只有一剑，刺的却是三人。苏离三人不得不回剑格挡，护住咽喉要害。
<br/>　　谁料剑光又是一变，绽开的剑光束成一缕，速度陡然暴增，刺向苏离喉咙。
<br/>　　苏离挡不住这一剑，脸露惊慌之色。剑恨南脸上一惊，在不可能发力的状况，生生扑在苏离身前。
<br/>　　本该杀掉苏离的一剑，洞穿了剑恨南的胸口。
<br/>　　“大师兄！”苏离绝眦欲裂，眼中爆出怒火。
<br/>　　程大雷也感慨剑恨南对苏离的感情，但现在明显不是发出感慨的时候，还是杀人要紧。他的剑身一抖，攻向苏离和李越英。
<br/>　　剑恨南一死，三才剑阵便被破，苏离和李越英便无法对程大雷形成威胁。完全是被程大雷压着大，程大雷占尽上风。
<br/>　　“小师妹，走啊！”
<br/>　　李越英吼了一声，扑向程大雷，这完全是玩命的打法，用自己的性命替苏离逃跑争取时间。面对这一幕，程大雷也不得不暂时收剑防御。
<br/>　　杀人者人恒杀之，想要程大雷的命，就要有被程大雷杀死的觉悟。很明显，到现在李越英才有这觉悟。
<br/>　　拼命猛攻，勇则勇矣，但却不能长久。程大雷避开他的锋芒后，轻轻一剑取了他的性命。
<br/>　　“放下剑！”
<br/>　　程大雷一怔，抬起头看到苏离制住了刘发财，咬牙切齿的盯着自己。
<br/>　　“不想他死的话，就跪着爬过来。”
<br/>　　刘发财叫苦不迭，道：“大姐，你搞错状况了，我和他不熟的。”
<br/>　　“住口！”
<br/>　　苏离用剑柄在刘发财嘴上砸了一下，砸得他满口流血。
<br/>　　“你想让他死么！”苏离恶狠狠的盯着程大雷。
<br/>(本章完)
