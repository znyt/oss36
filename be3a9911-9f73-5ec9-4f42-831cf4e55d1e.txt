        傅昼景压抑着心里的怒火，搂着童筱颖进了房门。
<br/>
<br/>        筱颖来之前，他就已经跟季思妍说好，让季思妍去别的酒店下榻。
<br/>
<br/>        他完全没料到，季思妍不听他使唤，擅作主张地订下他隔壁的房间是何用意？
<br/>
<br/>        “赵斌没有来吗？”进屋后，童筱颖不禁好奇地问。
<br/>
<br/>        傅昼景怔了一下，才回答道：“公司里还有其他事情需要他去做，所以这次我出差，他没来。我只带了一个秘书，和两个秘书助理过来给我帮忙。”
<br/>
<br/>        其实，他现在和季思妍所谈的业务，都是他的私活。
<br/>
<br/>        赵斌是傅氏集团的人，哪怕一直跟在他身边辅佐他，但赵斌终究跟他不是一伙的。
<br/>
<br/>        傅昼景虽然一直把赵斌当兄弟，但是赵斌效忠于谁，他心里一清二楚。
<br/>
<br/>        他和季思妍在外自己开了一家公司的事情，赵斌并不知情。
<br/>
<br/>        所以，傅昼景这次出差是公事和私事同时进行。
<br/>
<br/>        进屋后，童筱颖看到套房客厅里的茶几上摊满了资料，那边浴室门口又有一箩筐的衣服未洗，于是主动走过去给傅昼景先洗起了衣服来。
<br/>
<br/>        不得不说，童筱颖对于他傅昼景来说，确实是一个居家型的小女人。
<br/>
<br/>        他若是主外，她可以在内把家中打理的井井有条。
<br/>
<br/>        而这些家务活，作为千金大小姐的季思妍来说是做不来的。
<br/>
<br/>        童筱颖洗好衣服后，将衣服拿去了阳台上晾起。她刚晾完衣服，还未转身，身后就传来一股温厚的气息，下一个瞬间，她便被这个温暖的怀抱给包围。
<br/>
<br/>        傅昼景从后搂住童筱颖，轻轻地将她圈入自己怀中，双手中随之托起一个大大的粉色方形礼盒。
<br/>
<br/>        童筱颖看得惊喜不已，忍不住好奇地问：“这是什么？”
<br/>
<br/>        “你打开看看。”傅昼景温柔地说道。
<br/>
<br/>        童筱颖点了点头，抬起手来拆开了礼盒，将礼盒盖挪开。
<br/>
<br/>        里面躺着一件精致的粉色连衣裙，而且还是国外的牌子，看起来很贵。
<br/>
<br/>        傅昼景将里面的连衣裙拿了出来，微微俯身，附在童筱颖的耳边，轻轻地说道：“穿给我看。”
<br/>
<br/>        他温热的气息喷洒在她的耳垂边，有点儿痒，又有点儿令她怦然心动。
<br/>
<br/>        童筱颖两颊红润，点了下头，接过他手中的连衣裙，刚准备从他怀中走出，却被他大手一揽，带动着她的腰肢转了个身，面对着他。
<br/>
<br/>        “我帮你换……”傅昼景轻言轻语，放下手里的礼盒，牵着童筱颖的手，拉着她进了屋。
<br/>
<br/>        童筱颖看着傅昼景的俊脸，心跳加速，只觉自己浑身的血液都在沸腾，整个脸由原本的红润，瞬间涨得满脸通红。
<br/>
<br/>        傅昼景会心一笑，抬起手来，伸向了童筱颖的领口。
<br/>
<br/>        他的指尖，只不过不小心地触碰到她的锁骨。
<br/>
<br/>        童筱颖羞怯地抬眸瞥了傅昼景一眼时，一个恍神间，脑海里却浮现出傅夜沉的那张俊脸来。
<br/>
<br/>        傅夜沉束缚着她的双手，捂住她的嘴不让她出声，还吻遍了她的全身。
<br/>
<br/>        那种感觉，没有让她感到酥麻，而是起了一身鸡皮疙瘩。
<br/>
<br/>        她骨子里不是那种“嘴上说着不要，身体却很诚实”的女人。
<br/>
<br/>        一旦她排斥过于亲密的举动，她就会全身心地反感抗拒。
<br/>
<br/>        “我自己来！”童筱颖条件反射般地后退了一步。
<br/>
<br/>        傅昼景见童筱颖躲开自己，还以为她是害羞，大手捧着她微烫的脸颊，忍俊不禁道：“筱颖，你是我的女人。”
<br/>
<br/>        童筱颖怔了怔，抬眸对上了傅昼景的目光。
<br/>
<br/>        她不看他还好，一看他，四目相视，两人之间的气氛慢慢地变得暧昧起来。
<br/>
<br/>        他温柔的目光，投入她的眼底，暖意融融，如沐春风。
<br/>
<br/>        傅昼景微微低下头去，缓缓地朝童筱颖的双唇凑了过去。
<br/>
<br/>        童筱颖心里抛去所有杂念后，下意识地闭上了眼睛，期待着他下一刻温柔的吻。
<br/>
<br/>        就在两颗心彼此靠近的时候，门铃不逢其时地响了起来。
<br/>
<br/>        傅昼景随即吻上童筱颖的额头，温柔地说道：“我去开门，你先去换衣服。”
<br/>
<br/>        童筱颖倏然睁开双眼，红着脸，聋拉着脑袋，抱着连衣裙去了卧室。
<br/>
<br/>        傅昼景看着童筱颖走进卧室并将卧室的房门给关上后，笑容渐敛，微微蹙着眉头走过去开了门。
<br/>
<br/>        他连猫眼都不用看，就知道来叩门的人是谁。
<br/>
<br/>        果不其然，是季思妍。
<br/>
<br/>        季思妍手里提着一大袋子的水果，和一大袋子的甜品以及饮料，未经傅昼景的同意，就直接闯入了门内。
<br/>
<br/>        她笑盈盈地说：“傅总，我买了许多吃的，想和冷夫人一起分享。”
