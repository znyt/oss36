<br/>  天罗驻地，便是天罗部所驻扎的地方。<br/>　　<br/>　　  当初王康收服袁延，其带着整个绿林众山匪投靠，改名天罗，为王康手下，专行暗杀之事。<br/>　　<br/>　　  那时，王康便在新奉县寻了一个村庄，将其安顿。<br/>　　<br/>　　  这个村庄是在山间，位置隐蔽，暗中哨岗不知几何，前段时间，李清曼还专门来此，布了一些暗阵。<br/>　　<br/>　　  为何如此？<br/>　　<br/>　　  一来，这里是天罗部的大本营，比较特殊。<br/>　　<br/>　　  二来，王康的所有核心生产都是在这里进行，比如说火药，还有专门建造的武器作坊。<br/>　　<br/>　　  这个村庄现在经过经营，各种设施已经完备。<br/>　　<br/>　　  王康到这里，是因为之前他命袁延专门精训的一批人已经成型。<br/>　　<br/>　　  这批人是从所有天罗部中，通过比斗，精选，淘汰等机制，选拔而出。<br/>　　<br/>　　  又袁延亲自训练杀手技巧，又经过李清曼特训。<br/>　　<br/>　　  他们中武道最次的也是三流高手，精通暗杀，潜伏等……<br/>　　<br/>　　  并且，他们都装备着，由王康设计，用精钢制成的新式装备，可以说是完完全全的杀人机器。<br/>　　<br/>　　  每一个都是精锐中的精锐。<br/>　　<br/>　　  当然，这样的人不会多，也仅有九十二人。<br/>　　<br/>　　  不过现在只剩下了八十九人。<br/>　　<br/>　　  因为之前执行爆破任务时，就是从他们之中选出的三人，两死一伤。<br/>　　<br/>　　  袁延请王康来，就是让他检阅，并为这支队伍亲字命名。<br/>　　<br/>　　  王康到来时，袁延已经将他们集结在一片空地上。<br/>　　<br/>　　  “大首领！”<br/>　　<br/>　　  见到王康到来，这些人都是齐声行礼，单膝跪下，整齐划一。<br/>　　<br/>　　  王康并没有惊讶，因为在训练他们之时，就已经给他们灌输了统一的思想。<br/>　　<br/>　　  这是一种洗脑式的训练。<br/>　　<br/>　　  这也是王康想要的效果，成为他的死侍！<br/>　　<br/>　　  “起来吧！”王康沉声道。<br/>　　<br/>　　  随后所有人都起来，他们排成整齐的队列，皆是身穿黑衣，站在这里，自带一种气势。<br/>　　<br/>　　  给人一种强烈压迫感。<br/>　　<br/>　　  “康少爷，这些有的是我原有部下，有的是收拢的一些江湖游侠，”<br/>　　<br/>　　  在一旁袁延解释道：“他们中大部分是三流高手，二流高手也有几个。”<br/>　　<br/>　　  王康明白，这是对于习武之人的一个等级划分。<br/>　　<br/>　　  不入流，三流，二流，一流。<br/>　　<br/>　　  在这之上，便是宗师！<br/>　　<br/>　　  武道高绝，能够开宗立派，这样的人物每一个都是声名鹤鹤，任谁都得慎重对待。<br/>　　<br/>　　  而王康连不如流都算不上，因为他没有任何的武道基础。<br/>　　<br/>　　  如李清曼，李御遥便是一流之上，宗师以下，在这之间的武道高手。<br/>　　<br/>　　  能在这个年纪，达到如此程度，都是天资绝卓之人……<br/>　　<br/>　　  思绪一闪而逝，王康看着面前整齐划一的八十九人，心中也是很满意。<br/>　　<br/>　　  这时袁延又是道：“他们共有八十九人，共分九队，其中有九人成一队！”<br/>　　<br/>　　  “其中又设一名大队长。”<br/>　　<br/>　　  “哦？大队长是谁？”王康来了兴趣。<br/>　　<br/>　　  “匕附，出列！”袁延对着前方喊出了一个名字。<br/>　　<br/>　　  听到这个名字，王康微微一怔，这个名字倒是有些拗口。<br/>　　<br/>　　  随着袁延的声音落下，从黑衣队中，走出一人。<br/>　　<br/>　　  此人同样是身穿着黑衣，<br/>　　<br/>　　  但一眼就能发现他的不同，也不是说不同，而是这么多人中，他显得略微不同。<br/>　　<br/>　　  因为，他根本就不像一个杀手，或者说不像一个武者。<br/>　　<br/>　　  因为他太普通了，面相大众，身材普通，身高普通，这种人放到人群里，恐怕谁都不会注意一眼。<br/>　　<br/>　　  但就是如此，才让王康面色一凛，杀手，这才是真正的杀手！<br/>　　<br/>　　  不露痕迹，隐于大众。<br/>　　<br/>　　  在谁都不会注意下，给人致命一击。<br/>　　<br/>　　  而在他的身上也感觉不到任何习武之人的特质。<br/>　　<br/>　　  修习武道之人，或多或少都有一种特殊的气质，类似于一种气。<br/>　　<br/>　　  杀手，也是同样。<br/>　　<br/>　　  杀的人多了，身上自然而然便沾染这种气。<br/>　　<br/>　　  李清曼是太一教传人，是一流之上的高手，但她却在平常没有丝毫的气息显露，这就是一种返璞归真的境界。<br/>　　<br/>　　  而这个匕附，应该就是这般，因为袁延不可能，选一个普通人作为队长。<br/>　　<br/>　　  “属下匕附，见过大首领！”<br/>　　<br/>　　  在王康思绪间，此人来到他的面前。<br/>　　<br/>　　  “康少爷，您别看他这个样子，其实……”袁延怕王康误会，在一旁解释。<br/>　　<br/>　　  “不用多说，”王康摆了摆手道：“这才是真正的高手吧。”<br/>　　<br/>　　  袁延愕然，显然没想到王康竟然明白。<br/>　　<br/>　　  他又是介绍道：“匕附，此前跟我同属一个杀手组织，其本身武道就是二流高手，在杀手之道上，可以说，比我都强上几分。”<br/>　　<br/>　　  “那为什么……”王康又是问道，他话只说了一半，但他知道，他们都明白是什么意思。<br/>　　<br/>　　  这次没用袁延说，匕附直接沉声道：“康少爷，我是阳州城人，准确的说，我是阳州城西郊人！”<br/>　　<br/>　　  “西郊人？”<br/>　　<br/>　　  王康有些明白了，在阳州城居住的可不全都是富人，西郊便是贫民区，佃农的居住区。<br/>　　<br/>　　  这时匕附继续道：“我的父亲就是佃农，我们一家都是佃户，受尽地主的欺压，我父亲，母亲，因得罪地主，被生生害死！”<br/>　　<br/>　　  匕附的声音平淡，但王康能听出那其中所蕴含的恨意。<br/>　　<br/>　　  “我侥幸而逃，被杀手组织带走，从此成为一名杀手，待我学成之后，回去屠杀了那地主满门！”<br/>　　<br/>　　  他接着又道：“此次回去听闻，西郊已经有所改变，最大的地主，孙员外已经坐了牢狱，他的靠山李钰，也被收押！”<br/>　　<br/>　　  匕附说着，抬眼看着王康，“而据我调查，这一切都是康少爷所为，我在前段时间，又跟袁延相见，才有所了解。”<br/>　　<br/>　　  “匕附心中钦佩，便前来投奔，愿为康少爷尽心尽力！”<br/>　　<br/>　　  果然是有因，必有果。<br/>　　<br/>　　  当初他除掉孙员外，又除掉李钰，其根本还是为了折损董易武势力。<br/>　　<br/>　　  如今，倒是种下了善果，<br/>　　<br/>　　  “起来吧！”王康想着，将之托起。<br/>　　<br/>　　  他抬眼略过前方众人镇重道：“你们都是精选而出，又经过特训，每一个都是不凡！”<br/>　　<br/>　　  “因此，我将给你单独起名，从今以后，你们名为暗卫！”<br/>　　<br/>　　  王康沉声道：“意为我麾下，隐匿在暗中的卫士！”<br/>　　<br/>　　  就在今日，日后名震大陆，听之令人闻风丧胆，所杀之人，无人能逃的暗卫成立……<br/>　　<br/>　　  【作者题外话】：第二更！(匕附，由书友灰太狼，提供角色名，)<br/><br/><br/>
