　　而就在李承乾对整件事情感到迷惑不解的时候，在他的身后忽然传出一声惊呼。
<br/>　　接着，一件坚硬的物体顶在了他的后脑之上，耳畔传来一个熟悉的声音：“哥，你没想到会是我吧？”
<br/>　　“长乐？你……”李承乾惊讶的微微转过身，在他面前站着的正是他一直都在惦记的长乐，以及被长乐拿在手中已经扳开机头的左轮手枪。
<br/>　　“很惊讶么？”长乐的脸上带着一种让人无法言说的表情。
<br/>　　“为什么？”李承乾淡淡的问道，他相信长乐绝不会无缘无故拿枪指着自己，她一定会有自己的原因。
<br/>　　“为什么？呵呵……”长乐重复着李承乾的问题，口中发出一阵让人听上去十分心酸的笑声。
<br/>　　“长乐，你，你要干什么？快点把枪放下！”原本站在后面的苏玫已经急出了一头的冷汗，想要上前又不敢，只能远远的喊着。
<br/>　　只是苏玫的话并没有起到任何作用，反而被长乐一声怒斥：“你闭嘴，这里没有你们说话的份！”
<br/>　　程咬金等一群老货已经发现了这边情况不对头，一窝蜂的涌了过来，但在看清楚情况之后，也都远远的顿住了脚步，苍白而无力的劝道：“长公主，有什么话好好说，你先把枪放下，万事有老臣为你做主，成不成？”
<br/>　　但长乐似乎真的铁了心，手枪不离李承乾的要害，甚至连头都没回，只是淡淡说道：“几位老将军不用多言，今日千错万错都是长乐的错，待事情过去，长乐一定会给大家一个满意的交待。只不过，现在请你们都退开，我要跟我哥单独说几句话！”
<br/>　　话说到这个份上，已经没有了其他人说话的余地，也没有人敢冒险冲上去把长乐手中的枪抢下来，毕竟枪口下的人是当今大唐的皇帝陛下，万一有个闪失谁也担不起这个责任。
<br/>　　李承乾静静的站在那里，等到所有人都慢慢的退开之后，这才深深吸了口气说道：“长乐，我不管你是因为什么，咱先把枪放下，有什么事情咱们回去再说好不好？”
<br/>　　“哥，从小的时候你就最宠我，不出我提出什么要求你都会答应，不管我惹了什么样的祸事都是你来帮我，但这一次不行，你帮不了我!”长乐拿着枪的手有些颤抖，一双漂亮的眸子中泛起莹莹泪光。
<br/>　　李承乾沉默了，静静的看着长乐，似乎在等着她继续说下去。
<br/>　　“哥，还记得小时候在山洞里那次么？当时你整整昏迷了半个月，就是为了救我回来，我第一次滑雪，第一次骑自行车……”长乐喃喃的说着，泪水自眼中滑落。
<br/>　　“丽质，那都是过去的事情了，现在……”
<br/>　　“不，对于你是过去，对我却是全部！”长乐指着李承乾的枪口突然调转了方向，指向了自己的头颅。
<br/>　　“长乐，你要干什么？！不要做傻事！”李承乾这下是真的急了，刚刚他还有把握长乐不会对他开枪，可是现在长乐用枪指着自己的头，他的心立刻提了起来。
<br/>　　“哥，难道你现在还不知道我在想些什么？世人都说你聪明，难道你真的就看不出来我喜欢你？！”长乐的声音带着无尽的悲凉与绝望，泪水彻底模糊了她的双眼。
<br/>　　一瞬间，整个竞技场的人全都静了下来，没有任何一个人说话，所有人都将目光投向了长乐。
<br/>　　但这一刻的长乐已经不在乎这些了，盛装之下的美丽公主脸上带着凄美的笑容，双眼直直盯在木然的李承乾身上：“哥，我知道，这个世界容不下我的这份感情，父亲和母后也绝不会允许这样的事出现。”
<br/>　　李承乾沉默以对，他真的不知道应该说些什么，长乐现在的状态不管他说什么对她来说都是一种刺激，为了不让事情再激化下去，他只能沉默。
<br/>　　但长乐似乎并没有停下来的意思，任由泪水长流，将枪口紧紧的顶到了自己的太阳穴上：“哥，原本我已经布置好了人手，打算在你刚刚讲话的时候杀了你，然后我再自杀！可是最后我发现我做不到，我没有办法看着你死去，我也不能让你苦心经营的大唐就此陷入一场动荡。”
<br/>　　李承乾愣住了，虽然长乐没有明说，但他已经猜到独孤玉凤失踪，两只步枪丢失竟然都于长乐有关。
<br/>　　可是，就在他准备开口的时候，长乐突然退后一步，大声问道：“哥！现在我只问你一句，你爱不爱我！”
<br/>　　因为前面还有那个扩音器的存在，长乐的话几乎一定不落的传便了整个竞技场，不管是大唐的百姓还是异族的使节，一个两个全都听的清清楚楚，明明白白，在这一刻，所有人都屏住了呼吸。
<br/>　　八卦也好，同情也罢，所有人都在等待着李承乾的回答。
<br/>　　望远镜这东西早就已经卖疯了，来到现场的观众几乎人手一个，长乐的表现可以说已经被所有人看在眼中。
<br/>　　但是，在这一刻没有任何人笑话她，有的只是怜悯！
<br/>　　高高在上的大唐公主，凄美的爱情故事，无数女子看到眼前的一切流下了泪水。
<br/>　　但是，这份爱情并不为世俗礼法所容，李承乾不管怎么回答，对于长乐的结局都不会有任何的帮助。
<br/>　　可虽然如此，李承乾又能怎么办？宠了二十来年的妹妹就这样站在自己面前用枪口指着自己的头，他又能如何回答。
<br/>　　良久的沉默中，李承乾咬着牙抬起头，与长乐对视着，又是良久的沉默。
<br/>　　最终，在无数人的注视下，李承乾对长乐伸出右手：“爱，我也爱你！长乐，我也爱你！”
<br/>　　“哗……”整个竞技场一片沸腾之声，各种各样的议论数不胜数。
<br/>　　不过这声音并没有持续多久，观礼台之上骤然一声枪响，那位敢当着整个天下所有人宣布自己爱情的大唐公主缓缓倒了下去，在她的脸上还挂着一丝满足的笑意。
<br/>　　“长乐……”一声凄厉如野兽哀嚎的嘶吼，让所有人为之黯然！
<br/>　　……
<br/>　　……
<br/>　　……
<br/>　　昏暗的两仪殿中，李承乾一个人静静的坐着，窗外是皎洁的月光……
<br/>　　良久……黑暗中传出一个声音：“来人，宣崔钰觐见……”。
