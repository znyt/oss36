　　第650章
<br/>　　百里无常也当真豪横，整条小臂被砸断，在最初的疼痛过后，他强撑着站起来，再不吭一声。
<br/>　　三水关的兵丁立在他身后，等待他的一声令下，便冲杀出去，将百里胜以及他的部下歼灭在三水关外。
<br/>　　百里无常却没有下令，他强撑着站直身体，注视着百里胜的队伍集结，然后忽然道了一声：“七弟，我劝你莫往长安走，长安等待你的也必然是死局。”
<br/>　　百里胜没有回应，他已经懒得与百里无常说话。翻身上马，扬鞭一挥，战马唿律律一声咆哮，向前方狂奔。
<br/>　　身后的残兵驱马前行，缀在百里胜身后，一口气不知奔出多远。在穿过一座山谷后，百里胜才勒马停下。
<br/>　　他的部下也同时停住，百里胜回头望，八百残兵闯出三水关，现在剩下的不过五六百人。
<br/>　　前方很有很远的路，撑着走到长安，还不知会剩下几人。这一刻，百里胜突然有些心灰意冷。便是抵达长安又能如何呢？世家豪门之间的利益纠葛，纵然自己不明白具体情形，但从来都是千篇一律，搞不出什么新意。
<br/>　　这时候，百里胜心中忽然冒出一个想法：帝国有太多豪门利益，藏了太多肮脏，这样一个帝国真的已经没救了，还不如一把火烧了干净。
<br/>　　“我们……不回长安了。”
<br/>　　望着一路跟随自己闯到这里的残部，百里胜缓缓道了一句。众人并没有什么反应，仍然昂着头，所有人的目光聚焦在百里胜身上。
<br/>　　“帝国已无处可收留我们，他们叫我们做贼，我们便去做贼。”
<br/>　　百里胜吩咐下来，队伍原地休整，派出人去，查看附近何处可以落脚。
<br/>　　到第二天天亮的时候，派出的探子回来，禀报百里胜说：西南五十里处有座野猪山，山上聚集着一伙强人，首领姓方，也不知具体来历。
<br/>　　百里胜率领队伍向西南方向前进，便打算在野猪山扎根。那姓方的好说话则以，如果不好说话，便将他从这里抹掉。
<br/>　　……
<br/>　　野猪山的首领姓方，名唤方大虎，使一杆狼牙棒，手上也有几百喽啰。
<br/>　　这一日，野猪山来了一位不速之客，他衣衫褴褛，脚步慌张，手中擎着一杆鱼叉。刚来到山寨门前，便大呼小叫，口中道：“快开门，开门，是我来了。”
<br/>　　大门紧闭，看守大门的喽啰手中握着兵器，道：“我们寨主说了，什么人也不见，你哪里来的回哪里去。”
<br/>　　那人一楞，随之吼道：“瞎了你们的狗眼，连我都不认得了么？”
<br/>　　“正是因为认得你，才不让你进去。我们寨主不想伤了以往的和气，否则见到你，第一个先砸烂你的脑袋。”
<br/>　　这人正是混江龙，他和方大虎还有些交情。上次，他借着水遁摆脱了单雄信，可那单雄信没有半点放过自己的意思，一路对自己紧追不舍。混江龙便想着来野猪山躲一躲，但没料到方大虎给自己来了个闭门不见。
<br/>　　人在屋檐下，不得不低头，他打了个哈哈，道：“诸位兄弟，这中间是不是有什么，我和你们寨主可是过命的交情，他不认得旁人，可不能不认得我。”
<br/>　　“就是因为你，我们寨主才不见，给你留三分面子，你可别给脸不要脸。”
<br/>　　“兄弟们，这是何苦……”混江龙一楞。
<br/>　　“哼，你为自己逃命，出卖自己弟兄，你还有脸过来此地。”大门上突然出现一个人，不是旁人，正是混江龙原先的军师绿头蝇。
<br/>　　混江龙恍然明白过来，绿头蝇提前一步到此地，将自己独自逃命的事说了。那混江龙偏偏是个讲究义气的人，定是对自己深恶痛绝，所以才选择闭门不见。
<br/>　　他若争气，便该转身离开，大丈夫死则死矣，何必低三下四。可混江龙觉得，争气没有性命重要，那单雄信铁了心要杀自己，除了这里，自己又往何处去。
<br/>　　他也算见机得快，忽然哈哈大笑一阵，口中大声道：“让姓方的出来，我有话与他说！他再不出来，可别怪我骂他十八代祖宗。”
<br/>　　见仍旧没人搭理他，混江龙破口大骂，污言秽语十分刺耳。
<br/>　　骂了半晌，那大门缓缓开启，一握着狼牙棒的汉子狂奔而来，照着混江龙的脑袋便砸。
<br/>　　混江龙狼狈招架两招，忽然跪倒，口中道：“大哥，我委屈呐。”
<br/>　　这大汉便是野猪山的首领方大虎，他的狼牙棒举在空中，道：“你委屈什么？”
<br/>　　混江龙一把一鼻子一把泪，哭得十分悲痛：“我好大一座山寨，竟被这绿头蝇勾结外人毁了，我本来想投奔大哥，没想到这绿头蝇向行一步，颠倒黑白，蒙蔽是非，这是要陷我于不仁不义之地。”
<br/>　　绿头蝇一听这话便大步跑过来：“方寨主，你可不要听他胡说八道，这是从来不曾有的事情。”
<br/>　　方大虎是个老实人，老实人就觉得其他人和自己一样不会撒谎。他看看混江龙，再看看绿头蝇一时间有些糊涂，不知谁真谁假。
<br/>　　正这时候，前方山林中百鸟惊飞，一队人从林中穿出，来到山寨大门前。
<br/>　　这些人虽然衣衫褴褛，可都穿着帝国制式皮甲，人人双眼沁血，仿佛吃人的野兽。
<br/>　　是官兵呐。
<br/>　　卢俊义向前跨出一步，道：“你们谁是当家的？”
<br/>　　方大虎有些摸不准情况，但仍然挺身而出：“我是方大虎，你们是什么人？”
<br/>　　“好，我说句话与你听，从今天开始，这山寨不是你的了。你若识趣的，便乖乖归附我家侯爷。”
<br/>　　方大虎听罢是仰天大笑：“哈哈，是我今天没睡醒，还是你们在做梦，想要我的山寨，问我手中的兵器答不答应。”
<br/>　　卢俊义面向百里胜，道：“侯爷，怎么处理，留不留活口？”
<br/>　　百里胜不耐烦的挥挥手：“一群山贼留什么活口，杀了干净。”
<br/>　　一声令下，他的部下一拥而上。虽然是残兵溃将，能走到这里，是侥幸的成分居多。
<br/>　　然而，面对野猪山的山贼，他们仍然有绝对优势。
<br/>(本章完)
