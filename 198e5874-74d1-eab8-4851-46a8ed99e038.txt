<br/>        而就在这时，他的狐朋狗友中不知道是谁先操起一只酒瓶，猛地向傅夜沉砸去。

<br/>        傅夜沉眸光微沉，敏捷地侧了个身。

<br/>        岂料，那只像是长了翅膀的酒瓶，准确无误地命中季运恒的眉心，打得季运恒两眼冒星。

<br/>        傅夜沉退到一旁，淡定地坐在卡座里，从西装内衣兜里掏出一盒雪茄，抽了一根出来，淡定地点燃，优哉游哉地吸烟。

<br/>        这边打起了群架，“夜惑”内部高管自然不会坐视不管。

<br/>        为了不惊动警方，费尔南派出了保镖来管事，谁知他派来的保镖一并被傅夜沉的手下给治了。

<br/>        “四少，这群人渣，你打算怎么处理？”韩剑锋带着兄弟将这群惹事生非的男人制服后，随手揪起季运恒的衣领，甩到了傅夜沉的跟前。

<br/>        以前跟着傅夜沉出生入死的特种兵，退役后成为了他傅夜沉的私人保镖队。

<br/>        他的保镖队伍，都是通过层层考验的特种兵精英。
<br/>
<br/>        这群小保镖，小帮派，哪是他们的对手？

<br/>        费尔南得知自己派出去的保镖队也被那个王者风范的男人给制服了后，立即带着身边的两个得力女人唐倩兮和晚霞，马不停蹄地赶过来大献殷勤。
<br/>
<br/>        这昏暗的大厅卡座里，只有几处蓝绿色的舞台灯扫过。
<br/>
<br/>        而傅夜沉就坐在卡座的阴影中，淡定地吐着烟圈。
<br/>
<br/>        围过来的这群人，压根就看不清傅夜沉的真实面容。
<br/>
<br/>        “这位爷，您看，咱们和气生财。‘夜惑’向来是给大家带来欢乐的地方，今天怎么就打起来了呢？”费尔南毕恭毕敬地站在傅夜沉的身旁，点头哈腰地陪着笑脸说道。
<br/>
<br/>        晚霞见状，立即扭着腰肢，像条成了精的蛇妖美女一样，挺着两团美峰，往傅夜沉身旁一坐，妖娆妩媚地说道：“这位是连少，对吗？连少，您就大人不记小人过。饶了他们吧！”

<br/>        他们之所以称呼他为“连少”，完全是因为这些天，他一直用连华生的名义在给“小谢”打赏。

<br/>        那就是——以、沫！

<br/>        面前这位“连少”和“季少”，是因为筱颖才闹起来的吧！
