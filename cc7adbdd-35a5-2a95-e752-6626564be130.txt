　　第445章
<br/>　　李行哉说得咬牙切齿，崔相心中却十分不屑。谁不知道你和那程大雷狼狈为奸，凑在一起不亦乐乎。当初李行哉从蛤蟆寨离开的时候，程大雷还特意派兵去抢人，差点和凉州城开战。
<br/>　　这消息早已传到崔相耳中，但对于李行哉的话，崔相却不会去拆穿。只是长叹一声，道：“是呐，我孙女婷儿还在那恶贼手中，现在还不知什么情况。”
<br/>　　人非草木，孰能无情，即便城府深沉的崔相，在提起落入虎口的薛问婷时，也难免有几分真情流露。
<br/>　　李行哉心中感慨，微微侧身道：“怕早已经被糟蹋了。”
<br/>　　崔相的眼角猛地跳动了两下。
<br/>　　他压下心中沸腾的情绪，道：“却也没想到，那恶贼竟然如此了得。摆下天罗地网也杀不了他。只不过，老夫还有一件事想不明白，这程大雷甘冒奇险，孤身来到并州趟这浑水，究竟有什么目的。”
<br/>　　说这话时，崔相眼睛看着李行哉，想要观察他的反应。崔相一直怀疑，李行哉和程大雷之间还有一些联系，这次很大可能，程大雷就是李行哉召来的，双方嘀咕在一起，说不得搞什么阴谋。
<br/>　　“这有什么难猜的，他肯定是想救人呐。”李行哉轻描淡写道。
<br/>　　“救人？”崔相皱起眉头，意味深长的看着李行哉：“救什么人呐？”
<br/>　　“林少羽啊，还能是谁。”李行哉突然领悟崔相眼神的含义，道：“难道还能是救我么，你想什么呢。”
<br/>　　“这不太可能吧？”崔相难以置信道。
<br/>　　“有什么不可能的。”李行哉坐直身子：“程大雷这个人呐，我很了解，基本上，属于脑子有些拎不清。有时候脑子一糊涂，难免就会做一些不可理解的事情，我都习惯了。”
<br/>　　崔相还是第一次从这个角度思考，但如果排除掉程大雷和李行哉勾结的可能，他冒险来到并州，或许就是为了林少羽。
<br/>　　“但现在林少羽被困在杀虎口，伏诛也就在这几日，他就算来了，怕也救不了林少羽，甚至还会自己送命。”
<br/>　　“所以说，他脑子有些拎不清楚么。”李行哉话锋一转：“不过他既然敢来，肯定是把一切事情都想到了，也有救出林少羽的把握。”
<br/>　　崔相沉默下来，想起从前程大雷的所作所为，几乎每一次都是冒天下大不违，对于这等妖孽，确实不可掉以轻心。
<br/>　　“依殿下对那莽夫的了解，怎样才能擒住他？”
<br/>　　“无非就是多派些兵，多用些高手。能杀了他就杀了他，做不到也没办法，无非尽人事听天命而已。”李行哉将话题移开：“林少羽现在的情况如何？”
<br/>　　“他们被堵在杀虎口中，已经是瓮中捉鳖，在劫难逃。”崔相道：“我希望总攻那一日，由殿下亲手斩下林少羽的脑袋。”
<br/>　　“这个自然。”李行哉神色也变得认真起来：“何时发动攻击？”
<br/>　　“明日天亮之前。”
<br/>　　李行哉轻轻点头，眼神变得坚毅起来。
<br/>　　杀虎口是两山之夹，穿过山口是大河拦路，滔滔河水，绝了起义军的生机。而事实上，在过去的这段时间，大概每个人都感受到『死期将至』这四个字的含义。饿死的饿死，冻死的冻死，相比之下，死在敌人的刀枪之下，甚至算是一件幸运的事情。
<br/>　　见惯了死亡，便也就不觉得如何，残余的人马挤在大河和山口之间，倒不是视死如归，而是麻木。
<br/>　　杀虎口周围，屯兵有十几万，都是来自帝国各处的诸侯。
<br/>　　第二天是个阴天，天光亮了，但仍昏昏沉沉发暗。
<br/>　　李行哉走出了营帐，他换了一副打扮，穿九纹缠龙甲，脚踏虎头皂角靴，端地是英气逼人。就连崔相看到，也难免去想：此子竟真有几分帝王之相。
<br/>　　“殿下，老朽年弱体衰，顶不住山风，无法陪殿下上阵。还祝殿下旗开得胜，马到成功，斩下贼人的头颅。”
<br/>　　“好说。”李行哉摆了摆手。
<br/>　　崔相眼神示意，宋游渠牵着两匹马来到李行哉身边。两匹马一黑一白，黑的如墨，白的如雪，全身上下没有一根杂毛。
<br/>　　“殿下请上马。”崔相躬身行礼。
<br/>　　李行哉翻身上了马背，宋游渠将一杆银枪递到李行哉面前。
<br/>　　“今日就让游渠随侍殿下身边。”崔相道：“老朽拭目以待，见李家盘龙枪再次逞威沙场。”
<br/>　　宋游渠翻身上了黑马，跟在李行哉身后。此次出战，李行哉随行还有八十名亲兵。
<br/>　　出得大营，三军已经列阵完毕。都是帝国精锐的凉州骑兵，人如虎，马如龙，龙虎精神。
<br/>　　而其他诸侯的队伍，也已经列阵在杀虎口前。将军府、扬州兵、以及其他诸侯的队伍，从李行哉的角度望过去，乌压压一片，杀气直冲斗牛。
<br/>　　可如今杀虎口中的起义军，不过才两三千残兵败将。
<br/>　　然后，李行哉见到了帝国大皇子李乐天。
<br/>　　他从马上下来，快行几步，来到李乐天的马前，躬身行礼。
<br/>　　“大哥。”
<br/>　　李乐天在马上撇撇嘴，挥手让身边的亲卫散开，道：“六弟，你以前见我，可从未行过这大礼。”
<br/>　　李行哉直起身，尴尬的笑了笑，目光落在李乐天身上。
<br/>　　一个马上，一个马下，二人互相打量着彼此。这是长安之乱后，兄弟二人第一次见面，屈指细算，距离他们上次见面，已经过去五六年时间。
<br/>　　时光如刀，总不会饶过任何一个人。李行哉褪去脸上的稚气，模样已有一二分男人的硬朗。而至于李乐天，则平添了几分憔悴，没有之前帝国大皇子的丰神俊朗。
<br/>　　“六弟，你当初突然失踪，父王一直牵挂着你，现在见到你没事，父王在天有灵，当也能安心了。”
<br/>　　李行哉心头一酸，垂手道：“我也一直牵挂着父王和大哥，知道长安出事，我便想赶回来，只是……”
<br/>　　“你莫说了，我都已知道。”李乐天点点头：“婉儿现在可还好？”
<br/>　　“一切都好。”
<br/>　　“她是幸运的，不用搅在这乱事之中，而你我，则还是身不由己。”
<br/>(本章完)
