　　徐福往他面前走了一步，缓缓讲述起了自己的计划……
<br/>　　半个时辰后，三道人影，在一片荒芜的鬼域中疾行而过，前往阴司。正是道风和徐福。
<br/>　　徐福给道风开了一个无法拒绝的条件，道风最终决定，帮他劫狱救白起。
<br/>　　这么做，不管最终能否成功，道风都知道，这件事必将震动整个阴司，原本他与阴司之间，就保持着一种微妙的平衡，阴司很多人都想捉拿他，经过这件事，他会彻底得罪阴司，就算阴司中有人想要保他，也是保不住的。
<br/>　　不过，他必须这么做。
<br/>　　本来他想找几个帮手，不过徐福认为同行者必须足够强大，否则只会成为累赘，因为劫狱这种事，就是要出其不意，雷厉风行。否则一旦闹出动静，让阴司有机会集结兵力，就算是他跟徐福，也绝对做不到全身而退。
<br/>　　在整个风之谷中，最符合要求的其实是杨宫梓，不过杨宫梓最近在渡劫修行，不能出关，于是道风叫上了建文帝，至于小马和秦风等人，实力还要稍差一点，建文帝实力其实跟他们差不多，不过他有鱼肠剑在手，平添了几分实力。
<br/>　　建文帝不知道底细，对劫狱这种事当然是不支持的，不过既然是道风让他来的，他当然无法拒绝。
<br/>　　三人一路疾行，到了酆都城下，远远地直接运用修为撞了过去，将守城的士兵全用法力结界封印住，兵不血刃，直接冲进酆都城，直奔枉死城而去。
<br/>　　等他们离开之后，结界才告解除，这些士兵一个个瘫倒在地上，面面相觑，不知道发生了什么事。过了半晌，才有一个阴兵结结巴巴地叫起来：“有、有人闯进城里了！”
<br/>　　鬼将回过神来，吓的魂不附体，立刻命令关闭城门，然后飞马通报巡防司，逐级上报……
<br/>　　阴司，已经很久没有遇到过人有人擅闯这种事了。上一次有人擅闯酆都城，还是叶少阳……
<br/>　　枉死城昭狱，虽然在枉死城，却隶属森罗殿管辖，名为昭狱，有平反昭雪的意思，虽然能够住在这里的，基本都是恶鬼凶徒，不过阴司还是希望这些人中能有一部分都是被冤枉的，将来审查一切，给他们平反昭雪，所以有这么一层寓意。
<br/>　　昭狱不是地狱。如果比作人间的话，地狱就像是人间的监狱，那些被盖棺定论判定有罪的鬼妖邪灵，会被送往地狱服刑，但是有一些还没有审查清楚的，就得关在昭狱里，方便森罗殿提刑审问，这就像是人间的看守所一样的所在。
<br/>　　白起的元神，就关在这昭狱里。
<br/>　　昭狱隶属森罗殿，自然也由森罗殿负责看管，它位于枉死城的一角，从外面看就是一座普通的房舍，其中却是包括万象，内含乾坤，只有进去了才知道。
<br/>　　森罗殿有一批兵士，长期驻扎在昭狱外面，镇守此地，负责人却是跟轮回司一样，是从无量界法诀寺“借调”过来的两位尊者：开道和明法。这两位之前并不常驻在阴司，叶少阳当初擅轮回司的时候，跟他们打过一架，之后法诀寺与阴司谈判，因为一些缘故，两位尊者住进昭狱，镇守这里。
<br/>　　昭狱戒备森严，一向没有出过什么事，因此两位尊者平时都很清闲，除了打坐修行之外，就是在一起喝茶下棋。
<br/>　　这个时候，两人正在昭狱里下棋，一边闲谈一些关于三界的八卦。
<br/>　　“二师兄，你说这青冥界之战，最终谁能取胜？”开道尊者手拈着一颗棋子，说道。
<br/>　　明法尊者道：“以我之见，若无外力干扰，最终双方也只能化解干戈，划江而治。”
<br/>　　“何以如此？”
<br/>　　“战线太长，谁也无法压倒谁，最终为了双方的发展，也只能各退一步，息战分治。不过，这却是要看各方势力如何平衡。”
<br/>　　开道尊者放下一颗棋子，说道：“阴司自然是支持空界联盟的，如果太阴山与尸族联合，那却是结果未知。”
<br/>　　明法尊者道：“若尸族胜，必然天下围攻，若尸族败，必然倒向太阴山。”
<br/>　　开道尊者道：“师兄，空界之战，诸方势力都蠢蠢欲动，我法诀寺为何一直没有动静？真真让人好生不解。”
<br/>　　明法尊者道：“不可妄言，菩萨自有谋断，岂能轮到我等操心，守好这昭狱便可。”
<br/>　　开道尊者摇了摇头，笑道：“这昭狱，千年如一日，有什么守不守的，难不成还有人敢来劫狱不成？”
<br/>　　明法尊者手里夹着一枚棋子，刚要放在棋盘上，突然一股乱流，从昭狱的一层冲撞上来，将棋盘冲散，棋子哗哗掉了一地。
<br/>　　两人大惊，互相看去，就在这时，一个银甲鬼武士慌慌张张地冲上楼梯，拱手道：“二位尊者，有人前来劫狱！”
<br/>　　“什么！”明法尊者手指一颤，棋子掉落在地上。
<br/>　　“劫狱，怎可能会有这种事！”开道尊者惊的跳起来。
<br/>　　“是真的，他们……”话音未落，从下方腾起一股紫气，直接将银甲鬼武士撞飞，紫气旋转落地，化成人形，却是一道人影。
<br/>　　两位尊者还没有看清是谁，对方一伸手就是五道紫气袭来，试图困住他们。
<br/>　　两位尊者既然是尊者，修为已臻化境，虽然慌乱，但也立刻就回过神来，各自祭出法器，托住了紫气，一经碰撞，产生出了一波极强的余波，将双方都震开，各自后退。
<br/>　　“五朝元气！！”开道尊者大惊，定睛看去，却是有点面熟。
<br/>　　“道风！”明法尊者却是先叫了出来，“怎么是你！你大胆如斯，昭狱做什么！”
<br/>　　道风也不作声，事到如今，根本没什么好说的，只会浪费时间。道风袖子一抖，将打神鞭摸在手中，朝两人打了过去。
<br/>　　这时候，徐福和建文帝也冲了上来，以三对二，大战起来。
<br/>　　不过昭狱之中，自然不可能只有两位尊者，听到动静之后，从昭狱的深处，不断有听见动静的兵士涌过来，在看见眼前的场面之后，也都震惊了一瞬间，然后加入战斗。
