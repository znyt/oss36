        “来，让我也给你抹点。”明司寒见状，刚举起冰激凌，就吓得连星儿缩着脖子往后躲，顿时惹得他一阵爽朗地大笑。
<br/>
<br/>        “瞧你那怂样！”
<br/>
<br/>        明司寒接着打趣。
<br/>
<br/>        “哼！”连星儿忍不住白了他一眼。
<br/>
<br/>        两人一阵小打小闹后，终于坐上了返回学校的高铁。
<br/>
<br/>        一上车，连星儿就犯困，当她整个人沉睡过去后，身子的重心不稳，慢慢地往明司寒肩上靠了过去。
<br/>
<br/>        明司寒感到重物压了过来，微微偏头之际，鼻息间满是这丫头发间的香气，就像茉莉花盛开时那种清香沁人心脾。
<br/>
<br/>        或许是怕吵醒她，他抬起手，轻轻地替她将额前的刘海撩到耳后根后，便坐在不动了。
<br/>
<br/>        不知过了多久，明司寒只觉自己的手臂渐渐地酥麻起来，难受得像是有不计其数的蚂蚁在手臂上撕咬般，又刺又痛又痒。
<br/>
<br/>        但是，他仍旧选择一动也不动，只为这丫头一觉睡到舒心舒适。
<br/>
<br/>        一阵酥痛麻痒的感觉过后，明司寒也跟着困意袭来，脑袋不知不觉中地偏向连星儿的头顶。
<br/>
<br/>        最后，他的脸颊挨着她的头顶，大手情不自禁地覆在她的小手上，与她五指交叉，跟她一起睡了过去。
<br/>
<br/>        直到连星儿有了一丝动静，明司寒就像是被触电了般，机敏地竖直脖子，正襟危坐，然后将自己的手从她手上挪开，好像什么事情也没发生过一样。
<br/>
<br/>        “是不是快到了？”连星儿揉了揉惺忪的睡眼，迷迷糊糊地问道，压根就不知道刚刚在她睡着的时候，所发生了什么事情。
<br/>
<br/>        明司寒因为心虚，声音有些慵懒迷离地应了声：“嗯。”
<br/>
<br/>        下了高铁后，两人随之又坐上了计程车。
<br/>
<br/>        在车内，连星儿拿着手机给明司寒转了一千块钱，就是刚刚在北城买的那些特产的钱。
<br/>
<br/>        听到“叮咚”信息提示音，明司寒本能地掏出手机看到手机转账信息后，他不但没有领取，反而斜睨了身旁的连星儿一眼。
<br/>
<br/>        “你是觉得我很穷？”明司寒一脸不屑地质问。
<br/>
<br/>        连星儿看了明司寒的脸色一眼，咧嘴假笑道：“我知道你不穷，但这是我自己买的东西，我自己付钱就好。”
<br/>
<br/>        起初买东西的时候，都是明司寒抢着付款的。
<br/>
<br/>        在外人面前，她不好与他争执，只得让他先垫着，回头自己再数给他。
<br/>
<br/>        “我的钱可是我自己赚的，我想怎么用就怎么用。你要是非得跟我AA制，那解除婚约的事情……”
<br/>
<br/>        “行了行了！我接受了还不行吗？你一定要想办法跟我解除婚约啊！”连星儿摆了摆手，慌忙妥协。
<br/>
<br/>        明司寒暗自窃喜，嘴角不禁微微上扬：“这还差不多！”
<br/>
<br/>        以前他自己赚钱的时候没多大感觉，自从他为她连星儿买这买那之后，他突然觉得自己这钱赚得实在是太值，花钱似乎也变得格外有意义。
<br/>
<br/>        出租车到了学校门口，明司寒付了车费，先下的车，待到连星儿从车内出来时，他伸手挡在车门框顶，防止这丫头撞头。
<br/>
<br/>        这是他本能的一个动作，却被不远处停着的一辆豪车里一双黑眸尽收眼底。
<br/>
<br/>        “Boss，经过这段日子对明家的这个小少爷的考验，不知道他为少小姐所做的一切，可否还令您满意？”一个虽然年轻看上去却很干练的男助理，坐在副驾驶座上，侧身看向驾驶座后位上的傅夜沉，毕恭毕敬地问。
<br/>
<br/>        这个男助理名叫郑恒信，是郑忠威的儿子，郑忠威退休后，他成为了傅夜沉的右翼。
<br/>
<br/>        明司寒参加国际黑客战队大赛时，连星儿与他的关系，是傅夜沉派他去透露给明司寒的死对头贺泽川的。
<br/>
<br/>        傅夜沉在暗中已经做好了万全的准备，去护连星儿的周全。
<br/>
<br/>        好在，明司寒没有让傅夜沉失望。
<br/>
<br/>        明司寒不仅救回了连星儿，而且还赢得了比赛。
<br/>
<br/>        更重要的一点是，在酒店里，发生那种特殊情况，明司寒也没有趁人之危，对连星儿下手。
<br/>
<br/>        这边，连星儿一下车，抬眸的无意间便看到了那辆令她熟悉得不能再熟悉的豪车。
<br/>
<br/>        就在明司寒替她去后备箱搬行李的时候，连星儿低头从手提包里掏出手机，给傅夜沉打了通电话。
<br/>
<br/>        另一边豪车内，傅夜沉淡定地从裤兜里掏出手机，直接接听了电话。
<br/>
<br/>        “爹地，你是来我学校守株待兔的吗？”连星儿笑嘻嘻地问。
<br/>
<br/>        傅夜沉嘴角微扬，反问：“那我抓到兔子了吗？”
<br/>
<br/>        “抓到了！”
<br/>
<br/>        “几天没回家了？几天没往家里打电话了？”
