    矛盾缓和后，火咀三人带着陈兴和阿乔木前往城市中间的高塔。<br/>    <br/>    半路上，他们遇到了一伙黑骷髅的人，带头的就是刚才在石屋外面的喊话，自称“杨酷”的年轻男人。<br/>    <br/>    “火咀，你们又在忽悠新人啊？”杨酷打量着陈兴阿乔木两人，阴阳怪气地说道。<br/>    <br/>    火咀顿时面露怒容，大骂道：“再乱说话，小心我撕了你的嘴！”<br/>    <br/>    或许清楚火咀的火爆脾气，杨酷皮笑肉不笑地勾了勾嘴，带着人离开了。<br/>    <br/>    “陈兄弟，别听他胡说，他就是个小人，我们雷火团从来都是实实在在的。”可能是怕陈兴误会，在杨酷走后，火咀解释道。<br/>    <br/>    “是啊，黑骷髅那帮人才不讲信用。”“何止不讲信用，简直坏透了。”旁边两个同伴也跟着帮腔。<br/>    <br/>    “不用说了……”陈兴摆了摆手，干脆说道，“我相信你们！”<br/>    <br/>    “哈哈哈！”火咀放声大笑，随后面露疑惑，问道，“陈兄弟以前认识我们吗？”<br/>    <br/>    这句话看似随意，却饱含着另一层的深意。虽然火咀看起来大大咧咧的，但实际上是个心细如发的人。行走于生死边缘的佣兵，决不相信世界上有无缘无故的信任。蜜糖的里面可能包裹着炮弹，笑容的背后可能藏着尖刀，任何一个没有缘由的友善都可能是危险。<br/>    <br/>    对于火咀的疑虑，陈兴早就想好了说辞。只见他脸色如常，说道，“其实在你们之前，已经有好几拨人来找我们了。”<br/>    <br/>    “但他们只是说，正在找一种黑色的拼图，并没有告诉我们是高塔的钥匙，而你们没有隐瞒真相，直接告诉了我们。”<br/>    <br/>    “还有就是，他们只提出了收购，从来没有想过要带我们进去。”<br/>    <br/>    “所以……”陈兴摊开双手，“你们明显比他们值得相信。”<br/>    <br/>    “原来是这样啊，哈哈哈！”火咀拍着脑袋，大笑道，然后收起笑容，带上了几分认真，“陈兄弟，看来还是我以小人之心度你的君子之腹了，不过你也别见怪，我们不想害别人，但也不想被别人害，毕竟这里的环境……”他叹了口气，显得有些无奈，“实在太复杂了……”<br/>    <br/>    “可以理解。”陈兴说道。<br/>    <br/>    “那就谢谢了。”火咀抱拳说道。这时阿乔木也看了过来，两人目光交汇，都带着不信任。火咀二话不说，立即板起脸，转身继续带路。<br/>    <br/>    过了小一会儿，阿乔木故意落后几步，拉扯着陈兴的衣角，在他耳边低声说道，“我总觉得你有很多秘密。”<br/>    <br/>    陈兴斜看了她一眼，“听说过好奇害死猫的故事吗？”<br/>    <br/>    “别把沙漠豹当做猫，你会后悔的。”阿乔木说道。<br/>    <br/>    “我们现在是盟友。”陈兴提醒道。<br/>    <br/>    “是的，我是在提醒你，你有一个非常可靠的盟友。”阿乔木说道。<br/>    <br/>    陈兴再次看了她一眼，称赞道，“口才不错。”<br/>    <br/>    “谢谢。”阿乔木颔首致意。<br/>    <br/>    “不用谢。”陈兴面无表情地说道。<br/>    <br/>    “但我还是觉得……”阿乔木坚持道，“你有很多秘密。”<br/>    <br/>    “随你怎么认为……”陈兴耸了耸肩，然后声音变得低沉，“但我建议你，最好不要起什么歪心思，否则你会付出非常沉重的代价。”<br/>    <br/>    “我们沙民的世界很简单，不像你们黄猴子这么爱算计。”阿乔木说道。<br/>    <br/>    “希望是这样吧。”陈兴说道。<br/>    <br/>    两人有一句没一句地聊着，约摸一个半小时后，火咀带着他们穿过大片石屋、风车磨坊、马厩市集，来到了高塔前方的广场。<br/>    <br/>    抬眼望去，塔尖高耸入云，绽放着莹绿的光芒。<br/>    <br/>    塔身呈圆锥形，上小下大，共分为九层，每一层的外围有一圈石柱，建筑风格有点像比萨斜塔。塔上开有拱形的窗户，但里面黑漆漆的，什么也看不见。<br/>    <br/>    整个塔都散发着微弱的金光，仿佛蒙上了一层金沙。<br/>    <br/>    目光朝下，塔前的广场搭满了帐篷。中间留出十字形的通道，明显被分割成四块，营地里插着不同战团的旗帜。<br/>    <br/>    黑色骷髅头加刀剑的应该是黑骷髅的，长刀穿过桃心的应该是次郎团的，画着张开血盆大口的人猿头部的应该是巨猿团的。<br/>    <br/>    雷火的团的旗帜最简单，两个白色的大字——雷火！<br/>    <br/>    字体苍劲饱满，笔锋凌厉大气。<br/>    <br/>    除了“光”字的不同，其余都和上一世的一模一样，应该是出自同一个人的手笔。<br/>    <br/>    走进雷火团的营地，大大小小的帐篷延绵数百米，人数至少有两百以上，估计这只是进来探索的部分，外面还有临时营地，以及看守大本营的。<br/>    <br/>    按照佣兵行业的惯例，小规模的佣兵团被称作“团伙”，中规模的被称作“战团”，大规模的被称作“军团”。<br/>    <br/>    战团的人数，通常在五百到两千左右，有独立的侦察队、突击队、机甲分队、后勤大队等等。不过黑石大陆的战团肯定没有机甲，因为买不到。<br/>    <br/>    营地里的佣兵三五成群，蹲在一起抽烟聊天，或是打扑克赌钱。<br/>    <br/>    陈兴留意到，扑克旁边的注码都是铜币。上一世的雷光团有条不成文的规矩，同袍之间可以赌钱，但只能小赌，不能大赌。因为铁诺认为，赌注过大会影响兄弟情谊。<br/>    <br/>    营地里的雷火团成员看见火咀，都只是点点头，没有站起来。这也是雷光团的惯例，除非是在执勤，否则见到长官不需要起身敬礼，因为大伙都是平等的。<br/>    <br/>    “喂，火咀老大，上次你说要给我加钱的，这个月怎么没变啊？”<br/>    <br/>    走到营地中间的时候，一个十五六岁、身材瘦弱的小佣兵挡住了火咀的去路。<br/>    <br/>    “滚一边去，上次巡逻开小差还没罚你呢，还好意思说加钱。”火咀大手一张，拨开对方，边走边骂。<br/>    <br/>    “可我这个月没有犯错啊。”小佣兵追上来说道。<br/>    <br/>    “那就下个月再加。”火咀说道。<br/>    <br/>    “可是，可是……”小佣兵显得有些局促，“小莹月底就过生日了，我想送她一支强化药剂。”<br/>    <br/>    火咀停了下来，脸色变得有些阴沉。或许是心虚，小佣兵向后缩了缩。<br/>    <br/>    “你怎么还执迷不悟啊！”火咀皱着眉头，一副恨铁不成钢的样子，“你为了她，跑去偷东西，结果被抓了，流放到这里来，可最后呢？”<br/>    <br/>    “最后她为了自己的名声，连送行都不来，装着不认识你。这样的女人，还有什么好留恋的！”火咀近乎于吼。<br/>    <br/>    “小莹，小莹她也是有难处的……”小佣兵涨红着脸，争辩道。只见他握着拳头，身体因为激动而颤抖，仿佛一直以来的信念被否定了。<br/>    <br/>    “唉……”看着小佣兵执着的样子，火咀叹了口气，挥手说道，“好吧，去跟会计说，我同意给你加薪。”<br/>    <br/>    小佣兵顿时眉开眼笑，“谢谢老大！”<br/>    <br/>    “谢个屁，没用的家伙，迟早死在女人手里。”火咀骂道。<br/>    <br/>    “老大，你没听说过‘牡丹花下死，做鬼也风流’吗？”小佣兵嬉皮笑脸地问道。<br/>    <br/>    “快滚！”火咀板起脸，瞪圆眼睛，一副凶神恶煞的样子。<br/>    <br/>    “老大，我先走了。”小佣兵笑嘻嘻地跑了。<br/>    <br/>    看着逐渐远去的单薄背影，火咀叹息了一句，“可怜的小家伙……”<br/>    <br/>    “是啊，被骗得团团转都不知道。”“这会儿还不知道在哪个富少爷的床上呢，唉……”两个同伴议论道。<br/>    <br/>    陈兴和阿乔木都不是爱说闲话的人，保持着沉默，没有做出任何评价。<br/>    <br/>    这种“绿茶婊”的事情，陈兴以前在地球上听多了，见怪不怪，也不觉得同情，反正一个愿打，一个愿挨，两厢情愿，没什么好说的。<br/>    <br/>    他追求女孩子的时候，作为男性肯定会先付出几次，但如果对方没有足够的回应，他就会转身离去，哪怕再喜欢也会理智对待。死缠烂打没意思，放低身段、卑微迎合也不长久，还不如早点儿放弃，寻找下一个目标。<br/>    <br/>    在他看来，世上没有谁是“必须”的，女人各有各的美，各有各的好，只会欣赏一种美、吊死在一棵树上的，既可悲又可怜。<br/>    <br/>    通常这种人也很自恋，觉得世上就这么一个女人配得上自己，所以被虐了，也不算冤。<br/>    <br/>    思绪之间，陈兴跟着火咀来到指挥营帐前。<br/>    <br/>    这顶指挥营帐有五六米高，两个半圆形，有点儿像大型的蔬菜棚子。火咀掀开帘子，将陈兴让了进去。<br/>    <br/>    帐篷的中间是个长方形的沙盘，一名穿着短袖军服的男人背对着门口，双手抱胸，望着沙盘上的彩色小旗。<br/>    <br/>    背影高大壮实，肩膀又宽又厚，给人一种沉稳持重的感觉。<br/>    <br/>    “老大……”火咀走上前，低声唤道。<br/>    <br/>    男人缓缓转过身，或许是受上一世记忆的影响，陈兴的呼吸不由得停滞了一下。<br/>    <br/>    脸型方正，眉目英挺，正是那个在龙石镇的酒吧里，抓着他的肩膀，邀请他一起打天下的男人。<br/>    <br/>    “这位兄弟是大师团的团长陈兴。”火咀介绍道。<br/>    <br/>    “幸会。”铁诺伸出大手，陈兴握了上去。<br/>    <br/>    正如记忆中的一样，铁诺的手掌宽厚有力，充满了热情。<br/>    <br/>    目光越过厚实的双肩，一张张熟悉的面孔出现在眼前。<br/>    <br/>    壮得像头水牛的轰雷、高高瘦瘦的大鼠、矮壮结实的铁炮、嗓门粗犷的铜锣、圆滚滚的熊奔……<br/>    <br/>    他清晰地记得，他们每一个人是怎么离开的。南高地之战时，轰雷身先士卒，冲上高地，以一己之力，为友军破开一条血路。面对着数百名敌军，被打成了筛子。<br/>    <br/>    铁炮和铜锣是亲兄弟，熊奔是他们的堂兄弟，三人被称作三人组，分别死于陈兴加入雷光团的第七、第八和第十一年。<br/>    <br/>    大鼠和熊奔的关系很好，喜欢做些滑稽事逗大伙笑。比如，他经常开大猫的玩笑，说大猫是他的天敌，看见大猫就双脚发抖。熊奔死后，大鼠心灰意懒，黯然离开。<br/>    <br/>    看到一个个曾经的战友，陈兴只感到热血上涌，差点儿控制不住情绪，红了眼圈。<br/>    <br/>    让他冷静下来的是一个年轻的蓝血人。<br/>    <br/>    身姿挺拔，眉目清秀，只是皮肤有些病态的苍白。陈兴不认识对方，估计就是火咀之前提到的沈光明。<br/>    <br/>    陈兴在观察沈光明的时候，沈光明也在观察陈兴。目光平和，气质内敛。<br/>    <br/>    聪明人和聪明人之间，往往只隔着一层玻璃，一眼就能识别出同类。<br/>    <br/>    为了避免引起怀疑，陈兴很快就挪开了目光。<br/>    <br/>    <br/>
