　　在大祚荣的眼中，唐军的营地里面已经是浮尸遍里的景像，就算是有活人，他的五万骑兵也可以轻而易举的将之拿下。
<br/>　　所以他根本没有察看四周的环境，也没有想会不会被唐军反击，大手一挥便下达了全军进攻的命令。
<br/>　　要知道，为了给这靺鞨人足够的进攻机会，黑齿常之已经在开始的时候对大营进行了有针对性的布置。
<br/>　　一万人的大营布置的就算是容纳两万人都绰绰有余，对于五万靺鞨骑兵来说，营地的大小正好够他们展开进攻。
<br/>　　战马的速度一点点加快，泥水四浅，不过那些靺鞨骑兵已经顾不上这些了，他们的心中只有一个念头，那就是用人头换赏金，又或者去唐军大营中捡拾一些装备。
<br/>　　可是，他们并没有意识到，理想与现实终是有差别的。
<br/>　　就好像李承乾说过的一样，当你以为自己的计划已经百分百成功的时候，那么距离末日也就不远了。
<br/>　　……
<br/>　　事实证明，以上的理论并没有错，急于冲击唐军大营的靺鞨人很快就偿到了苦果。
<br/>　　刚刚进入营地，还没等他们观察周围的形式，便已经有一部分人猛的一陷，战马莫名其妙的就矮了半截，十分突兀的停了下来，从而将马上的骑士丢了出去。
<br/>　　刚刚开始的时候大祚荣还以为是人有失手，马有失蹄，毕竟唐军大营之中已经是一片泽国，出现什么情况都不奇怪。
<br/>　　可是，当他用手中长矛挑起一具浮在水面上的尸体时，他的脸色瞬间就变了。
<br/>　　因为那根本就不是人应该有的重量，那尸体太轻太轻了，就算是一个孩子也不可能有这样的份量。
<br/>　　而当他把那‘尸体’挑翻的时候，里面的露出来的芦苇，让他的心彻底沉了下去。
<br/>　　“不好，我们上当了，撤，快撤！”但还没有等他下达进一步的命令，远处已经有人喊了起来，明显是有人继他之后，发现了同样的情况。
<br/>　　可是，说撤真的那么好撤么？五万人发起的冲锋又岂能说停就停。
<br/>　　所以那喊声根本没起到任何作用，大队的骑兵依旧不断的冲进唐军的大营，随后便不断有人陷入黑齿常之提前为他们准备好的坑里。
<br/>　　这下大祚荣可真的懵了，听着那些落入坑中的战马发出的一声声惨叫般的嘶鸣，看着水中浮着的一具具‘尸体’，一阵不祥的预感开始在他心中蔓延。
<br/>　　……
<br/>　　而就在此时，在他西北的方向，三颗碧绿碧绿的信号弹升入了高高的天空。
<br/>　　而在他的身后，一支数千人的骑兵也已经摆出了进攻的势头。
<br/>　　“回军，快回军！”看到身后的骑兵，大祚荣的心已经彻底的凉了。
<br/>　　他的五万军队正处在一个地势低洼的所在，地面满是泥水不说，还有一个个被淹没在泥水下面的大坑。这样的情况下发起反击显然是不可能的，因为他的骑兵根本不可能把速度提起来。
<br/>　　可是不反击又不行，总不能眼看着被唐军把自己所有人都干掉吧。
<br/>　　情急之下，无奈的大祚荣只能命令部队调头，试图以人数上的优势冲破唐军的伏击。
<br/>　　……
<br/>　　黑齿常之为了对付大祚荣已经精心的准备了一个晚上，他又如何能让大祚荣真的打出一个反击来，所以就在那些靺鞨骑兵调头，向他们缓缓接近的时候，一架架连发的便携型床弩被架了起来，油纸包裹着的火箭筒也被拿了出来。
<br/>　　随着敌人的一点点接近，黑齿常之狠狠的挥动了手臂。
<br/>　　大唐的武器到底有多犀利？大祚荣以前只是听那些逃进深山的高句丽人说过。
<br/>　　但现在他终于知道了，特么这就不是在打仗，而是在屠杀。
<br/>　　火箭弹飞出，一炸就是一大片，况且那些来自大山深入的战马什么时候见过这种武器，被那巨大的声音一吓立刻又会乱成一团。
<br/>　　连发床弩在两个人的操作下就好像死神之吻，永不停止的吸取着人类的灵魂。
<br/>　　而那些唐人手中的手弩似乎只是床弩的缩小版，那东西竟然也是连发的。
<br/>　　面对这样接连不断的攻击，反击的靺鞨人连喘息的机会都没有，冲上一批死的批，冲上两批死两批，反击发起了一刻钟，他们除了付出数千人的代价之外，甚至连唐军阵前两百步都没有接近过。
<br/>　　反观那五千唐军，此时他们正以一种稳健的速度慢慢向他们推进，箭雨不断的覆盖过来，压的那些反击的骑兵不断后退。
<br/>　　不行，反击估计是不行了，后面这些唐军的武器太过犀利，就算是把这五万人都填进去估计也不可能冲出一个口子。
<br/>　　大祚荣观察了战场形势之后，果断下令反击取消，全军继续向前，穿过唐军大营，从另一端杀出去。
<br/>　　可事情真的那么容易么？他们刚刚进入营地的时候便已经知道，这营地其实就是个陷井，里面到处都是数不清的大坑，坑中还有一些尖锐的物品，不管是人是马，只要掉下去不会有好结果。
<br/>　　所以就算是穿过唐军大营，不付出一定的代价也是不可能的。
<br/>　　可是事到如今已经没有了选择的余地，反击只有死路一条，一直冲虽然可能受伤，但至少有命离开，是个人都知道应该怎么选择。
<br/>　　于是反击的靺鞨骑兵再次开始调头，在后面唐军的压迫下重新回到了营地当中，趟着及膝深的泥水向大营的另一头穿插。
<br/>　　……
<br/>　　前文书咱们说过，唐军的人数是一万，其中有五千人归黑齿常之统领，埋伏在沈城的西北。
<br/>　　而在沈城的西南，还有另外一直队伍，那就是副将所统领的另外五千唐军。
<br/>　　所以当大祚荣带领着靺鞨骑兵好不容易穿过大半个营地之后，愕然发现，原来在大营的另一头，早就有另外一支唐军在等着他了。
<br/>　　同样装束的骑兵，同样的装备，同样冰冷的眼神让大祚荣意识到，这一次估计真的是凶多吉少，估计一个不好这五万人全都要留在这片泽国之中也说不准。
