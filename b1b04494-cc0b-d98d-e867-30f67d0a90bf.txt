陆茵茵出身高贵，阅人无数，经历了林水瑶和柳璎珞那个时代，哥哥是一代商界帝王，侄子又是陆寒霆和陆子羡，所以眼光甚高，看人很准，她看上的也只有夏夕绾那样的天才少女。
<br/>
<br/>        所以陆茵茵一眼就看出了夏妍妍眼里的伪善和心机，她淡淡道，“钱教授是我们圣黎院最好的教授，你可以跟钱教授学到很多东西，医学是纯粹的，自己要多用心。”
<br/>
<br/>        夏妍妍很想在陆茵茵面前表现自己，但是她完全没有想到陆茵茵竟然态度如此冷淡的对她说教了一通。
<br/>
<br/>        “陆校长。”这时有人走了过来，陆茵茵没有再看夏妍妍，而是跟别人交谈去了。
<br/>
<br/>        夏妍妍很失望的拉了一下钱教授的衣袖，委屈巴巴的嘟嘴，“导师，校长好像不喜欢我。”
<br/>
<br/>        钱教授安慰道，“校长就是这样的性情，并不是针对你。”
<br/>
<br/>        夏妍妍的心里还是很不好受，觉得受了很大的打击。
<br/>
<br/>        这时夏妍妍一抬头，突然在门边看到了一道纤柔的身影，是夏夕绾。
<br/>
<br/>        夏夕绾竟然来了！
<br/>
<br/>        夏妍妍没有想到竟然在这里看到了夏夕绾，她怎么有资格来参加这个聚会的，她有烫金邀请函么？
<br/>
<br/>        夏妍妍迅速走了出去，“绾绾，你怎么来了？”
<br/>
<br/>        夏夕绾穿了那条明裸色的长裙，少女的身段被勾勒的纤尘玲珑，今天她随意绾了发，几缕毛绒秀发垂在天鹅般优美的粉颈里，肌肤赛雪，澄亮的眸子顾盼流转，一出场就像是明珠般莹润动人。
<br/>
<br/>        夏夕绾看着夏妍妍，勾了一下红唇，“夏妍妍，我来这里让你很惊讶？”
<br/>
<br/>        “绾绾，你知道这里是什么地方么，这可是圣黎院校长举办的聚餐，圣黎院你听说过吧，传说里医学的殿堂，也就是我的母校，当然了，像你这样才高中毕业的人是只能仰望了，你怎么敢来这里的，自取其辱么？”
<br/>
<br/>        看着夏妍妍得意的样子，夏夕绾笑道，“自取其辱这个成语你还是自己留着吧，我来这里当然是有人邀请我的。”
<br/>
<br/>        夏妍妍哼笑了一声，“绾绾，来这里的可都是医学界的教授名人，他们邀请你干什么，就算要撒谎，你也要撒一个可信度高的。”
<br/>
<br/>        这时门边的侍者也出声道，“这位小姐，进去的人都要出示烫金邀请函，请问你有邀请函么？”
<br/>
<br/>        夏夕绾摇头，“我没有。”
<br/>
<br/>        “不好意思，那你不能进去。”
<br/>
<br/>        夏妍妍嗤一声笑开了，“绾绾，你还是快点走吧，如果你再不走的话，我就要叫保安将你赶走了。”
<br/>
<br/>        “夏妍妍，我跟你说了真话你却不信，我虽然没有邀请函，但是的确是有人邀请我来的，这个人就是…圣黎院的校长啊。”
<br/>
<br/>        什么？
<br/>
<br/>        夏妍妍整个人一僵，她脑海里迅速浮现起了刚才陆茵茵冷淡傲慢的样子，她会邀请一个废才夏夕绾过来？
<br/>
<br/>        “绾绾，你当我是三岁的孩子么，你这么拙劣的谎言我会信？保安，快点将这个女人给赶走！”
