　　原本还带着些许醉意的普提查瞬间就清醒了，原本残留在他体内的酒精，也仿佛随着他惊出的冷汗从体内挤了出来。
<br/>　　普提查下意识的就想要去摸放在桌子上的uzi冲锋枪，可是他的手还没触碰到uzi冲锋枪，冰冷的刀锋就已经贴在了他的脖子上。
<br/>　　“别动！”
<br/>　　罗昊左手握着军刀架在普提查的脖子上，伸手拿起桌子上的uzi冲锋枪，卸掉了冲锋枪弹匣，关掉了武器保险，把冲锋枪扔到了木床上。
<br/>　　军刀锋利的刀刃划破了普提查脖子上的表层皮肤，在他脖子上留下一道淡粉色的划痕。
<br/>　　“你是谁？你想干什么？”普提查用生硬的英语向罗昊问道。
<br/>　　“你自己做了什么事情，难道还不清楚吗？”罗昊冷声反问。
<br/>　　罗昊的话令得普提查心头“咯噔”一跳，立刻明白了自己身边这个用军刀架着自己脖子的男人的身份，以及知道了他来这里的目的。
<br/>　　来得好快！
<br/>　　普提查暗暗心惊，感叹罗昊他们来的速度好快。同时，普提查的心中也有些疑惑，疑惑那个被自己绑架来的女人到底是什么身份，为何会惊动特种部队？
<br/>　　普提查眼珠子转动着，左手朝着桌子下伸去。在桌子底下，普提查还藏了一把手枪，就是为了应对如眼前这种特殊情况；可是在普提查的手触碰到桌子下手枪的时候，他又把手收了回来，心中有了新的盘算。
<br/>　　这些人是为了被自己绑架的那个女人而来的，那么自己何必要跟他们为敌？直接告诉他们这件事情是布克尔特所主谋，让他们去找布克尔特的麻烦，借刀杀人！
<br/>　　在T国有了自己势力，有了自己发展的普提查已经不再满足于现状，他想要有自己的“王国”，而不是一直做布克尔特的下线！
<br/>　　“那个被你们绑架的女人在哪里？”罗昊冷声对普提查问道。
<br/>　　“她不在这里。”布克尔特回答道：“我只负责把她带到这里来，她在这里只逗留了一个晚上，就被带走了。”
<br/>　　“她被带去了哪里？”
<br/>　　“南美洲。”普提查回答道：“是布克尔特，布克尔特把她带去了年南美洲。”
<br/>　　“布克尔特？”
<br/>　　罗昊皱眉，显然是对这个名字没有太深的印象。罗昊把架在普提查脖子上的军刀移开，警告道：“你最好不要耍花招，我能摸进来一次把刀架在你的脖子上，就能第二次把刀架在你脖子上。”
<br/>　　“我不会拿自己的脑袋开玩笑，所以，我不会骗你。”
<br/>　　罗昊收回军刀，右手握着uSP40手枪，枪口下方导轨上的激光指示器在普提查的脑袋上留下一个红色的圆点。普提查一动不动的坐在桌子边，看着罗昊朝着地板上那个他爬进屋子里的洞口退去，没有任何要阻拦，也没有任何要示警的意思。
<br/>　　等到罗昊从地板上的破洞跳下，离开屋子，普提查从桌子边站起身来，摸了摸自己脖子上被刀刃擦破了表皮，渗出少量血迹的口子，眼睛中寒芒闪烁。
<br/>　　“所有人准备撤退！”
<br/>　　罗昊通过耳麦向其他在渔村中的战术中队队员下达了撤退命令，并且通过PDA向正在海面上待命的葛铭发送了信号。
<br/>　　“撤退！？”
<br/>　　梅庆伟对罗昊的命令感到很意外，问道：“到底是怎么回事？”
<br/>　　进入渔村的十一个人，包括罗昊自己在内，全都已经做好了战斗的准备；可是现在什么都没有发生，就这么平静的撤退，无可避免的会让人感到疑惑。
<br/>　　其实不仅是梅庆伟，就连罗昊自己也是没想到事情会是以这样的方式收尾，完全是超出了罗昊在进入渔村前对事情的意料。
<br/>　　“具体的回到渔船上再说，我已经知道佟千雪的下落了。”罗昊按着耳麦，朝着海滩的方向移动。
<br/>　　海边沙滩上，A组和B组的队员先后抵达了撤离点，而C组的三个人却还没有看到他们的踪影。
<br/>　　哒哒哒哒哒！
<br/>　　AK-47自动步枪的点射声打破了渔村的寂静。
<br/>　　梅庆伟、张子航、刘子锦三个人出现在罗昊、齐凡等人的视野中，在他们身后有着好几个毒贩正追着他们。
<br/>　　毒贩手中的AK-47自动步枪不断冒出火光，逼迫着梅庆伟、张子航、刘子锦三个人利用“z”型机动，进行着规避。
<br/>　　子弹落在他们身侧的沙地上，因为土质松软的关系，当子弹击中沙地的时候，沙土被掀起一米多高，“扑簌簌”的掉落在C组的三个人身上。
<br/>　　罗昊是打算在不跟渔村中毒贩正面接触的情况下，从渔村中撤离的，但人算不如天算，没想到C组这边还是跟毒贩遭遇了。
<br/>　　“掩护他们！”
<br/>　　罗昊抓过背在身上的uMP-45冲锋枪，对着一个追在最前方的毒贩，打出一个点射，将对方放倒在地。
<br/>　　奔跑中，刘子锦突然一个转身，脚在沙地上一蹬，身体借助这蹬踢之力向后倒飞而出，他手中uMP-45冲锋枪，接连打出两个点射，把两个毒贩击毙；身体在落地的一瞬间，迅速向一侧滚开，避开了一串朝着他扫来的子弹。
<br/>　　噗噗噗噗噗！
<br/>　　子弹落在沙地上，沙土纷飞。
<br/>　　刘子锦摸出一颗手雷，拉掉保险销，用力按下保险夹，将保险夹松开，甩手朝着追来的毒贩扔去。
<br/>　　“轰——！”
<br/>　　手雷炸起一大蓬沙土，周围好几个毒贩被掀翻在地。
<br/>　　噗！
<br/>　　狙击组这边，李毅峰和慕容飞也是相继开火。
<br/>　　一颗.48口径的狙击弹击中一名毒贩的胸口，毒贩后背位置破开一个大口子，大蓬的鲜血中掺杂着些许内脏碎片从毒贩身体中飞出，扫落在沙地上，血液将地上的沙土凝成一个个小圆球。
<br/>　　慕容飞眼睛中迸射出道道寒光，拉动枪栓，退出弹壳，重新推上一颗子弹，又是瞄准了一个正在跑动中的毒贩，深吸一口气，狙击镜中因为氚元素而发出荧光的十字刻线套住了对方的脑袋，扣动扳机。
<br/>　　噗！
<br/>　　M200狙击步枪发出一声沉闷的射击声，子弹带着慕容飞的杀意，朝着毒贩飞去。
