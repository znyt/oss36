　　“苏莫，我们去星空！”傅同向苏莫轻喝一声，脚下一蹬，身形便冲天而起，直上九霄。

<br/>　　“走，我们去看看！”
<br/>　　“一起去！”

<br/>　　其他一些内宗大殿内的弟子见此，顿时纷纷跟随傅同和苏莫，向星空之上飞去。

<br/>　　“好像是苏莫，还有很多武尊境的弟子！”
<br/>　　“这是要干什么？”
<br/>　　“难道是有人要挑战苏莫，抢夺他的真传弟子之位！”
<br/>　　“走，我们也去看看！”
<br/>　　问道山上，很多弟子都发现了这一幕，这么多武尊境的弟子飞上星空，显然是有大事发生。

<br/>　　苏莫现在是道宗的风云人物，既然是与苏莫有关的事，他们自然想去一关。
<br/>　　苏莫跟随着傅同，直上九天，很快便穿过了太阴星的九天罡风层，飞到了无尽的星空之中。
<br/>　　两人伫立在星空中，巨大的太阴星悬浮在脚下，看似极近，实则已经相隔无数万里。

<br/>　　破空声不断，大量的道宗弟子跟了上来，纷纷四散而开。
<br/>　　这些到来的弟子极多，大多都是武帝境的弟子，也有少量的武尊境的弟子。

<br/>　　“苏莫，出招吧！”傅同凝视着苏莫，沉声说道。
<br/>　　“我若出招，你可就没有出招的机会了！”苏莫微笑着说道。
<br/>　　“狂妄！”傅同冷笑一声，认为苏莫完全是自大，就算你战力再逆天又能怎样？他可不是那些武帝境武者。
<br/>　　“那我出招了？”苏莫问道。
<br/>　　“废话少说！”傅同冷喝一声，手中光芒一闪，出现了一柄雪亮的长刀，他身上立刻气息暴涨，滚滚玄力涌入长刀之中，让得长刀光芒璀璨。
<br/>　　傅同没有小看苏莫，毕竟，能以武帝境八重的修为，横扫六大势力的妖孽，苏莫的实力便可见一般。
<br/>　　而现在，苏莫已经是武帝境九重的修为，他不敢有丝毫的大意。
<br/>　　“那你接招吧！”
<br/>　　苏莫点了点头，随即身形缓缓拔高，身上的玄力滚荡不休，如同瀚海沸腾。
<br/>　　一股无上的剑威，从他的身上升腾而起，如同滚滚狼烟，撕裂天穹，威临九霄。
<br/>　　傅同见此，顿时面色一变，虽然他知道苏莫不弱，但此刻感受到苏莫的气势，还是大感震惊。
<br/>　　一名武帝境九重武者，能有这种气势，的确不凡。
<br/>　　不过，他却是并未惧意。
<br/>　　意念一动，傅同的背后虚影升腾，一柄巨大的长刀战魂浮现，刀威弥漫八方。
<br/>　　“斩！”
<br/>　　一声轻喝声响起，只见飞临上空的苏莫，骤然出手了，手中剑光一闪，一剑笔直的刺出。

<br/>　　刹那之间，一道犀利无匹的剑气从天而降，仿佛是一支洞穿虚无的利箭，裹挟无可匹敌的锋芒，射向傅同。

<br/>　　“什么？”傅同顿时面色大变，所谓强者一出手，便知有没有，从这一剑之中，他便能感觉到苏莫的强大。
<br/>　　“给我碎！”电光火石之间，傅同手中长刀一挥，狠狠的劈向迎面而来的剑气。
<br/>　　这一刀势大力沉，威猛霸道，刺目的刀光撕开空间，狠狠的斩中了剑气。
<br/>　　轰！
<br/>　　一声爆响，震动四野，猛烈的劲气炸开，逆冲九天。
<br/>　　“啊！”
<br/>　　只听一声惨叫声响起，便见傅同高大的身躯，直接被劈落了下去，如同坠落的陨石一般，飞速的向太阴星而去。
<br/>　　转瞬之间，傅同的身影便穿过了九天罡风层，落向太阴星的大地。

<br/>　　稍倾之后，傅同砸落在了大地之上，即便相隔极远，都能听到一声轻微的轰响。
<br/>　　所有见到此幕的人都是一呆，忍不住心中狂震，仅仅是一招，武尊境的傅同居然就被击败了！
<br/>　　恐怖！

<br/>　　偶尔有些去过太阴族参加会武的弟子，亲眼见过苏莫的神威，还勉强不是太震惊，那些没有见识过的人，虽然知道苏莫很厉害，但此刻亲眼所见，还是免不了心中狂震。
