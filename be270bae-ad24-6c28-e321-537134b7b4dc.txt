　　两位老者相视一眼，捋了捋白花花的胡子，道：“我们真是井底之蛙，慕小友年纪轻轻就有如此成就，前途当真前途无限啊。”
<br/>　　听他们对慕清霄称赞，小医仙心中也极为开心，毕竟任何一名女子，恐怕都希望自己的男人优秀。
<br/>　　随着两人的安静，炼药房中变得平静起来，三道各有不同的视线停留在慕清霄身上，观察他炼药的细节。
<br/>　　慕清霄盘坐在药鼎前，心思都放在药鼎中。
<br/>　　轻松的掌控着九霄琉璃焱，控制着一株株药材送进药鼎中，药材遇到神火，其中的精纯能量瞬间就被提炼出来，速度快的令人乍舌。
<br/>　　最重要的是，这些能量中不参一丝杂质，仅仅是这一点，七成的炼药师都难以做到，起步如此高，炼出来的丹药品质绝对有所保障。
<br/>　　几分钟过去，慕清霄一直在提炼药材中的能量，两位老者皆是被他恐怖的控制力给震惊到了，饶是他们都远远不如，纷纷唏嘘不已。
<br/>　　此时，慕清霄神骏的面容上依旧古井无波，因为九霄琉璃焱，腾射出几丝暗黑色光线，使他的气质也显得妖异绝伦，一瞧便知道他是一位经验老道的高级炼药师。
<br/>　　四品聚气散的炼制过程并不长，最关键的还是控制力，而且必须掐准时间点，稍有差池，聚气散的功效都会大打折扣。
<br/>　　时间过的很快，转眼就过去一个多时辰，在这一个多时辰里，两位老者的脸色可谓是千变万化，都说炼药枯燥乏味，但两人丝毫不这样觉得，反而兴致勃勃。
<br/>　　小医仙则端坐在一旁，静若处子，美眸中温柔尽显。
<br/>　　炼药步骤不知凡几，最终药鼎中的火焰熄灭，慕清霄取出容器，轻轻一卷，药鼎中就飞出几颗碧绿通透，龙眼大小的药丸，诱人的清香席卷整个炼药房。
<br/>　　两位老者激动的站起身来，仅仅是这股香味，就能辨别出丹药的品质，绝对是极品！
<br/>　　在斗气大陆上，想要成为一名真正的斗者，前提便是必须在体内凝聚斗之气旋。
<br/>　　而斗之气旋，却是有着不小的失败率，失败过后，九段斗之气便将降回八段，有些运气不好的，说不定需要凝聚十多次，方才有可能成功。
<br/>　　而如此重复的凝聚，却让使得他们失去了最好的修炼时间段，从而导致前途大损。
<br/>　　而聚气散的功效，便是能够让一位九段斗之气，百分百的成功凝聚斗之气旋。
<br/>　　这种效果，使得无数想要今早成为斗者修炼者，都对其垂涎不已，日思夜想而不可得。
<br/>　　将聚气散收进容器中，慕清霄站起身来，只见小医仙率先走了过来，两位老者则紧随其后的跟了上来。
<br/>　　缓缓走到石台前，会长弗兰克深深的看了一眼慕清霄，语气中参杂着一丝若有若无的敬畏，道：“慕小友，请问你刚才所控，可是异火？”
<br/>　　慕清霄眼底闪过一丝古怪，区区异火，如何与自己的本命神火相提并论，但他也不想多做解释，于是微微点头，将装有聚气散的容器递了过去。
<br/>　　弗兰克小心翼翼的接到手中，将玉瓶中的聚气散倾倒而出，顿时几枚碧绿通透，龙眼大小的药丸就滚了出来。
<br/>　　若是仔细观察，就能看出内部有几圈绿色波纹，犹如扩散的涟漪，由于丹药本就是碧绿，所以看上去并不明显。
<br/>　　“丹纹，果真是极品聚气散，药材中的精纯能量估计全都提炼了出来，否则达不到如此效果，恭喜慕小友，你通过了四品炼药师的考核。”
<br/>　　将聚气散重新装进玉瓶中，弗兰克有些惊叹的说道。
<br/>　　弗兰克与另一位老者实在不知道该如何称赞，虽说是后辈，但他的炼药术却是比他们高上好几筹。
<br/>　　“慕小友，你先稍等片刻，我这就让员工给你准备炼药师的特制服饰，以及四品炼药师徽章。”
<br/>　　闻言，慕清霄摆摆手，道：“特制服饰就算了，穿这套蛮好的，将等级徽章给我就行了，另外，我的事情请不要宣传伸张。”
<br/>　　炼药师徽章的效果确实不错，但声名远播却不是他想要的结果。
<br/>　　“呃！”弗兰克老脸上的肌肉狠狠的抽了抽，不知道该说什么好。
<br/>　　人家成为炼药师，巴不得名声传的越快越好，哪怕是一品炼药师也是如此，可面前的青年身为四品炼药师，却要求保密，他真是第一次碰见这种事情。
<br/>　　“既然是慕小友的要求，我们自然会全力满足。”
<br/>　　片刻后，公会员工恭恭敬敬的送来一枚徽章，徽章呈药鼎形状，其上精心绘制着四条银色波纹，正是四品炼药师尊贵的象征。
<br/>　　慕清霄将徽章拿在手中看了一眼，倒是与两位老者的一模一样，依旧是面无表情，没有戴在服饰上，反而是装进了储存空间。
<br/>　　慕清霄的表现，不知道让两人说什么好，只好面带微笑，道：“慕小友，若是有时间，我想我们或许可以探讨一些炼药的经验。”
<br/>　　闻言，慕清霄剑眉一皱，他接下来还要去塔戈尔大沙漠，根本没时间在黑岩城停留，于是微笑一声，道：“最近我有要事在身，不便在黑岩城多做停留，还请两位见谅，待我何时有空，在前来一叙，你们看如何？”
<br/>　　两人顿时觉得有些可惜，如此妖孽，再过不久恐怕在炼药上的造诣需要他们仰望了，等他有空，也不知道需要等到猴年马月，但人家都如此说了，他们也不敢强求。
<br/>　　“如此甚好，甚好。”
<br/>　　“就此别过，仙儿我们走吧。”说罢，慕清霄就拉着小医仙向炼药师公会外走去。
<br/>　　待慕清霄消失在视线中，弗兰克唏嘘道：“刚才慕小友所填写的信息中，导师名叫药老，可在加玛帝国中，我却没有听闻过药老名讳，真想上门拜见一番啊。”
<br/>　　“能教导出如此妖孽的弟子，根本不屑与我们接触，是不是加玛帝国的炼药师都还是两说。”
<br/>　　“唉，等回去以后，得打压一下门下弟子的傲气，跟慕小友比起来，区区一品炼药师真算不得什么。”
