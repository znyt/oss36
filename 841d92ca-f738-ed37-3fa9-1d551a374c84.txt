　　“下一条是火龙，怎么办？要团么。”看着下一条小龙的属性，夜雪狂歌开口问道。

<br/>　　他们这个阵容在大嘴没有装备之前，就算是去团也打不出什么伤害，那样的话就算是他们的盾还有加血技能多，恐怕也拼不过对方。
<br/>　　而且时间才这么久，他们几个的装备也都没起来呢，这个时候无论是露露的盾还是扇子妈的盾，亦或者是奶妈的w技能加血还是有些少的，很容易被对方一波爆发直接带走。

<br/>　　“下一条是火龙，这元素龙对于白队这边太不友好了，蓝队这边如果拿下这条火龙的话，后期团战的胜率会增加一些的，毕竟火龙越到了后期加层是越厉害的。”林雅楠开口说道。
<br/>　　“是的，不过这一条火龙白队这边只能让了，他们这边只有一个输出核心大嘴，最起码他们也要等到大嘴有着一个破败、到达九级的时候才可以打团，要不然的话大嘴根本打不出多少伤害的。”米粒跟着开口。

<br/>　　每次看到有残血小兵的时候，他都会走上前来，先利用自己的e技能沉默对手，然后在用q技能清兵。
<br/>　　而且对方是将兵线向着自家的塔下推的，所以香蕉这边虽然补刀比之楚南要落后一些，但是也没有落后太多，时间到达10分钟的时候，他这边已经是补了79刀。
<br/>　　楚南那边的诺克则是补了88刀，双方仅仅只有9刀的差距，这已经是香蕉这边可以接受的范围了。

<br/>　　他知道，夜雪现在已经将他的动向给掌握了，弄不好自己两边的野区都有着对方的视野，要不然的话怎么可能每次他都能够那么准时的出现呢。
<br/>　　“这一把夜雪玩的是真的好啊，到现在场上都没有爆发人头，夜雪一个人要占了8层的功劳。”看着再一次忌惮夜雪狂歌而没敢有所动作的飞刀，米粒连忙开口说道。

<br/>　　“白队的这个阵容真的没法应对啊，要针对就只能针对香蕉这一个点，其他几路根本没机会，中路的露露有着大招，下路又是双人，还有虚弱本身就很难杀，加上香蕉的这个大招，难度更是加大了无数。”米粒接过话继续说道。
<br/>　　“这一局蓝队这边只能在小龙和大龙上做文章了，如果在30分钟之前他们找机会拿一波大龙，然后破掉白队这边一路高地后，他们的获胜机会就非常的大了。”林雅楠开口分析着。
<br/>　　“蓝队这个阵容推塔速度还是很快的，他们现在一定要利用好女警这个点，快速推塔。”

<br/>　　在拿到小龙之后，蓝队这边三人直接转到下路，白队下路这边的一塔本就已经被女警消耗掉了三分之二左右的血量，在三人的包夹下，自然是毫不费力的就推了下来。
