　　第656章
<br/>　　一路奔波，归来不易。程大雷回到蛤蟆城时，刘发财正坐在堂上高谈阔论，述说此行的凶险，他本有惑众的属性，添油加醋这么一说，唬得周围众人一愣一愣的。
<br/>　　见到程大雷进来，刘发财才忙站起来，止住满口的吐沫。
<br/>　　“回来了。”程大雷摆摆手让大家坐下，如今大厅内挤满了人，倒也不是每个人都有座位。程大雷在帅案后坐下，以目光询问刘发财：“路上怎样？”
<br/>　　刘发财擦了擦嘴角的吐沫，开始再次添油加醋，述说这一路上的不易。从凉州到徐州，相当于横穿帝国大陆，再加上如今又兵荒马乱。去的时候还容易，至少有黑衣老尼护送，难的是回来的时候。
<br/>　　这次不仅刘发财回来了，还寻到华氏一族，带着华家上百人来到蛤蟆城。
<br/>　　如今华正则正抱着自己的族人痛哭流涕，虽然华氏一族躲过这次东海之乱，保全了大部分族人，但还是有少部分族人在这次战乱中遭遇不幸。
<br/>　　其中，便包括华正则的那位青梅竹马。
<br/>　　乱世之下，不死人是不可能的，既然会有死人的事情，那便没有理由只死其他人，而不死自己身边的人。华正则曾经安稳度日的念头，只能是个春秋大梦。
<br/>　　程大雷见到了华氏一族的族长，年纪竟然不是甚大，不是程大雷想象中胡须发白的老者。不过是一个年逾四十的中年男人。
<br/>　　其人名叫华正怒，为人平和得很。华家不愧是悬壶济世传承诸多年的家族，对于山贼也没有歧视。如柳芷这种小家族，根本不能于其相提并论。
<br/>　　华正怒冲程大雷双手抱拳：“多谢程当家出手相助，华某代表族人感谢程当家的大恩大德。”
<br/>　　“言重了，言重了。”程大雷握住了华正怒的双手：“大家都是一家人，一路颠沛肯定是吃了苦头，如今到了这里就像到家一样，杨国忠，快带诸位休息。”
<br/>　　华正怒还是不太适应程大雷的热情，他不动声色的挣开双手，道：“多谢程当家。”
<br/>　　杨国忠带着华氏一族下去，程大雷兴奋莫名。华氏一族以医术传世，每人的等级都在刘悲之上，如果放出去，随便一个都是江湖上的抢手货。而这次刘发财却为自己带来上百位这样的高手。
<br/>　　等华氏一族退下，大厅内才变得空旷起来。程大雷面向刘发财，继续刚才没聊完的话题：“如此说，刘顶天已经被灭了？”
<br/>　　“是的，被逍遥王李行哉灭了。”
<br/>　　程大雷深吸一口气，手指敲着桌面，多日未见，这李行哉了不得呐。
<br/>　　“后来呢？”
<br/>　　“我当真窝了几日，打探李行哉的情报。”刘发财也是一个机灵人，正好程大雷问起，他表现得很兴奋：“李行哉在琅琊城也待不住，他将城中的居民财物带走，而且，万卷山好像投靠了李行哉。”
<br/>　　“这……”
<br/>　　程大雷不得不陷入沉思，落凤城与琅琊城相距太远，李行哉的确没有统治两城的能力。所以将琅琊城的人口带到落凤城就是理所应当的事情。
<br/>　　因此落凤城的人口扩充，李行哉的实力又进了一步。不过实力坐大，自然而然便引起周围势力的忌惮。
<br/>　　徐州、江南李星、兖州三家怕都容不下李行哉，在群狼环伺之下，李行哉的处境怕是会变得困难。
<br/>　　“大当家，路上还发生了一件奇怪的事情？”
<br/>　　“喔，还有其他情报？”程大雷不得不对刘发财刮目相看，他今天给自己带来惊喜的人太多了：“什么情报？”
<br/>　　“这个……那个……”刘发财吞吞吐吐，最后还是把话说出口：“现在似乎整个帝国都在骂大当家？”
<br/>　　“啊？”程大雷一激灵：“我最近可是没出去做坏事呐？”
<br/>　　“真的没有么？”刘发财下意识道了一声，又立刻道：“他们当然是不了解大当家，否则也不会胡说八道，但现在整个帝国都在胡说八道。”
<br/>　　刘发财述说一路见闻，东海一地自然不必说，他一路走过来，帝国各地都在述说程大雷的不是。倒也都是程大雷以前做过的事，现在却被大家都想了起来。
<br/>　　读书人对程大雷口诛笔伐，江湖人要去刺杀程大雷，帝国朝廷已经将程大雷列为第一要犯。
<br/>　　当然，这种状况持续的时间并不长，等刘发财快到凉州时已不太能听见。
<br/>　　程大雷依旧是稀里糊涂，心里琢磨不明白这是怎么会事。他忽然一楞：该不会是……
<br/>　　程大雷渐渐有些明白过来，那广播技能该当是生效了。这技能是在帝国范围内扩大自己的名声，而程大雷本身的名声又不好，于是也就加倍恶化。
<br/>　　不过，程大雷也是虱子多了不咬，债多了不愁，名声糟糕就让它糟糕去吧。
<br/>　　因此，自己还获得了大量恐惧值，如此，倒也不算有坏处。
<br/>　　刘发财一路过来不容易，交待完诸多事情后，程大雷嘱咐他好好休息。之后，程大雷换了套衣服，来到城主府后院。
<br/>　　一个粉雕玉砌的女孩，持着花扇在院子里扑蝴蝶。程大雷忙将自己的首领光环打开，能刷一波好感值是一波。
<br/>　　见到程大雷过来，杨小妹变得端庄起来，深施一礼道：“大当家。”
<br/>　　程大雷有些无语，这杨小妹也不知如何教育出来的。不见人还好，见人一板一眼，弄得程大雷也不得不正经起来。
<br/>　　他在亭中坐了，轻咳一声坐直了身板：“这些日子在城中如何，可有不适的之处？”
<br/>　　“一粥一饭都是大当家的恩情，如果不是大当家，我们兄妹怕早已遭了山贼的毒手。”
<br/>　　这些话说得那里都对，可程大雷却总觉得有些不是滋味，话里话外带着一种距离感。
<br/>　　程大雷也没什么办法，待首领光环的有效时间过后，程大雷便去后院找书苏樱。
<br/>　　他在朔方城耽搁的时间太长，倒是没有多少时间留在蛤蟆城。
<br/>　　待他走后，杨小妹仍拿着花扇扑打蝴蝶，一身粉裙翩翩若花舞，看上去纯真浪漫，纤尘不染。
<br/>(本章完)
