
<br/>　　<br/>　　    两天后，王惊蛰从原始社会回归到正常世界，找到个有人烟的地方之后，他就赶紧把电话先拿了出来，之前一直在山里面，手机信号始终都处于游离的状态，他和外界根本就没联系过，没人找他也打不进来，手机完全就成了个摆设。<br/>　　<br/>　　    电话开机，一堆提示短信就进来了，他粗略的翻看了几眼后发现，这些电话号码一半是陈臣打过来的，另外一半是菜刀文的，期间还夹着几条短信。<br/>　　<br/>　　    陈臣：“你死了么？怎么一直都没消息……”<br/>　　<br/>　　    王惊蛰搓了搓在山林里折腾的饱含风霜的脸蛋子说道：“哥都火成这样了，那株小草应该好好珍惜吧，再不下手小爷就被别人给吃了”<br/>　　<br/>　　    其实王惊蛰也品出来了，陈臣对他也是很有企图的！<br/>　　<br/>　　    没着急给陈臣回电话，王惊蛰先给菜刀文打了一个：“找我？”<br/>　　<br/>　　    “废话，你都要成失踪人口了，我这能不着急么？找不到你，我怕王冬至都能把我给撕了，你总算是有信了”听见王惊蛰的动静，菜刀文也宽心了，这位爷他要是伺候不好，他就得准备给自己安排安排后事了。<br/>　　<br/>　　    “最近在山里当野人来着，电话没有信号，说吧，找我都什么事？”王惊蛰一边打电话，一边进了家土菜馆让老板上了几瓶啤酒和几盘肉菜，这段时间他不光在身体上憋的够呛，胃口也挺遭罪的。<br/>　　<br/>　　    “两个事哈，丁武那边已经彻底没问题了，从ICU出来住进普通病房了，也没有什么后遗症，再过一月又是一条生龙活虎的好汉”<br/>　　<br/>　　    王惊蛰顿时松了口气，对于丁武他一直都是挺惦记的，人能没事那就最好了。<br/>　　<br/>　　    “另外一个是阎王坟场那块地皮的事”菜刀文接着说道：“我跟午桥开始观察了一段时间，地皮到手之后他们就已经开始操作动用了，没过多久地就被圈建了起来，上面盖了两栋厂房，后来午桥让陈重找了个机会进去看过了，厂房下面全部都被挖坑了，挖的很深，就跟他么要挖墓似的，现在已经过去一个多月了吧，我估计下面该要成型了，但我们也就能去看这一次而已，没办法接着在夜探紫禁城了，至于里面到底要干什么，暂时还不知道”<br/>　　<br/>　　    “我差不多已经猜到他们的意思了，你先不用管了，让他们干着吧”王惊蛰眯着眼睛说道。<br/>　　<br/>　　    “啊？不管了？”<br/>　　<br/>　　    王惊蛰淡淡的说道：“他们要干的事还没有成型，你这时候要是干点什么，顶多就算是给人捣乱，远远达不到一出手就让他们心疼的地步，你觉得啥时候打他一棒子是最疼的呢？”<br/>　　<br/>　　    菜刀文想了想，一针见血的说道：“一栋楼，挖完地基建起来没几层的时候要是倒了，那损失不大，可等楼建完了之后，你再给推倒了，那能给人疼死！”<br/>　　<br/>　　    王惊蛰点头说道：“所以，现在动手还早，等他们张灯结彩万事大吉了之后，咱再来一把狠的，他们是不就得吐血了？”<br/>　　<br/>　　    菜刀文叹了口气，说道：“何止吐血啊，肠子都能吐出来了！”<br/>　　<br/>　　    “所以，那就等着吧”<br/>　　<br/>　　    “小草那边……”<br/>　　<br/>　　    王惊蛰笑了，说道：“虚惊一场啊！”<br/>　　<br/>　　    两人闲聊了几句就把电话给挂断了，王惊蛰独自一人喝着啤酒吃着肉，一时间感觉无比的畅快，没有啥能比放下身上所有的担子，更令人舒坦的了。<br/>　　<br/>　　    而显然的是，王惊蛰此时就觉得，小草就是他心里最重的担子，至于别的他还没有想起来！<br/>　　<br/>　　    另外一头，这天夜里，川南那处工地外，开过来一辆车，随后从车里下来吴洋和周深，还有那个带着斗篷看不见脸的男子。<br/>　　<br/>　　    “嘎吱”吴洋伸手推开工地大门，这里白天施工，晚上连个打更的人都没有，里面是空无一人的。<br/>　　<br/>　　    吴洋交代过施工经理，告诉他一到晚上天黑之后就把所有的工人全都给撤出去，然后第二天再赶过来施工，这个要求其实挺不合常理的，因为一个工地基本都是连夜施工的然后赶着工期快点完事，特别像这种四周都没人烟的地方，就是晚上施工也不存在扰民的状况，所以更应该二十四小时两班倒的干活了。<br/>　　<br/>　　    但吴洋偏偏就是不让，说一到天黑了就得让工人出去，施工方也诧异，见过赶工期，没见过硬拖着的。<br/>　　<br/>　　    八月份的蜀地，白天的话气温得有三十多度了，到了夜晚也得二十度上下，但这工地的大门一开，瞬间就从里面冒出一股凉风，吹得人都忍不住的起了一层鸡皮疙瘩工地大门推开之后，吴洋侧过身子，周深说道：“师叔，里面差不多干完三分之二了，再有半个多月就能完事了，你先看看构造合不合你的心意，要是有点偏差的话，我们现在改还来得及”<br/>　　<br/>　　    吴洋打开车后备箱跟周深从里面一人提出两个长条的袋子来耷拉在地上，看起来里面的东西还颇有份量，挺重的。<br/>　　<br/>　　    戴着斗篷的男子背着手嗯了一声迈步就走进了工地中，片刻后三人进了其中一栋厂房里，这里被挖开的地下已经跟午桥进来那时有很大的不同了。<br/>　　<br/>　　    午桥之前就说过，这好像是在挖坟地似的，他这么说还真有点贴边，下面不是坟地但却跟地宫差不多，将近十米深的底下全被隔开了，就像一个一个的单间似的。<br/>　　<br/>　　    斗篷男子看了眼周深，他和吴洋拎着袋子就跳进了坑里，然后“呲啦”一声拉开之后，四个袋子中装的居然是一具具尸体，这个天气尸体还保持的挺新鲜的，看起来就跟刚死了不久一样。<br/>　　<br/>　　    “噗嗤”<br/>　　<br/>　　    “噗嗤”<br/>　　<br/>　　    吴洋和周深同时拿出一把尖刀，一刀就跺在了尸体的手腕上，顿时鲜血就喷溅出来，但由于人死了之后血液是不会流动的，周深和吴洋就开始用手揉搓着尸体，来促进体内的血从伤口涌出来。<br/>　　<br/>　　    但很诡异的是，从尸体里流出的鲜血，一落到地上就迅速的渗透而下，就跟洒在了海绵上一样，到后来根本都不用他俩在挤压了，尸体中的血还在源源不断的往出涌着，就好像是地下有什么东西在吸吮着一样。<br/>　　<br/>　　    不到十分钟，四具没死多久的尸体，就被生生的抽成了一具具的干尸，只剩下了皮包着骨头，看起来异常的恐怖。
