　　元始一念杀人后，通天也醒了过来，看他满脸笑容，想必是已有所突破。
<br/>　　“大兄，我们切磋一番，如何？”
<br/>　　这是通天醒来后说的第一句话，说完后，他眼中闪过炙热的目光，盯着元始不放。
<br/>　　元始笑了笑，道了一个字：
<br/>　　“好！”
<br/>　　演武场中，通天手持青萍剑，青色剑光闪烁，如同天上的太阳一般耀眼，凌厉的剑气纵横四面八方，割碎空间，但到了元始周边三丈之外，便不得寸进。
<br/>　　“接我一招，青萍耀天！”
<br/>　　通天的剑光化作九个青色的烈日，煌煌大日，璀璨夺目。
<br/>　　青色大日连成一串，化作一道巨剑，剑光辉煌，剑气逼人，大剑划破虚空，斩向元始。
<br/>　　元始看了一眼，随意抬起一只手，对着青色巨剑便是一拳轰去。
<br/>　　拳头朴实无华，看似随意，实则蕴含的力道极大。元始随便一拳，便可化为神通，拳术通神，不外如是。
<br/>　　剑光与拳头相撞，想象之中的爆炸声并没有出现，巨剑被元始一拳崩飞，化作一个个青色拳头飞向通天。
<br/>　　以彼之道还施彼身！
<br/>　　通天见此，毫不犹疑，一剑划出，九个拳头瞬间便被斩碎落地。
<br/>　　“再来一招，剑网恢恢！”
<br/>　　通天接连挥剑，剑光汇聚而来，形成一道巨大的剑网，笼罩元始。
<br/>　　剑网之上，剑光森寒，剑气逼人，若是一般大罗金仙碰上，瞬间便会被斩破灭杀。
<br/>　　元始站在原地，四周都被剑网尽皆包围，让他无处可躲，若是一般人见此，只能眼睁睁地看着剑网将自己慢慢绞杀，成为碎片。但，元始不会。
<br/>　　只见元始抬手便是一道惊天巨掌，拍向剑网，剑网与巨手相碰，顿时被泯灭，翻不起点滴浪花。
<br/>　　“好了，不玩了。”元始轻声道。
<br/>　　元始一指点出，瞬间，一道光产生，两道，三道……
<br/>　　顿时，无量光汇聚而来，形成一个光点，点向通天。
<br/>　　“不好。”
<br/>　　通天暗叫一声，光点太快，来不及闪躲，他急忙抬起手中青萍剑抵挡。
<br/>　　“叮！”
<br/>　　一声轻响，光点与剑身相撞，剑身悲鸣，巨大的力道将通天撞得倒飞数里之远。光点破碎，化作无量光辉，绕过青萍剑，射向通天。
<br/>　　光芒之中蕴含光明法则，太过耀眼，使得通天的眼睛情不自禁地闭上。
<br/>　　圣光带着强烈的穿透力，透体而过，消散于虚空之中，好似从来没有出现过。
<br/>　　“噗——”
<br/>　　通天一口鲜血喷出，脸色瞬间变得苍白无比。
<br/>　　“怎么样？”元始一步踏出，来到通天面前，轻声笑道。
<br/>　　“大兄还是那么狠，都是自家兄弟，怎么不肯手下留情。”通天翻了翻白眼，没好气道。
<br/>　　“到了外面争斗，可没人会对你手下留情。此时多吃些苦，以后少受点伤。”元始语重心长道。
<br/>　　“呵呵。”
<br/>　　通天摆了摆手，表示自己很无语，这句话他已经听了不知多少遍了，每一次切磋之后，他都要听上这么一句话。
<br/>　　“呵呵！”
<br/>　　元始亦是呵呵一笑，他知道通天就是这个样子，也没有放在心上。
<br/>　　“好了，以后多加修炼，争取早日超过我，再来找我比试。不过，平时你要来，我亦是非常欢迎。”
<br/>　　元始拍了拍通天的肩膀，安慰道。这孩子，被打击得已经够多了，没必要再打击他了。
<br/>　　通天：“……”
<br/>　　你这还不是打击我？通天翻了翻白眼，回去养伤去了，他决定，下次再来找大兄切磋。他立志，终有那么一天，要打败大兄，将他也打得鼻青脸肿，让他也尝尝我此时心中的滋味。
<br/>　　至于这个目标能不能实现，通天也没有想那么多。
<br/>　　就这样，三清再次陷入闭关修炼之中，昆仑山顿时变得安静无比。
<br/>　　而洪荒之中，此时却不是那么平静。
<br/>　　随着时间的流逝，转瞬间，洪荒已过数万年。
<br/>　　太阳星上的两只三足金乌，仗着修为高深，使得一手太阳真火，再加上生来就有先天至宝混沌钟与先天灵宝河图洛书之利，当世洪荒之中，能够与他们匹敌者着实很少。
<br/>　　有了实力，自然就会有野心。于是，两兄弟在经过一番历练之后，纠结了大批的草木精怪，魑魅魍魉，成立了妖族。他们广纳生灵，不论是动物还是植物，凡能开窍修行者，皆可加入妖族！
<br/>　　两只三足金乌自称妖皇。
<br/>　　就这样，妖族势力空前壮大，洪荒之中，鲜有实力与之对抗。
<br/>　　此时洪荒正处与远古与上古时代的交替的阶段，他们借助大势，妖族很快就发展壮大，除开了两大妖皇之外，麾下更是汇聚了计蒙、英召、飞廉、九婴、商羊、钦原、白泽、毕方、穷奇、诸犍十大妖圣，又有大妖如飞诞、呲铁、鬼车、饕鬄等数十名。
<br/>　　一时之间，妖族呈现大治局面，天地万妖齐心协力，实力变得空前强大。
<br/>　　在妖族崛起的同时，洪荒大地之上，另一个庞大的种族亦开始迅速崛起，这一种族名曰：巫！
<br/>　　巫！
<br/>　　巫者，上顶苍天，下踏大地，中间两人，乃顶天立地之人，盘古神尊的后裔，为首十二魔神，帝江、烛九阴、奢比尸、蓐收、句芒、共工、祝融、天吴、强良、弇兹、玄冥、后土，号称十二祖巫。
<br/>　　十二祖巫，乃盘古肉身及精血融合地脉先天煞气所化，天生肉身强横无比，能吞天噬地，亦能操控风水雷电，移山填海，端是厉害。
<br/>　　十二祖巫之下，亦有诸多大能，诸如刑天、九凤等，具皆非凡，拥有强大无匹的力量，等闲修行之辈，难以与之匹敌。
<br/>　　随着时间的推移，巫妖二族越渐壮大，继而成了当今洪荒大地的掌控者，一如远古时代的神龙、凤凰、麒麟三族。
<br/>　　一时间，两族风头正盛，大能隐世不出，他们在洪荒之中闹腾的欢。
<br/>　　一山岂能容二虎？两族之间的壮大，少不了发生各种争斗、摩擦事故，少不了利益的冲突，故而，二族之中，亦是战斗不休。
<br/>　　当然，洪荒之中除了这两族之外，也有其余散修大能之辈，广纳门徒，招收势力，只是，他们并没有此二族这般声势浩大罢了。
<br/>　　如，万寿山五庄观，有一名为镇元子大仙的大神，会合红云老祖，两者招收了一批散仙，聚在万寿山，形成一股势力。再如南海群岛之上，东王公亦聚集了一批散仙，形成一方势力。
<br/>　　妖族之中，一些妖族不想加入到争斗之中，来到不周山，寻找女娲伏羲二兄妹的庇护，就这样，他们也形成了一方不小势力。北冥之中，鲲鹏收了周边小妖，形成北冥宫一方势力。
<br/>　　……
<br/>　　就这样，洪荒打破过往的平静，如同大海的平静被打破，开始变得汹涌澎湃。
