这时她的脚步突然一顿，因为前方出现了一道沉稳挺括的身影，陆司爵。
<br/>
<br/>        陆司爵已经在等她了。
<br/>
<br/>        夏夕绾所有的思绪瞬间回笼，她毫无意外的看着陆司爵，纤柔的美背挺直，娇美的菱唇勾出一道浅笑，“陆伯父，你在等我？”
<br/>
<br/>        陆司爵越过她的肩膀看了一眼后面的洗手间，不动声色的问道，“阿霆怎么样了？”
<br/>
<br/>        他知道她跟陆寒霆在里面。
<br/>
<br/>        “我小小的教训了他一下，他应该没事。”
<br/>
<br/>        陆司爵将幽沉的眸子落在了夏夕绾的小脸上，女孩那双澄亮剔透的翦瞳正碎亮的看着他，坦然自若，陆司爵嗓音低沉道，“夕绾，你现在是子羡的未婚妻，我希望你能妥善处理好和阿霆之间的关系。”
<br/>
<br/>        “陆伯父，你只想跟我说这些吗，我以为陆伯父的性格是喜欢掌控一切，那些不确定的因素都会扼杀在萌芽里，我重回帝都，现在又夹在你两个儿子中间，你应该让我消失才对，可是你竟然转性了，像个慈父一样包容了我的一切，我真的受宠若惊，诚惶诚恐的，这样吧，我们明人不说暗话，为什么，是因为我妈咪林水瑶吗？”
<br/>
<br/>        陆司爵省视着对面的女孩，其实早在海城，他就应该想到她是林水瑶的女儿！
<br/>
<br/>        只有林水瑶的女儿，才会这样光芒万丈，璀璨而耀眼。
<br/>
<br/>        陆司爵点头，“是的，是因为你的妈咪林水瑶，这些年我倒没有找你，既然当年林水瑶携女离开，就不会让人找到的，但是我知道，早晚有一天你会回来的，林水瑶的女儿会回来帝都的。”
<br/>
<br/>        夏夕绾双眼发亮，灿若星辰，她上前一步，“你知道我妈咪什么，你知道我妈咪来自于哪里吗？”
<br/>
<br/>        陆司爵点头，“知道。”
<br/>
<br/>        夏夕绾又上前了一步，“那你知道我妈咪现在在哪里吗？”
<br/>
<br/>        陆司爵又点头，“知道。”
<br/>
<br/>        夏夕绾又上前了一步，“那你知道我亲生爹地是谁吗？”
<br/>
<br/>        陆司爵还是点头，“大概能猜到。”
<br/>
<br/>        夏夕绾停下了脚步，停在了陆司爵的面前，“但是，你不会告诉我的，对吗？”
<br/>
<br/>        “对。”
<br/>
<br/>        夏夕绾就知道，陆司爵不会告诉她的，她想了一下，“那你想从我的身上得到什么？陆伯父，我们可以等价交换，我给你你想要的，你将你知道的一切都告诉我。”
<br/>
<br/>        陆司爵点头，“好，我想要从你身上得知…柳璎珞现在在哪里。”
<br/>
<br/>        什么？
<br/>
<br/>        夏夕绾一顿，陆寒霆的妈咪柳璎珞不是在十几年前已经从阿娇房的高台上一跃跳入大海里，死了吗？
<br/>
<br/>        “我想要柳璎珞，如果你帮我找到柳璎珞，我就把我知道的一切告诉你，当然了，”陆司爵意味深长的看了夏夕绾一眼，“你也有你的价值，通过你，我也会找到柳璎珞的。”
<br/>
<br/>        说完，陆司爵离开。
<br/>
<br/>        夏夕绾站在原地看着陆司爵消失的身影，一个五十多岁的中年男人，时光已经磨平了他身上所有锋锐的棱角，曾经的一代商界帝王被赋予了岁月的厚重与神秘感，没有人知道他在想些什么，夏夕绾耳畔也只剩下他那句我想要柳璎珞。
