她将酒杯递到了他的唇边。
<br/>
<br/>        陆寒霆一手托着她纤柔的腰肢，缓缓将她喂来的红酒给喝了下去。
<br/>
<br/>        他喝酒的时候还拿眼神看着她，夏夕绾俏脸更红，总觉得自己跟他做了什么见不得人的事情。
<br/>
<br/>        这时霍西泽鼓掌道，“好，这酒我二哥是喝下了，但是这三千万怎么算啊？”
<br/>
<br/>        霍西泽带头一起哄，众人纷纷附和，
<br/>
<br/>        就是啊小美人，这三千万可不是小数目，不然你让陆总帮你还了？
<br/>
<br/>        虽然陆总随便玩一玩都不止三千万这个数，但是陆总也不能白白花了这三千万啊，所以小美人，你也要拿出什么东西跟陆总做交易。
<br/>
<br/>        霍西泽笑道，“我们1949的陪酒小姐可都是有价格单的，什么价位对应什么服务，来啊，把价格单拿出来看看。”
<br/>
<br/>        迅速有人将价格单送到了夏夕绾的面前，这三千万已经够买1949当红头牌一个晚上了。
<br/>
<br/>        夏夕绾迅速将价格单合上了，她感觉自己上了贼船，这些人都是故意的。
<br/>
<br/>        她想要起身。
<br/>
<br/>        但是陆寒霆扣着她柔软的纤腰不许她乱动，将她霸道的禁锢在自己的怀里，“夏夕绾，我是不是太惯你了，让你在我怀里想来就来，想走就走？”
<br/>
<br/>        夏夕绾抬眸看着他，“要不然呢，今晚我是跟你在这里现场表演，还是跟你回酒店？”
<br/>
<br/>        陆寒霆将薄唇勾出了一道邪佞的弧线，“你选择，我都可以。”
<br/>
<br/>        “你！”
<br/>
<br/>        陆寒霆的目光从她盈亮的翦瞳落在了她的面纱上，“都不选也可以，你把面纱摘了。”
<br/>
<br/>        他想要她摘面纱。
<br/>
<br/>        以前有几次他也想要摘她的面纱，不过她拒绝了，他就没有再勉强，这一次不同了，他一定要她摘面纱。
<br/>
<br/>        夏夕绾看到了他眸里不容置喙般的强势，她不禁勾唇，冷笑道，“陆总确定看一眼我的脸就值三千万，如果我真的貌丑无颜呢，你岂不是亏惨了？”
<br/>
<br/>        陆寒霆挑眉，染上了几分笑意，“我花我的钱我都没心疼，你在心疼什么，你不是不想做我的陆太太了么？”
<br/>
<br/>        夏夕绾语结，然后道，“你想看，那我就摘给你看。”
<br/>
<br/>        她同意摘面纱了。
<br/>
<br/>        陆寒霆看了霍西泽一眼，霍西泽迅速赶人，“走了走了，都出去，不要留在这里碍眼。”
<br/>
<br/>        所有人都被赶走了，霍西泽自己没走，夏夕绾从小就戴着面纱，他也很好奇夏夕绾究竟长什么样？
<br/>
<br/>        是貌丑无颜，还是貌似天仙？
<br/>
<br/>        但是很快陆寒霆就一个眼神杀了过去，霍西泽一溜烟的走了，“我走我走，二哥你看了之后告诉我哈，我先走了，给你们把风。”
<br/>
<br/>        所有人都走了。
<br/>
<br/>        耳畔清净了，夏夕绾抬起纤白的小手，缓缓摘下了自己的面纱。
<br/>
<br/>        陆寒霆第一次看到了她的俏脸。
<br/>
<br/>        两蹙柳叶眉，盈亮夺目的翦瞳，秀琼如管玉般的小鼻翼，下面是唇线无比姣美的菱唇。
<br/>
<br/>        这些精致清丽的五官拼凑在一起，说不出的纤尘绝色。
