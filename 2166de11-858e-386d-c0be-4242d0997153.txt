　　因为，和之前有所不同，这个任务没有在规定时间完成的话，是有很严重惩罚的。
<br/>　　名称：解除封印
<br/>　　内容：解除画像上的封印。
<br/>　　等级：A
<br/>　　时限：离开《僵尸先生》位面之前。
<br/>　　失败惩罚：永久抹除宿主随机一项技能。
<br/>　　目前为止，这是刘哔第一次接到有惩罚的任务，况且他身上的技能，也只有‘吞噬’和‘不死’两个，不管是抹除哪一种，都会对他产生很大的影响。
<br/>　　看着眼前画像上的紫衣女子，刘哔舔了舔嘴唇，有些可惜。
<br/>　　这样一来的话，他就没有办法吞噬画像了，他不可能为了未知的效果，而放弃自己现有的任何技能。
<br/>　　但如果任务时限结束之前，还不能完成这项任务的话，他倒是可以尝试着吞噬一下，用以挽回一些损失。
<br/>　　而刘哔的沉默，在紫衣女子的眼中看来，似乎是默许了这项交易，脸上露出了得意的笑容。
<br/>　　“帮你解除封印可以，但我又怎么知道你出来会不会杀掉我？”心中犹豫了片刻，刘哔紧盯着紫衣女子，缓缓开口：“况且，就算你不杀我，也大可直接跑掉，又怎么保证会如约传授我功法？”
<br/>　　现在的情况，确实让他陷入了两难之中，紫女现在是封印状态，就可以挡下自己全部的攻击，如果放出来，谁知道力量会强大到什么地步。
<br/>　　假设紫女没有害他的心思还好说，可谁知道情况会如何发展？
<br/>　　到时候如果真的出现了意外，他自身的生命安全都无法得到一个保障，技能和奖励固然重要，但又如何和性命相比。
<br/>　　“你帮我解除封印，功法我可以先教给你一半。”紫女眼珠一转，想出了一个折中的办法：“等我出去以后，我在教给你另一半的功法，这总该可以了吧。”
<br/>　　刘哔盯着画像，眼神有些松动，但还是没有开口，他在考虑这件事情的可行性。
<br/>　　“我和你无冤无仇，为什么要害你？”见到没有回应，紫女有些气结：“我只是想让你帮我解除封印而已，一个大男人，怎么这么胆小！”
<br/>　　嗤笑一声，刘哔看着画卷，他心中并不相信紫女的话，但还是决定帮她接触封印了。
<br/>　　没有任何事情是不需要担待风险的，哪怕是机缘，如果一直这样畏手畏脚，那还如何提升自己的实力？
<br/>　　紫女是他的任务，帮她解除封印，不禁能学到功法，还可以获得系统的任务奖励，一箭双雕，在这样的利益驱使之下，也是很值得冒险的。
<br/>　　“怎么解除你的封印？”既然已经下了决定，刘哔也就没再犹豫，直接开口，向紫女询问道。
<br/>　　脸上带着俏皮的笑容，紫女立即回答：“用五颗百年尸丹碾成的粉末，和四两糯米掺杂在一起，然后用黑狗血浸泡四十九天，晒干后洒在上面就可以了。”
<br/>　　说话的时候，紫女的语气颇为轻松，似乎是在叙述一个随手就能完成的事情，但刘哔听完，嘴角却是一抽。
<br/>　　其余的东西倒还好说，但是尸丹是什么东西？
<br/>　　虽然听名字很容易理解，可是在他的印象中，这东西一直没有在原著中出现过。
<br/>　　“额，尸丹……要怎么搞？”不明白的事情，就要去问，他可是个非常谦虚的人。
<br/>　　“你不知道尸丹？”紫女眯着眼睛，脸上满是惊讶：“就是僵尸体内结出的丹丸啊，只有百年以上的走僵才会结出，剧毒无比，需要很特殊的手法才可以收取。”
<br/>　　若有所思的点了点头，刘哔并没有从这件事情上纠缠下去，虽然百年份的走僵肯定不好对付，但想完成的话还是有解决办法的。
<br/>　　况且，这项任务被评为A级，固然存在很大的风险，但完成后的任务奖励，也一定会非常丰厚。
<br/>　　“恩，可以，先把功法传授给我吧。”点头应承下来之后，刘哔随意的开口说道。
<br/>　　功法这种东西，越早获得作用就越大，不管如何也还是要先学到手里再说。
<br/>　　对此，紫女也没有反对，当下从画像中射出一抹紫光，进入了刘哔的身体，速度之快，让他根本无法反应。
<br/>　　刹那之间，刘哔的脑海中多出了很多东西，大量的信息涌入脑海，让他稍稍有些眩晕，不过很快，他就恢复了过来。
<br/>　　先是查看了一下脑海中的功法，一切洞悉之后，他从意识中退了出来，眯起眼睛，盯着手中的画像。
<br/>　　他感受到了危险，虽然一直对画像抱有戒心，但刚才的那道紫光，他还是没有办法反应，如果紫女想要攻击他的话，估计那一下就会让他重伤。
<br/>　　“别这样看着我，我只能传出意识，但根本没有办法对封印之外的人产生攻击。”似乎是明白刘哔心中再想什么，紫女撇了撇嘴，脸上露出一抹无奈。
<br/>　　对方的谨慎，出乎了她的意料，和这样一个家伙合作，她也是感觉有些心累。
<br/>　　移开了目光，刘哔并没多说什么，将画像卷起，不顾紫女的抗议，塞进了自己的腰间。
<br/>　　这边的事情都解决完了，他也该着手准备任府的事情了，明天他就要和九叔一起去帮任家迁坟。
<br/>　　按照原著上的剧情，此时过后没有多久，任发就会被他爹变成的僵尸给咬了。
<br/>　　既然他接取了保护任家的任务，那自然不会让这件事情继续发生下去，只是不知道，任家的那只僵尸，大概是属于什么级别。
<br/>　　心中思索着接下来的计划，刘哔赤裸着上身，从树林中走了出去。
<br/>　　片刻之后，在他原本呆在的大树旁边，猥猥琐琐的显出来两个人影，是之前跟随血迹，一路追查过来的任发和秃头老人。
<br/>　　因为之前一直警惕着紫女，刘哔一时不察，居然没有发现近在咫尺的两个人，一直到他离开之后，他们才从藏身地走了出来。
<br/>　　看着眼前已经烧成一堆的木炭，任发脸色阴沉，之前他们并没有留下刘哔的把握，害怕打草惊蛇，也就没有贸然动手。
<br/>　　事实看来，他们的选择绝对是正确的，走到木炭旁边，秃头老人弯腰，捡起一块破碎的心脏，来到任发旁边，低声说了些什么，惹得任发不住点头。
<br/>　　片刻之后，两人也一同离开了此地，树林之中，恢复了最初的安宁……
