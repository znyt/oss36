　　加入通天阁？
<br/>　　通天阁，可不同于其他门派，他们很少收弟子，其中的强者，大部分都是供奉。
<br/>　　但是这并非代表加入通天阁没有多少好处，恰恰相反，以通天阁的资源，加入其中，比起加入七大宗门都丝毫不差。
<br/>　　在这个时候，通天阁突然来邀请楚风眠加入，也正好是看中了如今夜牧老祖的威胁之下，楚风眠回到北邙学院，必然会受到报复。
<br/>　　这个时候加入通天阁，绝对是千载难逢的机会。
<br/>　　可是下一刻，楚风眠回答，却是让在场所有人都是愣住了。
<br/>　　“多谢前辈好意，楚某既然是北邙学院弟子，自然一直都是，没有离开北邙学院的打算。”
<br/>　　楚风眠平静的开口道。
<br/>　　谁都没有想到，在这个时候，楚风眠居然会选择拒绝。
<br/>　　楚风眠现在杀了夜天君，回到北邙学院，也是危险重重，在这个时候，通天阁抛来橄榄枝，这几乎是在所有人看来，根本不需要去考虑的事，肯定会一口答应下来。
<br/>　　可现在，楚风眠居然是拒绝了。
<br/>　　这样千载难逢的机会，楚风眠居然是一口拒绝。
<br/>　　围观武者不少人都以为楚风眠疯了，在这种时候，还拒绝通天阁的邀请。
<br/>　　留在危险的北邙学院，跟加入到通天阁去受到保护，这样的选择，楚风眠选择了第一点。
<br/>　　听到这一声声的议论，楚风眠心中倒是极为平静。
<br/>　　他有着他自己的选择。
<br/>　　如今离开北邙学院，时机并不对。
<br/>　　“哦？”
<br/>　　通天阁老者听到楚风眠的回答，似乎都有些诧异，好像什么没有想到居然会被楚风眠拒绝。
<br/>　　在他看来，这个时候，正是拉拢楚风眠的最好时候，楚风眠的拒绝似乎有些令他出乎意料。
<br/>　　不过他也并不生气，脸色极为平静的开口道。
<br/>　　“既然楚公子现在不愿加入，也不强求，通天阁的大门，一直想着楚公子打开这，只要是楚公子愿意，随时可以加入其中。”
<br/>　　一边说着，一边通天阁老者便是拿出了一个玉牌，递给了楚风眠。
<br/>　　“这是我们通天阁的身份牌，只要是拿着玉牌，前往任何一座通天阁，都会被奉为上宾，楚公子要是有事，也可以前往通天阁来寻找我们。”
<br/>　　“嗯。”
<br/>　　楚风眠点了点头，便是将那玉牌接下。
<br/>　　对着通天阁的示好，楚风眠倒是并不拒绝。
<br/>　　通天阁在北域，有着通天的本事，跟他们交好，也是一件好事。
<br/>　　“归墟天的开启，还有一月时间，诸位在武斗大会上，取得前十名次的武者，一月之内，便是达到归墟天前，倒时候七大宗门的人，会安排开启进入。”
<br/>　　通天阁老者开口道。
<br/>　　这一届武斗大会，便是就此落寞。
<br/>　　楚风眠夺得头名，令人惊讶。
<br/>　　但是楚风眠展现出来的实力，的确是无愧于年轻一代这北域第一人的称呼。
<br/>　　从今日起，楚风眠的名字，也将响彻整个九域了。
<br/>　　无数武者，从武王城中，纷纷离开。
<br/>　　这一届武斗大会，实在是发生了太多的事情。
<br/>　　夜天君的死，葬天的死，楚风眠的强势崛起，还有夜牧老祖的突然出现，等等，这些消息，不少人都想要尽快的传播出去。
<br/>　　“我们也走。”
<br/>　　楚风眠看了一眼寒月漓，悄然之间没有过多张扬，直接是化为了一道遁光离开。
<br/>　　“北域第一人，今日，北域第一人便就是一个死了。”
<br/>　　就在看台上，一道冰冷的目光，看向楚风眠。
<br/>　　这一道冰冷的目光，正是那毒手魔君。
<br/>　　“外强中干的货色，如今被夜牧那老家伙打伤，正好是杀他的最好机会！”
<br/>　　随着楚风眠的遁光，他也是化为为了一道遁光，直接向着楚风眠飞的方向走去。
<br/>　　武王城外。
<br/>　　楚风眠跟寒月漓，离开了武王城，便是化为一道遁光，直接遁入到了虚空之中，在其中连连穿行。
<br/>　　这遁光的速度极快。
<br/>　　但是就在后面不远处，另外一道遁光，也是紧追不舍。
<br/>　　“毒手魔君，似乎追了上来了？怎么办？”
<br/>　　寒月漓察觉到背后的遁光，不由有些紧张的开口道。
<br/>　　“你现在的伤势？要不要我去引开挡住他？”
<br/>　　楚风眠之前硬抗那夜牧老祖的刀锋，现在看起来受了重伤，这个时候，在毒手魔君一战，绝对是极为不利。
<br/>　　“无妨，将他们引入到虚空深处去，我想要杀了他们，轻而易举。”
<br/>　　楚风眠冷冷的开口道。
<br/>　　“武王城内，强者众多，在其中出手，很容易被人制止，只要带到虚空深处去，这些蝼蚁都得死。”
<br/>　　楚风眠话中，充满这杀意，在他开口的一刻，只见肉眼可见，楚风眠身上的伤势，突然恢复了不少。
<br/>　　不过很快，又是变化成了那一副重伤的样子。
<br/>　　想要趁着楚风眠受伤，来对付楚风眠？
<br/>　　楚风眠有着战龙血脉，建木树枝的生机，他身上的伤，根本是瞬间就可以恢复好。
<br/>　　但是楚风眠却是故意没有恢复，正是想要看看有没有人，前来找死。
<br/>　　这毒手魔君，果然是想要打算趁着楚风眠受伤，来追杀楚风眠。
<br/>　　既然如此，那楚风眠也就跟他们好好玩玩。
<br/>　　反正这毒手魔君，也是在楚风眠的必杀之列。
<br/>　　楚风眠跟寒月漓，在这虚空之中，不断遁形，越来越远，开始向着虚空深处飞去。
<br/>　　“想要逃出我的手掌心？可笑，得罪了我们地魔门的人，都只有死路一条，逃吧，逃的越来越好，倒时候就算你们的死期，先天神体，天级灵器，这些东西全部都会属于我的！”
<br/>　　毒手魔君看着前方楚风眠跟寒月漓的遁光，也是不断冷笑，眼神中尽是贪婪。
<br/>　　现在楚风眠跟寒月漓，全部都是重伤，哪里有比这个时候更好的机会。
<br/>　　杀了楚风眠跟寒月漓，得到他们两人的奇遇，一个资质再差的武者都能一步登天，更何况毒手魔君。
<br/>　　毒手魔君的遁光，一直是追随在后面。
<br/>　　就在来到这虚空深处的一刻。
<br/>　　这毒手魔君的身躯，却是动了。
<br/>　　他的身躯，直接化为了一团毒雾，猛然之间，速度超越遁光，从四面八方，直接将楚风眠跟寒月漓，围困在了其中。
