　　楚风眠催动吞天秘术，笼罩在圣院宗地底之下的灵泉之上。
<br/>　　不足半分钟的时间，这灵泉之上的禁制，就已经破碎大半。
<br/>　　楚风眠旋即凝聚灵力，催动战龙之爪，这灵泉便是从地底之中，也被楚风眠抓了出来。
<br/>　　轰隆隆！
<br/>　　圣院峰倒塌，跟刚刚六绝峰的倒塌，一模一样的场景。
<br/>　　从这圣院峰之下随着龙爪被抓出来的灵泉，足有万丈之长。
<br/>　　要比起那六绝峰地底灵泉，要足足庞大百倍。
<br/>　　这么大的一座灵泉，楚风眠看到的一刻，都是有些震惊，怪不得圣院宗可以培养出如此之多的弟子来。
<br/>　　这样一座巨大的灵泉，如果是给剑道门来培养弟子，足以支撑剑道门万年时间了。
<br/>　　这样庞大的灵泉，在如今这个时代之中，已经是无比罕见了。
<br/>　　“收取！”
<br/>　　楚风眠脸色一喜，将那灵泉凝聚起来，旋即收入到了空戒之中去。
<br/>　　同时另一边，在这倒塌的圣院峰之上，楚风眠也见到了一座宫殿，里面蕴含着琳琅满目的宝物。
<br/>　　功法，武技，灵药，灵器，圣药，材料，因有尽有，比起六绝峰之中的宝藏还要多上数百倍。
<br/>　　其中的许多奇物，甚至是对楚风眠都是有着不小的用处。
<br/>　　圣院宗虽然名义上是四宗第二，但是在圣龙秘境内，绝大部分的资源，都是交给了圣院宗。
<br/>　　至于圣龙宗，根本没有得到过圣龙秘境的资源。
<br/>　　经历数万年的时间，这圣院宗积攒下来的资源财富，简直是海量，不可计量。
<br/>　　现在却是全部便宜了楚风眠。
<br/>　　不过楚风眠也看得出来，这宝库之中，真正对于提升半步天人境级别强者的奇物丹药，却是没有。
<br/>　　不可能是圣龙秘境之中没有，而是只怕是这种级别的宝物，现在都在圣地之中。
<br/>　　现在楚风眠解决了圣龙秘境内的事情之后，下一步也就是要对付圣地之中的那些老古董了。
<br/>　　那些老古董，才是圣龙秘境中最为棘手的对象，不过现在楚风眠已经是有着自信。
<br/>　　楚风眠看了一眼那宝库宫殿，大手一挥，从其中飞出来了数十种奇物圣药。
<br/>　　这些东西都对神树，对楚风眠的肉身有着提升的宝物。
<br/>　　楚风眠运转起吞天秘术来，将这数十种奇物圣药，纷纷吞噬，刹那之间便是被神树所炼化，令神树的力量再度提升。
<br/>　　神树强横的炼化吞噬能力，剑鼎都可以轻易吞噬，更何况这点奇物圣药。
<br/>　　在吞噬了这些奇物圣药之后，楚风眠感觉到，他的境界，也已经是正是踏入到了九劫古帝巅峰了。
<br/>　　应该说是达到了九劫古帝的最后一个层次，但是从巅峰，到踏入半步天人境，还有着一段距离。
<br/>　　这些奇物圣药的效果，不亚于一位半步天人境强者的精血了。
<br/>　　不过真正的宝物，应该还都在圣地之中，等楚风眠灭了圣地之中的那些老古董之后，才算是大赚特赚。
<br/>　　现在的这些，只是开胃菜罢了。
<br/>　　“算算时间，那些老古董也应该要到了。”
<br/>　　楚风眠目光闪烁，突然看向圣龙秘境之外，身影一闪，飞了出去。
<br/>　　青梦的遁光，紧随其后，同时跟随楚风眠离开了。
<br/>　　这圣龙秘境的天星河，已经是被楚风眠彻底掌控，出入圣龙秘境，对楚风眠来说如入无人之境。
<br/>　　两道身影，瞬息之间来到了圣龙秘境之外的星空之上。
<br/>　　楚风眠催动灵识，横扫而去，足以看见千万里之外，数十道身影正向着圣龙秘境的方向飞来。
<br/>　　这数十道身影，最弱的都是九劫古帝，其中半步天人境层次的强者，都足足有着七位。
<br/>　　这就是圣地之中的老古董，圣龙秘境的真正底蕴。
<br/>　　不过只有楚风眠催动灵识看到了那些老古董，圣龙秘境的那些老古董，却全然无法察觉到楚风眠的窥探。
<br/>　　楚风眠的灵识，随着楚风眠境界的增长，也在不断的增长。
<br/>　　在这星河之中，楚风眠的灵识足以跨越千万里，这可是半步天人境强者都无法做到的事。
<br/>　　楚风眠二世为人，除了继承了他前世的剑道，记忆之外，楚风眠的灵识也强的吓人。
<br/>　　楚风眠的窥探，那些圣龙秘境的老古董，根本没有察觉。
<br/>　　“快来了，正好就在这里，送他们一份大礼。”
<br/>　　楚风眠嘴角露出几分冰冷之色道，大手一挥，天星河再度出现，笼罩在圣龙秘境之上。
<br/>　　这样子，好像是天星河，没有任何的变化一样，不过天星河的内部，已经是被楚风眠打入了属于他的烙印。
<br/>　　除了楚风眠，没有任何人可以掌控天星河了。
<br/>　　“师姐，你先躲在天星河之中，看我对付那些老古董。”
<br/>　　楚风眠转过头看向青梦开口道。
<br/>　　青梦的实力，虽然堪称帝尊无敌，可毕竟没有踏入半步天人境。
<br/>　　楚风眠跟半步天人境的强者交手，她可无法插手。
<br/>　　青梦也知道这一点，身形一闪，躲藏进了天星河之中。
<br/>　　天星河之大，根本难以探察，更何况天星河还有着遮掩气息的能力。
<br/>　　楚风眠同样是身形一闪，隐藏起了气息，旋即进入到了天星河之中。
<br/>　　从外面看来，根本看不出来，圣龙秘境内部已经是发生了翻天覆地一般的变化。
<br/>　　千万里距离，对九劫古帝来说，并不算遥远，几分钟的时间，这一群人，已经是来到了圣龙秘境的附近。
<br/>　　从远远的，一些人就已经是看到了天星河笼罩的圣龙秘境，其中几人都是眉头一皱。
<br/>　　“怎么回事？不是说天星河被一个小家伙收取了吗？怎么天星河好端端的在这里？无不源那个家伙怎么说的？”
<br/>　　一尊半步天人境的圣龙秘境老古董，遥远的看见天星河，不由的眉头紧锁说道。
<br/>　　天星河，是圣龙秘境的根本，一直守护着圣龙秘境。
<br/>　　像是这些老古董，可以一直躲藏在圣地之中修炼，也是因为有着天星河，圣龙秘境根本不惧外敌的入侵。
<br/>　　所以听闻天星河有变，这些半步天人境的老古董，都顾不得参悟天人大道，闭关而出。
<br/>　　可现在，天星河就待在了哪里，根本没人动过。
