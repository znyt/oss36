　　莫悠打量着身旁两只傀儡，随后点点头，打出两个禁制，面露满意之色。
<br/>　　这些傀儡，既作为奖励，自是无主之物。
<br/>　　武者在其上打下禁制，即可操控。
<br/>　　其余黑云学教弟子，亦是人手两只傀儡。
<br/>　　不过，除却莫悠的两只达到绝世9999道层次外，其余傀儡，尽皆只在绝世9994道左右，差了不少。
<br/>　　当然，这对一般的学教弟子来说，已经是极其惊人的奖励了。
<br/>　　众人纷纷打下禁制，操控下自己的两只傀儡。
<br/>　　有了这些傀儡，他们无异于如虎添翼，在这危险的古帝洞府一行，起码安全性高了许多。
<br/>　　哗…哗…哗…
<br/>　　15位拥有名额的带队者，尽皆将手中流光挥洒而出。
<br/>　　各支队伍的奖励，凭空而现。
<br/>　　冉琦那边，是一件件圣器。
<br/>　　那是一颗颗白色珠子，拳头大小，表面散发着惊人的风属性气息。
<br/>　　“这东西…”冉琦微微皱眉，手掌搭于其中气息最甚的珠子上。
<br/>　　“上品巅峰圣器，帝风珠。”
<br/>　　冉琦感知珠子的气息，脸色先是一喜，随后又撇撇嘴。
<br/>　　这珠子，竟是上品巅峰圣器，自然是不可多得的重宝。
<br/>　　不过，他不修风之一道，这帝风珠拿了也无用。
<br/>　　除他外，其它的白色珠子，只是中品巅峰圣器，不过对于跟随他而来的那些一流势力天骄来说，这已经是相当满意的奖励了。
<br/>　　另外，这些奖励他们也选不了；他们既走的是风道通道，得的奖励，自然与风之一道有关。
<br/>　　其它队伍亦一样。
<br/>　　叶流那边，是一件件散发着血色的圣器。
<br/>　　同样的，其中最强的一件，达到上品巅峰层次。
<br/>　　其余的，则达到中品巅峰。
<br/>　　……
<br/>　　各支队伍，看着自己的奖励，无不惊喜连连。
<br/>　　奖励，刚好是人手一份。
<br/>　　通过通道后，人数多的队伍，奖励数量便多些。
<br/>　　人数少的队伍，奖励便少些。
<br/>　　如张南峰那边，就两份奖励，那是两把纯白利剑，一把是上品高阶圣器，一把则是中品高阶圣器。
<br/>　　显然，他二人的奖励，比其余各支队伍的，要稍微低一些。
<br/>　　……
<br/>　　洞府之外。
<br/>　　傲东楼以及各大势力长老，正替一个个天骄疗着伤。
<br/>　　这些，都是过通道时支撑不住，提前放弃离去的天骄。
<br/>　　一个个天骄，边受着疗伤，边交待着如今洞府内的情况。
<br/>　　“哦？在走禁制通道了吗？”傲东楼笑笑。
<br/>　　“你们从洞府被轰出来后，已经许久了，并未有人再被轰出来。”
<br/>　　“想来，各支队伍都通过了。”
<br/>　　一旁，十八府长老点了点头，满脸喜色。
<br/>　　“想起来，当年我们年轻时入这帝墓，可远没有现今这班小子厉害。”
<br/>　　“十多支队伍，真正能过通道的，也就寥寥数支，而且尽皆损兵折将。”
<br/>　　“哪怕是几个最强的带队者，也身负不轻的伤势。”
<br/>　　“呵呵，这还得算上运气因素，若是碰到了难度极大的通道，本是实力排在前列的队伍，怕也只能黯然放弃。”
<br/>　　一个个长老回忆起前事，一阵唏嘘。
<br/>　　傲东楼笑道，“这一次，怕是15支队伍悉数通过了，起码还没有一个拥有名额的带队者被轰出来。”
<br/>　　“最妖孽的一代，果然名不虚传。”
<br/>　　“另外，老夫没记错的话，过了通道之后，那般奖励，可是丰厚得让人狂热。”
<br/>　　“不错。”十八府长老点点头，“我记得，不知多少年前的那一届帝墓盛事，我家府主，可是在里头得了一件中品巅峰圣器。”
<br/>　　傲东楼笑笑，“这一次，连从未有人通过的炼药禁制都存在其中。”
<br/>　　“各条通道禁制融合后，难度大增，其最后的奖励，必然也会高一个层次。”
<br/>　　“恐怕，上品圣器都会出现其中。”
<br/>　　……
<br/>　　傲东楼猜得没错，以往从无人能过的炼药禁制，这次被过了。
<br/>　　而禁制融合后，各条通道的难度，也因此大增。
<br/>　　故之后出现的奖励，亦是层次更高。
<br/>　　15位带队者，几乎清一色得了上品巅峰层次的圣器。
<br/>　　普通的跟随天骄，亦得了中品巅峰圣器。
<br/>　　这次的古帝之墓奖励，怕是无数次开启来，最丰厚的一次了。
<br/>　　……
<br/>　　洞府之内。
<br/>　　各支队伍，都在熟悉着自己的奖励，以及恢复伤势。
<br/>　　他们很清楚，接下来等待他们的，将是更加可怕的危机，他们必须做好足够的准备。
<br/>　　萧逸这边。
<br/>　　萧逸拿过摘星丹，其余众人，则拿过绝世皇极丹。
<br/>　　顾怜星、公孙火舞与十数天骄，看着手中丹药，无不面露狂热。
<br/>　　萧逸笑笑，“绝世皇极丹，其效果更在百道丹之上。”
<br/>　　“不过这种丹药的限制，是在绝世9992道之下。”
<br/>　　“故我更建议你们踏入绝世之境后再服用此丹，届时直接便能增一道修为。”
<br/>　　百道丹的效果，是绝世之境以下，服用者，凭空增加100条完整武道。
<br/>　　这是适合于9800道左右的武道大能服用的丹药。
<br/>　　而绝世皇极丹，则是适合刚入绝世之境的武者服用。
<br/>　　看起来，其他队伍的奖励，比萧逸队伍的奖励更好；一把中品巅峰圣器，似乎比一粒绝世皇极丹要珍贵得多。
<br/>　　实则，对于这些一流天骄而言，一粒绝世皇极丹，反倒重要得多。
<br/>　　这种丹药，足以让初入绝世之境的强者，瞬间打好绝世根基，绝世之路他日将会平稳许多。
<br/>　　再者，对武者而言，修为才是本质；而圣器，终究只是外物。
<br/>　　另外，现今之大陆，早就没了绝世皇极丹这等珍稀丹药的丹方。
<br/>　　哪怕在药尊殿总殿，萧逸也没见过这种丹方。
<br/>　　这种丹药的珍稀性，还有其逆天之效，不是外物可比的。
<br/>　　事实上，即便是萧逸，在见到这种丹药作为奖励后，都吃了一惊，更对上古之时的药之一道感到惊讶。
<br/>　　上古之时的炼药师，其手段，绝非现今的炼药师可比的。
<br/>　　创造出的丹药，其效果，更是足以让人惊骇。
<br/>　　上古时期，那个武道繁衍至巅峰的时期，到底精彩到何等地步？
<br/>　　萧逸暗暗摇头，随后，收敛了心头杂念。
<br/>　　回过神来，却见顾怜星的目光，直勾勾地盯着自己手中摘星丹。
<br/>　　“呵。”萧逸淡淡一笑，“怜星姑娘，可是眼热我手中这摘星丹？”
<br/>　　“咕。”顾怜星不自觉地咽了口口水。
<br/>　　不等顾怜星说话，萧逸手掌一张，丹药飞跃而出，“送你了。”
