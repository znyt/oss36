　　武胜国度的另外三大宗门，在加上武胜学院，几乎便是代表了武胜国度的最强力量。
<br/>　　要是今日楚风眠跟他们为敌，彻底的撕破脸皮，几乎便是代表着楚风眠跟整个武胜国度抗衡。
<br/>　　“三仙岛岛主的话不错，楚风眠，今日你跟我们各退一步，我们会给你赔礼道歉，这一次的事情就算是过去了，不过你必须要将圣使给放了，北邙学院，不是你可以得罪的起的。”
<br/>　　风神宗宗主风华，出声附和道。
<br/>　　“不然的话，我们三大宗门会支持武胜学院，来对付你。”
<br/>　　“不错。”
<br/>　　烛炎门门主也是站了出来，冷哼一声说道。
<br/>　　这三人看着楚风眠，同样冷笑，等待着楚风眠的退步，如今他们便是坚定不移的站在了那年轻男子一边。
<br/>　　毕竟北邙学院，可是称霸北域的庞然大物，而楚风眠不过是一个孤家寡人罢了，就算是实力再强，也不可能跟真正的北邙学院抗衡。
<br/>　　在这两方的选择上，几乎不需要犹豫，他们三人稳稳站在北邙学院一边，只是将武胜学院给踢了出去，作为给楚风眠的补偿。
<br/>　　他们有着自信，楚风眠肯定会答应了这个条件，毕竟单单对付一个武胜学院，楚风眠也许可以。
<br/>　　但是若在加上三大宗门的话，以楚风眠一己之力，也很难抗衡。
<br/>　　这三人如今便是串通一气，要逼得楚风眠退步，忍气吞声了这一次。
<br/>　　只是可惜，他们三人却是不了解楚风眠的性格，换做是其他武者，这一次肯定是退步了不错。
<br/>　　可是如今的人，却是楚风眠，在楚风眠的心中，从来没有退步这一个说法。
<br/>　　更何况，只要是敢跟他为敌的人，都要死，这三大宗门，也是一样。
<br/>　　“你们三个废物，谁给你们胆子，敢来威胁我？”
<br/>　　楚风眠眉头翘起，怒声喝道。
<br/>　　“还有，你以为我会怕了你们三大宗门？今日的事，你们三大宗门一样有份，既然你们今日敢来站队，那好，那你们三大宗门，也没有在存在的必要了！”
<br/>　　楚风眠冷哼一声，这三大宗门的人，一直也没少针对他，曾经楚风眠大闹太子大婚的一刻，这三大宗门的人坐山观虎斗。
<br/>　　上一次楚风眠没有来得及去解决他们，但是这一次可不一样了。
<br/>　　他今日便是就要将这三大宗门，彻底解决，以后武胜国度，只有一个寒雪阁，再也没有什么四大宗门。
<br/>　　这三大宗门的人还敢来威胁楚风眠？敢威胁楚风眠的人，都要死。
<br/>　　“什么？”
<br/>　　“狂妄！”
<br/>　　“好大的胆子！”
<br/>　　那三人，皆是脸色一变，随后怒声吼道。
<br/>　　“楚风眠，你以为你是谁，还真以为你是无敌人物，可以跟我们三大宗门抗衡？就凭你一个人，我看你有着什么本事！”
<br/>　　他们三人可都是一方宗主门主一般的存在，向来都是他们呵斥被人，那有被人呵斥的时候。
<br/>　　一个个脸色涨的通红，都是愤怒暴起，一身灵力，猛然爆发了出来。
<br/>　　风神宗宗主，风华大声喝道。
<br/>　　“你等着！等着我们三大宗门无穷无尽的报复吧！你，还有寒雪阁，都会被毁灭！”
<br/>　　听到风华的话，楚风眠嘴角露出一抹不屑笑容。
<br/>　　“你们以为，今日你们还能活着离开吗？”
<br/>　　“你敢！”
<br/>　　听到楚风眠的话，那烛炎门门主怒声喝道。
<br/>　　“你敢杀了我，就等待我们烛炎门无穷无尽的报复吧！我们烛炎门中的老祖宗，肯定不会放过你的！”
<br/>　　这烛炎门门主的话音未落，一股庞大无比的战龙之爪，已经是从天而降，直接是砸向了他的身躯，将他的身躯直接打碎，化为了一团血雾。
<br/>　　“不敢？你们三大宗门的巅峰圣者来了又如何？今日谁来谁死，没有人可以救得了你们！”
<br/>　　楚风眠冷笑一声，手一抓，那烛炎门门主一身圣者精血，已经被他收取。
<br/>　　“走！”
<br/>　　看到烛炎门门主被楚风眠一招灭杀，那三仙岛岛主跟风华，对视了一眼，一身灵力暴起，直接要逃。
<br/>　　楚风眠现在的实力，远远要超越他们，如今必须只有他们门派中巅峰圣者的老古董，才能够跟楚风眠抗衡。
<br/>　　突然之间，在三仙岛岛主的面前，一道巨大无比的铜门出现，一下子将他们二人的身形给吸了进去，在空中准备消失。
<br/>　　“仙玄之门？看来上一次我还没有将这灵器彻底砸碎，居然还被你们给修复了？”
<br/>　　楚风眠看到那仙玄之门，冷笑一声。
<br/>　　上一次方仙道利用仙玄之门来偷袭，被丰饶碑直接砸碎，不过没想到居然是被三仙岛如今给修复了。
<br/>　　再一次的唤出仙玄之门，想要从楚风眠的面前逃走。
<br/>　　就算是巅峰圣者，也不可能逃得出楚风眠的手掌心。
<br/>　　这区区仙玄之门，又怎么可能。
<br/>　　“给我过来！”
<br/>　　楚风眠大手一抓，一手直接深处到了虚空深处，看着其中疯狂逃窜的仙玄之门，他冷哼一声，手一抓。
<br/>　　这仙玄之门，顿时便是从虚空深处中被楚风眠爪了回来，捏在手中。
<br/>　　“碎！”
<br/>　　楚风眠手心一用力，这被他抓在手心中的仙玄之门，一下子被彻底捏碎，捏碎成了粉末，再也无法恢复。
<br/>　　同时那三仙岛岛主跟风华二人，也是将那从仙玄之门中逃窜出来。
<br/>　　仙玄之门的余波，更是将他们震的吐血连连。
<br/>　　“此人的实力，什么时候居然是变的如此恐怖！”
<br/>　　三仙岛岛主看着楚风眠徒手捏碎仙玄之门，更是吓得脸都白了。
<br/>　　仙玄之门，可是地级灵器，他们三仙岛的阵宗之器，上一次被楚风眠打破，也算是毁在丰饶碑的手中。
<br/>　　丰饶碑毕竟也是天级灵器，能够打碎仙玄之门，也是情有可原。
<br/>　　可是这一次，楚风眠居然是徒手之间就将仙玄之门给捏碎，这地级灵器，在楚风眠的手中，居然是脆弱到了这般地步。
<br/>　　这等实力，简直是比起他们三仙岛的巅峰圣者，都要恐怖许多。
<br/>　　“好强。”
<br/>　　“少主的实力，已经是达到了这一步了吗？居然已经快要逼近曾经主人的实力了。”
<br/>　　远古龙龟在寒雪阁中，看着空中出手的楚风眠，也是吃了一惊。
