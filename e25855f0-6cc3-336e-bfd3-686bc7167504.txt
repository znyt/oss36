<br/>　　陆画当机立断，直接化为一道遁光，准备要逃。

<br/>　　只是楚风眠看到这陆画的遁光，只是嘴角露出一抹冷笑。

<br/>　　不管是陆画的遁光，速度如何之快，他也无法跳出这一片区域之中去。
<br/>　　这种手段，简直惊为天人。
<br/>　　“几位长老，这……”
<br/>　　在北邙圣域的虚空之中，隐藏的几位长老，看到龙未央死在楚风眠的手中，都是一愣。
<br/>　　他们本以为，龙未央击败楚风眠，根本不是什么难事，等到那个时候，想要去阻止龙未央的。
<br/>　　毕竟楚风眠可是得到了北邙圣君的传承，他们也不想楚风眠，就这样的死在龙未央的手中。

<br/>　　就连陆画，楚风眠也都有着打算将他灭杀的意思。
<br/>　　龙未央，只是有着一些奇遇的弟子，他死了也就死了，虽然说损失了一个巨头，但是对北邙学院来说，却并不算的什么。

<br/>　　陆画要是被楚风眠轰杀，陆家肯定不会善罢甘休，整个北邙圣域，都要震动。

<br/>　　一尊长老，突然开口道。
<br/>　　众人顺着目光看去，就在远方，一道遁光，已经是突然飞了过来。
<br/>　　这道身影，看起来是一尊白发老者。

<br/>　　楚风眠一双眼睛，也是看向天空之上，看向这陆雀。
