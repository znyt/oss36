　　一晃眼，时间已经过去了两三个月，
<br/>　　叶尘一路穿梭，跨越了数百个大大小小的星域，离迷罗星海也越来越近。
<br/>　　而在着两三个月的时间里，他的修炼丝毫没有拉下，时而也会采购有些药材，炼制一些对修炼有用的丹药。
<br/>　　因而这一路走下来，他的修为不但已经提升到了元婴巅峰境，连炼丹之术也提升到了六品巅峰，甚至如果状态好的话，应该已经可以尝试炼制七品丹药了。
<br/>　　而以叶尘如今的修为，再加上对灭世魔刀的运用更加纯熟，即便是返虚真君，他也已经丝毫不惧。
<br/>　　这一日，他终于抵达迷罗星海的边缘地带。
<br/>　　只见前方星空之中，犹如一处巨大的黑洞般，在漫漫星河之中，显得十分突兀，而且令人望而生畏，根本不敢靠近，似乎只要一靠近，就会那无尽的黑暗吞噬其中。
<br/>　　迷罗星海，之所以被称作迷罗星海，就是因为，一旦人踏足其中，就会迷失在这片星域之中！
<br/>　　甚至就连渡劫帝君，都未必能从这里面逃脱。
<br/>　　当然，这种情况并不是绝对的。
<br/>　　每隔百年，迷罗星海最中心位置，就会有一颗无比明亮的星辰冉冉升起，届时整个迷罗星海就会被全部照亮，大约会持续十年左右。
<br/>　　而就在这十年之内，便是迷罗星海最安全的时候，想要进入其中探险的修真者，都会选择在这段时间进入。
<br/>　　但，即便是这最安全的十年，迷罗星海里面依旧是风险重重，而且各种大大小小的星罗幻阵，遍布整片的星域，有些修为弱的或者定力不足者，极有可能会永远陷在里面，找不到出来的路。
<br/>　　“根据时间来推算，离下次迷罗星海开启，还有两个月左右的时间，我不如趁这个机会，将修为再作一下提升，最好能一举面如化神境，这样把握也能更大一些……”
<br/>　　叶尘打定主意之后，便朝离迷罗星海最近的一片星域飞掠而去。
<br/>　　叶尘还隐约记得，这是一片叫做云来星域的地方。
<br/>　　因为靠近迷罗星海，每当迷罗星海将要开启之时，就会有许多外星域的来客，事先来到这片星域。
<br/>　　久而久之，这里就变成为了一个，类似于地球上，旅游景点和服务区站点之类的地方，超过一半以上的人口，都是来自外星域，而且有些人干脆在这里定居了下来。
<br/>　　时间一久，就这么一个偏僻的小星域，居然也发展成为了一个六级星域。
<br/>　　不过，这里和其他星域最大的不同在于，在天荒星域，完全是自治的状态，没有任何规则可言，只信奉强存弱亡的生存法则，只要你的拳头够大，灵石够多，就可以得到你想要的一切。
<br/>　　很快，
<br/>　　叶尘降落在天荒星域最边缘，一颗叫做木狼星的小星球上。
<br/>　　天荒星域的主星是天荒星，不过叶尘并没有打算去那里，他现在只想找一个安静的地方，打算冲击一下化神境。
<br/>　　……
<br/>　　一晃眼，又过去了月余，
<br/>　　木狼星上，在一片幽静的小湖边，有着一个很普通的小村庄。
<br/>　　而在这个小村庄的边缘地带，有着一座崭新的小茅屋，很显然应该才刚刚搭盖不久。
<br/>　　而此时，在这小茅屋之中，正盘坐着一名少年，似乎正在打坐修炼。
<br/>　　“呼！”
<br/>　　一直双眼紧闭的叶尘，霍然睁开双眼，眼眸露出一抹失望之色。
<br/>　　“果然！对于修真者来说，元婴境和化神境之间，是一道很明显的分水岭，我算起来，突破元婴境，也不过才短短数月，现在就想要突破到化神境，还是有些太勉强了！！”
<br/>　　这一个月以来，叶尘已经尝试冲击了不下二十次，甚至连可以提升突破概率的破神丹，都吃了不下十颗，结果每次都以失败而告终，始终无法突破元婴境和化神境之间的那层壁障。
<br/>　　“厚积才能薄发！看来是我过于浮躁了，若是继续强行突破，反而会有走火入魔的风险！！”
<br/>　　一念及此，叶尘不由得摇了摇头，打算先放弃突破化神境的执念，调整一下自己的心态。
<br/>　　随手撤去周围的禁制，叶尘正要出去走走，放松一下心情。
<br/>　　就在这时，
<br/>　　屋外却忽然响起了一阵吵闹之声，
<br/>　　叶尘不由得眉头一皱，随手打开房门，缓步从茅屋中走了出去。
<br/>　　只见此时，不远处的村口道路之上，有着一群凶神恶煞之人，而在这些人面前，则正跪着一名白发苍苍，几乎已经是皮包骨头的老者，和一名同样瘦弱矮小的少年。
<br/>　　“各位大爷，我求求你们了！不要把我孙子带去挖灵矿啊！孩他爹就是被你们拉去挖灵矿，到现在都没有回来！他还是个孩子啊，求求你们饶了他吧！”
<br/>　　听到那老者的声音，叶尘不由得眉头微微一皱，大致已经猜到了事情的经过。
<br/>　　他眼下所在的这个村子，名叫长乐村，不过这里的人，却过的并不怎么快乐。
<br/>　　因为此处土地肥沃，灵气充沛，所以非常适合种植药草。
<br/>　　但是，自从数十年前，这方圆数千里之力，成为了一个叫做恶狼帮的势力范围。
<br/>　　长乐村的这些药农们，就必须定期按时向恶狼帮，交纳一定数额的药材，一旦完不成的话，就会被毒打惩罚，甚至家里的青壮年劳动力，也会被恶狼帮的弟子，拉去做苦力，挖灵矿。
<br/>　　“我去你妈的！”
<br/>　　面对老者的哀求，为首的那名恶狼帮弟子，根本不为所动，直接一脚，就将那老者踢飞了出去，同时嘴里骂骂咧咧，道：
<br/>　　“老不死的东西！还敢和老子讨价还价？真是活的不耐烦了！”
<br/>　　“噗！”
<br/>　　那老者只有炼气境初期的修为，而且身体瘦弱年迈，如何挡得住那恶狼帮弟子一脚，顿时喷出一口老血，整个人也飞出十几米远，重重栽倒在了地上，眼看进气少，出气多，已经是活不成了。
<br/>　　叶尘目睹了这一切，不过却并没有出手阻止。
<br/>　　或者是，有那么一瞬间，叶尘想要出手，不过最终却放弃了。
<br/>(本章完)
