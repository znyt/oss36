　　第12章
<br/>　　青牛山大小五十多股山贼，飞虎寨是其中最大的，山寨有人三五百众，老大高飞虎性情更是蛮横，更重要的一点原因是，飞虎寨和蛤蟆寨的关系很不对付。
<br/>　　“大当家……你是要害死我么？”徐神机。
<br/>　　“什么话，我是让你套上马车，带上咱们的好酒，去拜会高大当家，就说黑石城要攻打我们青牛山的兄弟……不仅飞虎寨，狗头岭，秃头坡，杏花岭，野猪林等等这些山寨都是要去的。”
<br/>　　“可他们会听我的么？”徐神机忐忑不安。
<br/>　　“那就要发挥军师三寸不烂之舌的本事，对他们晓之以情，动之以理，用你的真诚，你的情怀去打动他们。”
<br/>　　“可大当家说的这些我都没有。”
<br/>　　“忽悠！”程大雷大手一挥：“忽悠死他们。”
<br/>　　“可我还是害怕，大当家可能不太了解高飞虎，他和咱们老当家有些过节。”
<br/>　　“我让林少侠与你同去，你还怕什么。”
<br/>　　“其实，我更希望大当家和我一起去。”徐神机小声道。
<br/>　　“这是什么话，很危险的好不好。”程大雷道：“军师，你放心大胆地去，灵儿就交给我照顾，你放心，我们会幸福的。”
<br/>　　徐神机还准备再说什么，只听程大雷高喝一声：“抬我的大斧来，让我剔剔牙。”
<br/>　　徐神机一激灵，识趣的闭上嘴巴。他和林少羽套上马车，装上系统奖励的三十坛烈酒，一步三回头的去往飞虎寨。
<br/>　　“大当家，飞虎寨会相信徐军师么？”秦蛮问。
<br/>　　“不管他们信不信，总会派人看一看，那就是我们的机会。”程大雷。
<br/>　　没有力气，只能借力打力，这也是没有办法的办法。徐神机走后，程大雷也没有闲着，他还要试验如何将黑火药的威力发挥到最大。
<br/>　　对于火药程大雷半些不懂，只能一点点琢磨，屡次试验，比如将火药和碎石子搭配在一起，最后还真是被程大雷琢磨出一个方法。
<br/>　　他将火药填入棉被中，点燃后做成类似燃烧弹之类的武器，山寨内的棉被被他搜刮一空，徐神机大呼小叫，直喊：我们冬天要不要活了，要不要活了……
<br/>　　又将火药填入系统奖励的木箱中，装上碎石，稻草……用浸油的棉布条为引线，这耗费程大雷颇多时间，主要是因为山寨劳动力太少，派出两个人就更少了。
<br/>　　程大雷负责搭配比例，秦蛮挖来石子，徐灵儿往里填，三人都累得满头大汗，却是一刻也不敢闲着。
<br/>　　“大当家，你觉得我们能守住么？”秦蛮。
<br/>　　程大雷抬起头，看到秦蛮黝黑的脸膛上挂满汗珠，他问道：“你觉得呢？”
<br/>　　“我是个蛮人，懂的事情不多。”秦蛮道：“但我想守住。”
<br/>　　程大雷怔了怔，随即道：“其实我也不知道能不能守住，但……我想守住。”
<br/>　　“好啦，秦大哥把火药箱搬远些，最后一次试验。”
<br/>　　程大雷令秦蛮将火药箱安置好，然后一点点把引线拖过来，三人躲在相对安全的距离，看这次爆炸的效果。
<br/>　　苏樱正在屋中静坐，程大雷这几天的忙碌都被她看在眼里，却也不觉得如何。螳臂挡车，蚍蜉撼树大概就是这个样子，四个人——好吧，就算五个人竟然想挡住黑石城的攻打，简直就是痴心妄想。
<br/>　　轰！隆！
<br/>　　一声惊天巨响突然响起，整座山似乎都在颤动，土屋房顶扑索索落土。苏樱吓了一跳，身体下意识颤抖……这，这……地龙翻身么！
<br/>　　好久不见动静，苏樱提心吊胆的往窗外看一眼，就见烟尘中程大雷三个人熏得一张黑脸，在那里拍手相庆。
<br/>　　“成了，成了……”
<br/>　　……
<br/>　　黑石城，校场，队伍列成整齐的方阵。
<br/>　　有骑兵二百，兵强马壮，刀兵三百，黑甲黑刀，弓兵二百，背后是满满的箭筒，重甲兵一百，全身上下笼罩在铁甲内，不动如山，长枪兵二百，一柄柄长枪朝向天空，宛若要对天宣战。
<br/>　　韩玄之站在队伍最前方，身上白衣，手中羽扇，腰佩三尺青锋剑，当真是浊世佳公子，翩翩美少年。
<br/>　　别，别被这一幕吓着。
<br/>　　整齐的队伍在很短时间后，就出现松动。出兵不是一件小事，总要有祭天祭地各种仪式，然后城内的大小官员还要发表讲话，回忆过去，遥望未来，总还要选择一个黄道吉日出兵。
<br/>　　一般午时最好，因为杀机最重。而现在又是夏天，一到正午太阳高悬头顶，阳光肆无忌惮撒在校场。
<br/>　　韩玄之轻摇羽扇，感慨道：“晴空万里，好兆头呐。”
<br/>　　“好……你个芭芭拉……”
<br/>　　重甲兵心底早就开始骂娘，几十斤重的铁甲罩在身上，密不透风，你又巴拉巴拉讲个不停，让身在铁甲内的战士如同煮在锅里，他们直到现在还没人中暑，只能说身体素质还不错。
<br/>　　“王叔，您也来打仗，您今年有六十吧？”
<br/>　　“六十三啦，是不是看上去不像，哈哈。对了，小毛你的手不是上个月摔断了么，怎么还能当弓兵？”
<br/>　　“嘘，小声点，这次一千人打几个小蟊贼，多我一个少我一个不打紧。”
<br/>　　……
<br/>　　……
<br/>　　韩玄之满意的看着这一幕，此刻他感觉自己不是一个人在战斗，陈玄之、李长庚，钟昧离这些帝国历史上的名将全部附身，自己的名将之路将从今天开启。
<br/>　　『老爹你的荣光是三十年前么，而我最光荣的时候就是现在啊』
<br/>　　韩玄之忽然将剑鞘落在一个士兵的肩头，问：“高不高兴？”
<br/>　　“高兴？”士兵高昂胸膛。
<br/>　　“我也替你高兴。古时名将出征，都会选一人祭旗，而我选中的是你。”韩玄之拔出佩剑。
<br/>　　“少城主……你……你不要说笑……”
<br/>　　“说笑！”韩玄之目光如刀：“军中无戏言，我怎能说笑。”
<br/>　　“安息吧，从今日起，你的家族将和我的名字一起荣耀。”手中剑朝着士兵脖子重重挥下。
<br/>　　“荣耀……你麻痹……”士兵倒在自己的血泊中，死不瞑目。
<br/>　　韩玄之昂起头，手中沾血的长剑重重一挥：“出发！”
<br/>　　人群一片寒意，看着韩玄之夸张的身影，默默想的都是一句话。
<br/>　　“这货……是个白痴呐。”
<br/>　　
<br/>(本章完)
