　　徐福……
<br/>　　四人大惊。
<br/>　　马面回过神来，摇身一晃，化作一匹白马，朝远处奔袭出去。“我去阴司汇报，你们去找徐福，无论如何缠住他！”
<br/>　　徐福，现在是阴司头号通缉的要犯，一旦发现，任何事情都要暂缓一边，以捉拿徐福为首要！
<br/>　　黑白无常和牛头凑近孽镜碎片，从中间看到了徐福的身影。
<br/>　　“这是哪里？”白无常问道，然后立刻看到了徐福身下的那个球面体的巨大物体。
<br/>　　东皇钟！
<br/>　　三人二话不说，立刻调头往回赶，还好他们刚离开不久，速度也是极快，很快又回到山谷之中，老远就看到了徐福，黑无常道：“正要拿你……”
<br/>　　牛头用胳膊怼了他一下道：“拿什么拿，拿得住吗，换个台词！”
<br/>　　“呃……徐福休走！”
<br/>　　三人飞身而上，扑向徐福。
<br/>　　徐福转头看过去，从袖子里摸出一支判官笔，在空中抹了几下，形成了一道不定式符，飘了过去。
<br/>　　只是这随手化成的一张符，就将三位阴司赫赫有名的正神挡住。
<br/>　　徐福不再看他们，右手摊开，手里托住一方大印，轻轻一掂，放出了一束流光，从上到下罩住了整个东皇钟，照得中间睁不开眼睛。
<br/>　　三位阴神打碎不定符，冲了过去，这时候金光也消散了。
<br/>　　什么都没有了。
<br/>　　叶少阳不见了，徐福也不见了，所有人都不见了，如果这些人的消失还可以理解的话，那么，最不可思议的是，连东皇钟也不见了……
<br/>　　“怎么会这样！”摇光仙子等人悚然大惊，看着这不可思议的一幕，一个个眼睛瞪的像铜铃。
<br/>　　法术不是幻术，更不是魔术。
<br/>　　东皇钟是一个实体的法器，法器可以借助法力收缩和幻化，但绝不会凭空消失……
<br/>　　“完了！”白无常跌足长叹，“他们去异界了！”
<br/>　　“异界？”牛头不解。
<br/>　　“别忘了徐福手上有山海印，可以破开时空，穿梭轮回，原则上可以去往时空中的任何一个节点！徐福一定是把小天师他们都带走了，去了另一个时空。”
<br/>　　众人面面相觑。身后一个鬼差问道：“另一个时空，那是……过去的时间？”
<br/>　　“是，任何一个朝代，与我们这个世界完全平行，如果没有山海印，我们永远都找不到他们的踪迹。”
<br/>　　黑无常看着徐文长给的那块孽镜碎片，这块三界六道无所不能追踪的灵宝，现在也完全失去了光华，灰暗下去。
<br/>　　“徐福为什么要这么做？”又有鬼差问道。
<br/>　　“鬼才知道！”黑无常骂了一声，突然又想到自己就是鬼，但自己并不知道，忍不住仰天长叹，“出师未捷，出师未捷！这下真是麻烦大了，徐福没抓到，还把应劫的小天使一干人等给弄丢了，这可怎么是好！”
<br/>　　一句话说到众人的心坎里，这次真是……丢人丢大了。
<br/>　　白无常叹道：“如果可以的话，我都想穿越时空，免得去阴司报道了。”
<br/>　　众人无法，只好继续去追击尸族大军，路上黑无常一直不作声，突然开口说道：“七哥，你之前说的话，让我好生费解……别的时空，会有我们这个时空的人吗？”
<br/>　　白无常道：“怎么会，不同的朝代，哪里会有一样的人。”
<br/>　　“但是阴司总是有的吧？”
<br/>　　白无常怔了一下道：“那是必然，毕竟是过去的时空，一切都是按照我们这个世界发展的，唐宋元明清，一点都不会错，有人间，自然有阴司了。”
<br/>　　黑无常道：“那问题就来了，既然有阴司，那就有我们，有崔府君，有大帝……那个空间的我们，又是什么人？”
<br/>　　白无常怔住，摇头说道：“是我们，但也不是。”
<br/>　　黑无常还想说什么，白无常摆手说道：“这不是你我关心的事，不要再乱想了，我们不懂这些的。”
<br/>　　“那么谁懂？”黑无常表现出了强烈的兴趣。
<br/>　　牛头说道：“我手下有个副官，是不久前死的，他是个物理学家，懂很多这方面的知识，我听他说过什么平行宇宙之类，跟七爷说的这个很像，老八你回去可以去找他聊聊。”
<br/>　　黑无常急忙道谢。
<br/>　　三人继续去追杀尸族大军了。
<br/>　　叶少阳的失踪，对道风来说也是个巨大的意外，他没有想到徐福会来，用山海印将叶少阳一干人带回过去……他猜不透徐福这么做的用意，但可以肯定，这么做是为了叶少阳。心中反而感到一丝轻松：
<br/>　　叶少阳这时候消失，其实反而是最好的结果，否则以他的性格，一定会去找后卿讨要芮冷玉，如果失败，自然只有一死，如果成功，他带着芮冷玉，一样会成为众矢之的，到那个时候，就算是阴司也不会保护他，反而会四处缉拿他。
<br/>　　而他回到过去，这些事情当然就不用去考虑了。自己也减轻了很多压力，可以去做自己想做的事。
<br/>　　想到这一层，道风心中也是轻松了许多。不过，又是一个问题浮上心头：
<br/>　　少阳，什么时候才能回到这个世界，或者，他还能回来吗？
<br/>　　道风飞出了山谷，找到了杨宫梓等人，一起闯出了星宿海的范围。
<br/>　　没有任何人牵绊，以他们的实力，在千军万马中也是来去自如。
<br/>　　逃出来的时候，他才想到一个很重要的问题：自己之前在山谷中，一心都在思考叶少阳消失这件事，却忽略了一个人的存在：李浩然。
<br/>　　在东皇钟敲响的时候，自己明明就在他身边，虽然没有打斗，但事情了结、自己离开山谷的途中，李浩然居然没有出来阻拦，自己甚至没有看到他。这不符合常理。
<br/>　　道风想来想去，觉得李浩然可能是因为伤了左手，实力不如从前，没有把握能留住自己，干脆就去养伤去了。以李浩然的性格，这极有可能。他历经几百年轮回，对时间的概念比一般人轻的多，不会着急报仇。
<br/>　　不过，道风还是想到了另外一种可能，如果真是那样……叶少阳可就麻烦了。
