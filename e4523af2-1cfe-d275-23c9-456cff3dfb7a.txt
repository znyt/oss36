        &lt;h3&gt;第三章第三百二十六章  耗材推广会&lt;/h3&gt;
<br/>
<br/>        推广
<br/>
<br/>        陈子昂公司的大楼前集聚了各路媒体，还有一些官方人员及各地的代理商。
<br/>
<br/>        这是一次盛大的推广会，来自五湖四海的人们，还有很多信息行业的人士。
<br/>
<br/>        陈子昂公司的墙面上都是一张张大屏幕，大屏幕上会显示每一个城市的推广活动。
<br/>
<br/>        落雪从美洲发来了贺电，这是这么久以来，落雪第一次公开和陈子昂的关系。
<br/>
<br/>        落雪的贺词是这样写的：涘为水生，水为万物之源，愿如水清澈，愿如水宽广，愿如水般源远流长，愿初心不改！
<br/>
<br/>        涘为陈子昂公司里的其中一个字。
<br/>
<br/>        陈子昂看着落雪在大洋的那一边念着贺词，热泪盈眶。
<br/>
<br/>        落雪代表落家站在了陈子昂的身边。
<br/>
<br/>        继落雪发来贺词之后，吴家发来了贺电：为梦而生，为民而计！
<br/>
<br/>        寒家的寒伯安、郝家的郝景文、江家的江俞轩都站在陈子昂的身边。
<br/>
<br/>        宁家也发来了贺电，就连远在甸城的谷家也发来了贺电！
<br/>
<br/>        陈子昂一个从农村走出来的人，踏上了她自己的征程，得到了各界的认可，站在各路媒体面前，她眼含热泪微微而笑。
<br/>
<br/>        “感谢大家对我的支持，在今天这样一个特别的日子里，感谢这个美好的社会让我有了用武之地，感谢这个社会对我们的包容和支持！我要感谢那些在我生命之中的人们，是你们成就了我，感谢和我一起奋斗的同事们，谢谢大家！涘为水生，水为万物之源，会如水清澈，会如水宽广，会如水般源远流长，会初心不改！会为梦而生，会为民而计！”
<br/>
<br/>        陈子昂深深的鞠躬再鞠躬，从今天起，她确实要为民而计了！
<br/>
<br/>        大屏幕上一个一个的切换，当切换到西城的时候，陈子昂没有发现陈子霖，只有易倾秋和李秀、谭九儿等人在忙碌着。
<br/>
<br/>        突然现场一片混乱，有一个部门的人来检查西城分公司的营业执照和耗材的许可证等一系列文件。
<br/>
<br/>        还准备赶走要签合同的客户，陈子昂眼睁睁的看着西城分公司一片混乱，却不见陈子霖出现。
<br/>
<br/>        “不好意思，我们这一次是全国联动，每一个地方我们的流程和本部是一模一样的，西城是我的家乡，原本想着在家乡多出一份力，看来我这个心愿怕是不能达到了。”陈子昂对着媒体微微叹息一声。
<br/>
<br/>        全国各地的流程都一样，唯独西城会有人检查，那么这就说明西城有人在捣乱！
<br/>
<br/>        陈子昂公司的营业执照，各种许可证等一切手续都很齐全，这么多双眼睛盯着呢。
<br/>
<br/>        “陈总，你是不是在你的家乡得罪过什么人？”一个媒体的记者突然发问。
<br/>
<br/>        “木秀于林风必摧之，陈总太过优秀，必然会遭人眼红，这很正常，只是今天的这种场合有点不打自招的味道，西城的媒体朋友如果愿意的话，可以问问现场的人员都是那个部门的，在挖一挖其身后的背景就知道原委了。”江俞轩替陈子昂回答。
<br/>
<br/>        “前几天江总高调宣布退出现在的这个公司，今天又在这个发布会上出现，我想问的是，江总是想回归吗？”
<br/>
<br/>        “这个公司倾注了我很多心血，我虽然离职了，但是我作为公司的一员我肯定是会来参加这个推广会的。”
<br/>
<br/>        “听说寒家、江家和郝家是一体的，今天一见果然，三位年轻的少主是不是已经在一起合作了？”
<br/>
<br/>        江俞轩看了一眼寒伯安。
<br/>
<br/>        “这个问题我来回答，我们三家父辈就很要好，到我们这里也不例外，我们已经合作了很久了，今天我们还有一套运营系统马上上线，这也是我们四个人合作的结晶。”寒伯安微笑着面对媒体。
<br/>
<br/>        “寒总，自从寒总求婚之后便没有下文了，我们想知道寒总什么时候结婚？”
<br/>
<br/>        “寒总结不结婚和你有关系吗？”李长卿接过了记者的话，让记者一愣。
<br/>
<br/>        “没有关系。”
<br/>
<br/>        “没有关系还不去找和你有关系的事情做，你看见了没有？那边是我们的运营系统，你可以好好去看看，绝对会写出一份有内涵的稿子！”李长卿指着远处赵绾绾正在调试的软件对记者说。
<br/>
<br/>        赵绾绾因为要和陈子昂一起去甸城，将手中的软件赶着在和耗材同一天时间交工，陈子昂和寒伯安他们四个人在电话里已经沟通好了，放在一起同时上线。
<br/>
<br/>        西城的混乱并没有引起巨大的动荡，相反，有些记者已经开始联系西城的同行在连线西城的情况，看看是哪个部门在现场。
<br/>
<br/>        如此的搞事情，要么是和陈子昂有深仇大恨，要么陈子昂的公司真的有问题，无论是哪一种情况都是大爆料。
<br/>
<br/>        陈子昂原本就是一个迷一样的人物，她的崛起和她的私人生活似乎没有一点点杂质，清清楚楚，干干净净很透彻，所以大家就更加对她有兴趣。
<br/>
<br/>        如果能挖出来什么大爆料，茶余饭后也许就有了谈资，而且也很有卖点。
<br/>
<br/>        陈子昂心里还是在担心西城的事情，她不是担心工作上的问题，而是担心家里的问题，担心她老父亲的身体。
<br/>
<br/>        媒体已经被吸引到别的地方了，陈子昂低声问江俞轩：“西城怎么回事？我哥怎么不见人了？”
<br/>
<br/>        “我也不知道，我去打电话问问，你去那边招呼人去。”江俞轩低声说。
<br/>
<br/>        “好！”
<br/>
<br/>        陈子霖正在冯睿的家里忙碌着，陈天天和陈子建还是痴痴呆呆的伤心，陈伟冯连个鬼影子都不见，从昨天到今天陈子霖连眼睛都没眨一下。
<br/>
<br/>        陈子慧、陈子悦和陈子乔都来帮忙了，怎么说，冯睿都曾经是陈家人，现在人已经走了，千错万错都已经没有错了。
<br/>
<br/>        陈子霖接到江俞轩的电话对江俞轩说：“今天的推广会我没有办法参加，我二嫂走了，我一直在忙着家里的事情呢。”
<br/>
<br/>        “什么意思？你二嫂？”
<br/>
<br/>        “冯睿啊，出车祸走了，昨晚走的，唉，好好的一个人说没了就没了，对了今天的推广会成功吗？”
<br/>
<br/>        “推广会只有西城的不成功，别的都非常成功。”
<br/>
<br/>        “啊？西城怎么了？”
<br/>
<br/>        “你忙你的吧，子昂就是没有看见你很担心你，你也不知道给她打个电话说一声。”
<br/>
<br/>        “我不是怕影响今天的推广会嘛，我就叮咛李秀他们不要给子昂说了。”
<br/>
<br/>        “行了，我估计子昂会回西城，我等一会买晚上的机票回西城！”
