　　苏钦章不是傻子，顿时就明白过来，转头望着这帮宗师泰斗们说道：“原来今日之事，是法术公会主持的。”
<br/>　　听他这么说，一直没有开口的释信无说道：“没有的事，老衲等人是前往龙虎山参加龙华会，却是先来茅山，与苏天师你知会一声，共商大计，与轩辕氏却是没有关系。”
<br/>　　苏钦章立刻拱手说道：“大师你的话我信，但你们刚来，这二位转瞬即到，跟大师您肯定没关系，但跟别人……”
<br/>　　说着眼睛朝玉玑子飘过去，玉玑子避开了他的目光，装作没有听见，上前招呼两位轩辕氏人。这让苏钦章坐实了自己的猜测：这两个轩辕氏人的到来，必然跟玉玑子有关。
<br/>　　不过既然来了，自己作为代理掌教，礼数还是要有的，苏钦章上前拱手行礼，说了几句客气话。
<br/>　　两人也站住，那个少年没开口，也没还礼，依然保持着高傲的姿态，前面那个中年男子微微拱手，自我介绍，他是星月奴座下弟子，名叫神道清，后面这个是他的小师弟，名叫悦然。
<br/>　　在场的人，虽然大部分都没见过他们，不过一听他们的自我介绍，立刻就明白了。张无生挑了挑眉毛，望着那少年说道：“你就是星月奴收的七位关门弟子之一？”
<br/>　　悦然弯起嘴角笑了笑。
<br/>　　“这位是龙虎山掌教张天师，不得无礼！”神道清瞪了师弟一眼说道。
<br/>　　悦然这才不情愿地施了一礼，也不说话。不过张无生倒也没太在意，他已经确定了这个悦然的身份：
<br/>　　法术公会是一个神秘的组织，虽然最近一直在渗透人间法术界，不过真正摆在名面上的人很少，神道清算是一个。在人间法术界所了解的情况来说，法术公会这个稍微现代的词，是人间的叫法，公会的老大、也就是轩辕氏的族长，向来十分神秘，人间多数人只听过他的名字：龙。
<br/>　　说是名或者姓都可以，反正就这么一个字，人间都称其为龙先生，据说他是上古轩辕氏的血脉传承的族长，手下有五大弟子，有四个都很神秘，只有星月奴和清长风两人，实际掌控着法术公会，星月奴据说已经证道，人间至少在几百年前就有她的传说了，当然所知道的也不过寥寥数人，张无生也是听道渊真人说的。
<br/>　　最近，法术公会在人间的行动越来越多，星月奴也被摆上台面，被更多人所知，据说为了掌控人间法术界，早在十几年前，她就在人间悄悄选了一批幼儿，培养了十几年，如今个个都已经成材，实力之强，冠绝人间法术界的二代弟子。
<br/>　　因此，张无生听神道清介绍悦然是星月奴的弟子，看他的年纪，立刻给出了猜测。
<br/>　　其余几位宗师自然也听过这些消息，得知面前这个少年就是星月奴的关门弟子之一，一个个立刻对他投去了好奇的目光，在他身上扫视起来。
<br/>　　悦然微微昂着脸，用骄傲的姿态迎接众人的目光。
<br/>　　“不知道两位大师今日来我茅山，有什么事？”苏钦章不冷不热地说道。来者不善，他的态度也不会太好，并没有请他们进殿的意思。
<br/>　　神道清笑了笑，“我也是听到消息，诸位宗师今日齐聚茅山，因此前来拜会……听说诸位要在龙虎山办一场龙华会，我们法术公会，也想出一份力，还请张天师和诸位大师应允。”
<br/>　　“人间的事，还是由人间法术界自己了解得好，不劳法术公会费心。”张无生皮笑肉不笑。
<br/>　　苏钦章听他们这一来一去的对话，顿时明白之前是误会张无生了：本来以为这次张无生发起龙华会，背后是有法术公会主持，如今才知道不是，不过，也有可能是张无生跟他们串通好了，为了避免非议，当着大家的面故意这么说，不过这种可能性极小，对张无生的人品，苏钦章还是相信的。
<br/>　　悦然听张无生说完，插了一句道：“听说这次龙华会，是为了在年轻弟子中选出一位首席弟子，好带领法术界对抗天劫，我们师兄弟也是来自人间，正好为法术界出力。”
<br/>　　张无生鼻子里哼了一声道：“不见故乡，只见闻道。你既然拜入法术公会，那便不是人间的弟子。我人间之事，自有人族决断。”
<br/>　　张无生一句话将悦然所要说的话堵死。
<br/>　　悦然笑了一声，没有搭腔。但神色中带着几分轻蔑。
<br/>　　神道清道：“这次龙华会，我们法术公会也是全力支持各大门派，我觉得光选出首席弟子还不够，一个人的力量毕竟有限，我们应该选出一批优秀弟子，引领法术界抵抗天劫……还请诸位不要有门户之见。”
<br/>　　悦然往前走了一步，看着苏钦章，说道：“久闻茅山是道门翘楚，苏道长身为代理掌教，一定也有过人之处了？”
<br/>　　这番话有点锋芒毕露，苏钦章听出来，有些不快地说道：“不敢，我实力低微，只是替掌门师兄暂且管理山门而已。”
<br/>　　悦然道：“苏道长不要谦虚，我在轩辕山学艺十多年，如今回到人间，很想跟各位宗师请教一番，尤其是苏道长这样的同龄人……”
<br/>　　苏钦章这才明白了他的意思，眼睛猛地一睁，说道：“你是想跟我斗法？今日这种时候，怕是不妥当吧？”
<br/>　　悦然道：“捡日不如撞日，我也没别的意思，就是想见识一下茅山的手段而已。”
<br/>　　苏钦章冷哼一声，心如明镜，说什么请教和见识，就是想当着这帮宗师的面，打败自己，好在人间立威，为以后法术公会的行动做好铺垫。
<br/>　　苏钦章那个恼火，当下刚要开口，悦然突然欺身而上，双手合十，打出了一个手印，朝着自己撞来。
<br/>　　“苏道长，得罪了！”
<br/>　　苏钦章一惊之间，大手印已然来到自己面前，只好结印反击，将这大手印顶住，身体却是晃了一晃，这一手……好强的力道。
<br/>　　自从叶少阳下山之后，苏钦章就服侍在青云子左右。
<br/>(本章完)
