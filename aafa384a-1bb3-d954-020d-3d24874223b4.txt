        童筱颖听得惊怔，她万万没想到，那次她故意去找何明旭的茬，想要拿到何明旭杀害马智杰的正确，却没料到反被何明旭给偷偷录了音。
<br/>
<br/>        幸好，听到这段录音的人是大哥傅夜沉而不是昼景，不然，她掉进黄河也无法跟昼景解释清楚了。
<br/>
<br/>        大哥傅夜沉误会了没关系，要紧的是昼景幸好不知道这件事情。
<br/>
<br/>        童筱颖心里顿时松了口气。
<br/>
<br/>        傅夜沉紧紧地搂着童筱颖，非常淡定地听完这几段录音后，咧嘴一笑：“我还真的是得多谢你的‘劈腿’之恩，才能让沫儿成为我的未婚妻。”
<br/>
<br/>        “看样子，傅大少爷你是鬼迷心窍了！这女人可是冲着你的钱去的！”何明旭狠狠地瞪了童筱颖一眼。
<br/>
<br/>        傅夜沉突然放开了童筱颖，摩掌擦拳，捏得手指咯咯作响，看着何明旭咧嘴一笑：“替我问候你主治医生。”
<br/>
<br/>        话音一落，傅夜沉一个箭步上去干脆利落就是一拳，当场把何明旭打翻在地，之后又是痛快淋漓地几脚，要不是林若琴拼命拦着，估计何明旭真的就被傅夜沉给废了。
<br/>
<br/>        童筱颖就站在一旁，木讷地看着何明旭被打，默不吭声。
<br/>
<br/>        “大少爷，求你，别打了！”林若琴张开双臂，拦在了傅夜沉的面前，将何明旭护在了身后，看着傅夜沉苦苦哀求道。
<br/>
<br/>        傅夜沉伸手直接挪开林若琴的身子，微微屈身，一把揪住何明旭的衣襟，将他从地上提了起来：“小子，回去好好照照镜子。”
<br/>
<br/>        “呵！你只不过是穿了我的破鞋而已，嘚瑟什么？”何明旭微眯起眼睛，看着傅夜沉冷哼。
<br/>
<br/>        两人面对面，彼此的鼻尖几乎都快要碰到了一起。
<br/>
<br/>        傅夜沉勾唇一笑：“我的沫儿，身心都很干净。你连给她提鞋都不配！”
<br/>
<br/>        何明旭身心一怔，忽然间意识到了什么，感觉胸口突然空了一下，像是有股什么东西提到嗓子眼。
<br/>
<br/>        上次，童筱颖其实是想算计他，若不是林若琴找关系罩着他，不然，他一定会被当成“嫖客”抓进警察局。
<br/>
<br/>        “大少爷，今天阿旭他心情不好，说话难听，你别往心里去。”林若琴再次上前，拽住何明旭，唯唯诺诺地对傅夜沉说道。
<br/>
<br/>        傅夜沉放开了何明旭，瞥了一眼林若琴，淡淡地说道：“你两好自为之。”
<br/>
<br/>        他说完，回到童筱颖的身边，牵住童筱颖的手，与他俩擦肩而过。
<br/>
<br/>        林若琴和何明旭面面相觑后，也终于明白马智杰的那件事，警局里的人为什么不抓童筱颖了。
<br/>
<br/>        他们找错了替死鬼。
<br/>
<br/>        其实，那次童筱颖的出现是个意外。
<br/>
<br/>        何明旭也是临时想到这点，才诱骗童筱颖去第四教学楼的天台，然后与林若琴里应外合。
<br/>
<br/>        他以为他们做得天衣无缝，却没料到，童筱颖的后台比他们还要硬。
<br/>
<br/>        “他以前不是这样的人。”童筱颖任由傅夜沉牵着，黯然神伤地说道。
<br/>
<br/>        以前的何明旭，压根就不是现在这个样子。
<br/>
<br/>        到底是什么东西，让他变成了现在这个样子？
<br/>
<br/>        童筱颖心里揪得发痛，闷得发慌。
<br/>
<br/>        为什么？
<br/>
<br/>        为什么曾经她认为是她身边最亲的人，都变呢？
<br/>
<br/>        苏漫雪是！何明旭也是！
<br/>
<br/>        傅夜沉走着走着，发现童筱颖不吭声，手也很凉，不由自主地停住脚步。
<br/>
<br/>        他侧过身去看着她，她聋拉着脑袋，双眸湮没在刘海的阴影里，看不透她的喜怒哀乐。
<br/>
<br/>        童筱颖将自己的手从傅夜沉的手心里抽了出来，她仍旧低着头，声音低沉地说道：“既然大哥没醉，那就早点儿回家吧！我也要回宿舍去了。”
<br/>
<br/>        “筱颖……”傅夜沉欲言又止。
<br/>
<br/>        童筱颖从他面前，一步步地渐行渐远。
<br/>
<br/>        他知道她心情不好，所以他一路尾随在她身后，直到她安全抵达宿舍楼下。
<br/>
<br/>        童筱颖刚回宿舍，三个舍友便开始跟她八卦傅夜沉的事情。
<br/>
<br/>        以傅夜沉那俊颜，放他们学校里，那绝对是首席校草级别的人物。
<br/>
<br/>        三个舍友对他感兴趣，也算是人之常情。
<br/>
<br/>        但当这三位“花痴”舍友，从她这里打听到傅夜沉已有未婚妻而且还同居了的消息后，傅夜沉瞬间就被她们三打入了“冷宫”。
<br/>
<br/>        相比起当时在傅氏集团里上班的那群“花痴”女同事们来说，她的这三个舍友还是比较放得开点。
<br/>
<br/>        至少，她们三不会对“有妇之夫”死缠烂打。
