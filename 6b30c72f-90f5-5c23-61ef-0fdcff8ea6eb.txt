　　由此可见，之前经过的那个地方，应该是这座山里之前的模样，被建墓的人利用了，成为古墓的一部分，再挪来通灵佛子花，前来镇守。
<br/>　　好大的手笔……
<br/>　　不光如此，对叶少阳来说，他下过很多大墓：当初建文帝的墓，白起的墓等等，虽然也都是凶险无比，但是从来没有见过这么古怪的守墓手段，通灵佛子花，那种如同真实一般的幻象，给叶少阳的内心留下了深深的震撼。
<br/>　　“你之前在幻境中，经历了什么？”叶少阳不知不觉中跟梧桐走在了最后面，于是跟她搭话。
<br/>　　梧桐摇摇头，“很恐怖，我不想再提。”
<br/>　　说完加快速度从她身边走过。
<br/>　　一道长长的墓道，两边墙壁上不再用那种可以发光的石头，但是镶嵌了灯台，里面都存着一些鲛油，一路上每遇到一个，大伙就把它点亮，因此路上并不黑暗。
<br/>　　突然，本来笔直向前的墓道上出现了一个岔路，并不很深，借着身后的鲛油灯，能看到尽头处有一扇石门。
<br/>　　大伙互相看了看，出于好奇，走上了这条岔路。
<br/>　　跟之前的石门一样，还是有一个锁孔，应该是用钥匙可以打开的。
<br/>　　“我很纳闷，这里明明是古墓，为什么要弄成可以开的门锁？”叶少阳挠着后脑勺说道。
<br/>　　“什么意思？”妙心问。
<br/>　　“你想啊，古墓是埋人的地方，最不希望的就是让人进去，所以如果有石门之类的，直接封死就好，为什么要留下门锁、机关这样的东西，难道修墓的人希望让后代有机会还可以进来？这钥匙又掌握在什么人的手中呢？”
<br/>　　妙心等人沉吟不语。显然，这也是让他们感到费解的一个问题。
<br/>　　妙心点头说道：“我也没有下过这样的古墓，头一次遇到。”
<br/>　　“就拿面前这扇门来说，你说你这里面有什么，如果有人找到钥匙，进去会怎么样？”叶少阳说完，自己走过去，上下检查这扇门。
<br/>　　因为是在山体内部，十分潮湿，岩石上都渗着水，湿漉漉的。
<br/>　　叶少阳注意到门锁附近的位置，有几块岩石有剥落的痕迹，伸手抠了一下，发现一整块岩石都是松动的，心中一动，用勾魂索前端的探爪挂上去，用力拉扯，竟然将一大块岩石拽了下来，立刻让毛小方举起鲛油灯照过去，能看到一根粗大的石条，横在岩石的夹缝中。
<br/>　　这就是石门的锁芯，边上有打磨出来的槽子，可以想见，这锁芯本来是插在岩石中间的，结果现在岩石剥落，锁芯也暴露出来。
<br/>　　也就是说，门锁的功能已经失去了。
<br/>　　叶少阳有点兴奋，再度用勾魂索挂在石条锁芯上，招呼毛小方一起用力往外拉，然后……竟然真的将石门给拉开了。
<br/>　　“运气真好！”毛小方笑着说，眼中闪烁着兴奋。
<br/>　　妙心走上前来检查了一下门锁附近的石槽，皱眉道：“为什么别的地方都没事，只有这里的岩石松动了？”
<br/>　　岩石松动，这个其实好解释，毕竟是几百年前的古墓，受到地下水的腐蚀，这些镶嵌在山体里的砖石，出现问题也是正常，但妙心的好奇也是有道理的。
<br/>　　“大概是因为这里塞着锁芯，中间开槽的缘故，中间是空心的，更容易被腐蚀。”毛小方提出自己的解释。
<br/>　　妙心点点头，也没有再说什么。
<br/>　　大伙一起上前，将石门完全拉开。里头一边漆黑，妙心担心里面是封闭的，没有空气，让大伙等了一会，另外用罗盘测了一下，在里面没有发现任何邪气，这才手拿鲛油灯，陆续走了进去。
<br/>　　两盏鲛油灯，将里面的空间照亮。
<br/>　　这是一个不算很大的石室，里面什么都没有，只有对面靠墙的地方，摆着一只只的坛子。在这些坛子的后面，是一片被打磨光滑的石墙，墙上刻着很多字。
<br/>　　总算是见到文字了，大伙都很好奇，立刻走上去观察，石墙已经开裂，上面满是渗透出来的地下水，长满了霉菌，很多地方字迹都模糊了。叶少阳找了几张黄裱纸，上去把这些霉菌都擦掉，仔细看去，这些字迹都是繁体字，依稀可辨，于是从头读了起来：
<br/>　　“余家负皇命，寻此龙脉，埋葬先皇后人，并以余下家资，深埋此地，以期势起之后，用做起兵，吾之后人，须辅佐主公，驱除鞑虏，光复江山社稷，不得有误，若辜负皇恩，将此财用做他处，必将暴毙而亡，不得善终……魂魄亦万劫不复，永世不得超生……”
<br/>　　石壁上的文字，就是这些，没有落款的姓名和时间。
<br/>　　众人看完之后，恍然大悟。叶少阳望着妙心说道：“看来你祖先说的没错，这古墓真的是宋朝遗臣建的。”
<br/>　　“财宝，这上面提到财宝，哪里呢？”陈晓宇两眼放光。
<br/>　　众人都把目光落在那些坛子上，走了过去，见坛子上面都有牛皮纸封口，不过经过这么多年，就算是当时最结实的牛皮纸，也已经完全被腐蚀了，坛子上被刻着一些符号，叶少阳认不出，梧桐看了一眼说道：“这是佛门密宗的封印。”
<br/>　　叶少阳疑惑之下，撕开一个坛子上的牛皮纸，用鲛油灯照进去，一片黄澄澄的，心下一惊，伸手摸到一块沉甸甸的，拿起来一看，是一块金元宝……
<br/>　　我勒个去，这是黄金啊！
<br/>　　大伙一起把所有坛子都打开，一共九个坛子，八个都是装的满满的金元宝，最后一个坛子里是一些珠宝之类的，看上去也是璀璨夺目，绝对是好东西。
<br/>　　大伙面面相觑。
<br/>　　十坛黄金……
<br/>　　“对个人来说，这绝对是一笔巨富的财产了，十辈子也花不完。”妙心喃喃道，“不过对于起兵来说，也只是少量的帮助而已，看来这已经是修墓的人全部身家了。”
<br/>　　“已经不少了！”卢晓清深吸一口气，转头分别看了看众人，说道：“现在怎么办？”
<br/>　　这句话问的听上去没头没脑，但是大家都明白意思。
<br/>(本章完)
