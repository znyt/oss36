　　从一尊碎命之境的强者手中逃脱活下来。
<br/>　　现在让楚风眠来面对碎命之境的强者，他也有着自信可以活下来，不过这还是因为楚风眠的身躯，练就了远古龙王身。
<br/>　　他的身躯，本来就极为坚韧，在加上楚风眠血脉的恢复能力，只要不将他瞬间轰杀，楚风眠都可以活下来，生命力远超一般武者，他才有着如此自信。
<br/>　　但是风无心不同，风无心是真正以实力，足以跟碎命之境的强者抗衡，才逃出生天的。
<br/>　　从这一点上来看，这风无心，还要更加可怕。
<br/>　　尤其是风无心，甚至都不是什么特殊血脉，也不是先天神体，他的强大，全凭借的都是他的实力。
<br/>　　如今楚风眠也想要见见这北域百绝榜的第一位，看看这风无心，到底是一尊什么人物。
<br/>　　就在楚风眠跟寒月漓还在思考之际，突然几道身影，便是走了过来，这几道身影，便是从刚刚北邙学院聚集的哪一方，一起过来的。
<br/>　　一共五个人，三男两女，皆是北邙学院的核心弟子。
<br/>　　其中在生死台周围，看到过楚风眠出手的两名核心弟子，古傲寒，卢清平，也在这行列之中。
<br/>　　不过这一群人中，为首的人却是一名女子。
<br/>　　这女子冰肌玉骨，身材完美，脸上始终是带有着几分淡然笑容。
<br/>　　只是楚风眠清楚，这女子，显然是没有看起来的这般和善，她在北邙圣域之中，可是有着一个称呼，血罗刹燕碧。
<br/>　　相传她的一身冰肌玉骨，都是以武者的鲜血，来沐浴而来的。
<br/>　　“各位前来，有何贵干？”
<br/>　　楚风眠看到这五人前来，平静的开口道。
<br/>　　这些人前来，显然不可能只是为了看看楚风眠，肯定是有着自己的目的。
<br/>　　这五个人的实力并不弱，都是北邙学院的核心弟子，全部都是北域百绝榜上的人物。
<br/>　　虽然对楚风眠来说，他并不惧怕这些人，甚至交手，楚风眠有信心足以取胜。
<br/>　　但是现在对方人多势众，更是在武王城中，在这武斗大会前，一切的私斗都是没有任何意义的，所以楚风眠才会如此谨慎开口。
<br/>　　“也没什么，只是想要邀请二位，放弃这次武斗大会。”
<br/>　　那女子还未开口，旁边的古傲寒却是冷冷的开口道。
<br/>　　“在武斗大会上，我们便是很有可能会互相碰到，会白白削弱学院的力量。”
<br/>　　“你们两个人，都是新人，这一次武斗大会，你们还是放弃吧，等到下一届的时候，在前来参加，倒时候我们也会给你们让路，行个方便。”
<br/>　　这古傲寒开口，虽然看起来是像劝解，但是听起来，其实便是已经跟命令，没有任何的区别，一脸狂傲，高高在上的样子。
<br/>　　楚风眠跟寒月漓的实力，他们清楚，不过眼下，他们五个人联手，以五对二，他们可是丝毫不惧。
<br/>　　他们一齐联手，就算是碎命之境的强者，也不是没有办法抗衡，对付楚风眠，当然是轻轻松松。
<br/>　　另一尊核心弟子，卢清平开口道。
<br/>　　“还请二位，考虑一下吧，为了学院着想，也还请两位将这次的名额交出来，然后放弃这一届的武斗大会。”
<br/>　　“为了学院着想？退出？交出名额？”
<br/>　　楚风眠听到这几人的话，脸色也是阴冷了下来。
<br/>　　这果然是，来者不善。
<br/>　　看起来这些核心弟子，便是如今联合起来，先让北邙学院的一些强者退出，这样一来，他们踏入前十的可能性，便是会大得多。
<br/>　　楚风眠跟寒月漓，这两个刚刚崛起的新星，在他们的眼中，便是绝对的心腹大患。
<br/>　　所以如今这些人，才前来跟楚风眠来谈判，想要让楚风眠交出来名额。
<br/>　　不过。
<br/>　　别说是五个核心弟子，就算是让五个太上长老来，想要逼迫楚风眠，也是痴心妄想。
<br/>　　“做梦。”
<br/>　　楚风眠口中，淡然吐出两个字。
<br/>　　听到楚风眠的话，那五名核心弟子的脸色，都是变得阴冷了许多。
<br/>　　似乎是没有料到，楚风眠居然会如此强硬，都不敷衍，就在他们五人的面前，如此的斩钉截铁，不留任何一点情面，
<br/>　　其中那古傲寒，更是毫不客气的开口道。
<br/>　　“小子，放肆！我们来好心劝你，你却如此不识抬举！”
<br/>　　“你招惹到我们，这一次的武斗大会，你也别想要杀入前十。”
<br/>　　“是吗？你们算是什么东西，也赶来逼迫我？”
<br/>　　楚风眠脸色，瞬间变得冰冷许多。
<br/>　　“都给我滚！”
<br/>　　楚风眠这一次开口，言语之中，已经是动用了许多灵力。
<br/>　　这一个滚字，顿时便是震得那五人，连连后退，一个个的眼神中，都是充满着战意。
<br/>　　气氛凝重，几乎战斗，在瞬息之间便是足以爆发。
<br/>　　“等等！”
<br/>　　就在这时，那为首女子，血罗刹燕碧，突然开口了。
<br/>　　“两位，莫非是真的不打算给我们这个面子？放弃这一届武斗大会，下一届我们保证你们足以进入前十。”
<br/>　　“不需要！”
<br/>　　楚风眠冰冷的开口道。
<br/>　　他楚风眠，还没有沦落到需要别人施舍的地步。
<br/>　　更何况，放弃这届武斗大会，还需要在等待三年。
<br/>　　三年时间，对武者来说，的确不长，开出这个条件，也许许多武者，都会答应，但是这并不代表楚风眠。
<br/>　　三年时间，楚风眠可是已经耽误不起了，楚风眠的身份，很有可能随时都会被七大宗门的知道。
<br/>　　那个时候，楚风眠需要面对的，便是无穷无尽的追杀。
<br/>　　他必须要在这次之前，提升自己的实力，达到足以跟七大宗门对抗的地步。
<br/>　　所以三年时间，楚风眠等不起，而且他也不需要等。
<br/>　　楚风眠脸色冰冷，一字一字的说道。
<br/>　　“再不走，你们的下场，就跟龙未央一样！”
<br/>　　“狂妄！”
<br/>　　古傲寒猛然怒声开口道。
<br/>　　跟龙未央的下场一样，那不就是代表着让他们陨落。
<br/>　　他们可是亲眼看见，楚风眠是如何在生死台上，轰杀龙未央的。
<br/>　　“住手！”
<br/>　　燕碧大手一挥，直接将古傲寒制止住。
<br/>　　她瞥了一眼楚风眠，平静的开口道。
<br/>　　“我们走！”
