　　单单是这一股威势，就不亚于一尊八劫巅峰古帝。
<br/>　　仅仅是打开了这一本古籍，居然就有着如此恐怖的威势。
<br/>　　而且这还是这一本古籍，经历了岁月的消磨，上面的力量已经是损失大半了。
<br/>　　这一头雄狮，仿佛是镇压在这了古籍之内妖兽一般，随着古籍的打开，解开封印，冲天而起，想要逃出生天。
<br/>　　“想逃？”
<br/>　　楚风眠看的出来，这雄狮应该是这古籍之中的一部分。
<br/>　　只是不知道怎么，凝聚成形，相要逃出生天。
<br/>　　这一头雄狮，现在还是气势所化，可如果在给他几万年的时间，也许真的可以成就一尊真正的雄狮，化为生灵。
<br/>　　楚风眠冷喝一声，掌心凝聚战龙血脉力量，直接施展出始祖第一式来，一掌轰击在这雄狮上面，
<br/>　　这一头雄狮，承受了这一掌的力量，被凌空打破，四分五裂，逐渐的分化为了四个大字，落入到了古籍之中。
<br/>　　这实力不亚于八劫古帝巅峰的雄狮，居然是四个字所化。
<br/>　　楚风眠眼神之中都流露出了无比惊讶的神情。
<br/>　　他本以为这一头雄狮，应该是封印在这古籍之中的什么妖兽魂魄一样的存在。
<br/>　　可没想到，居然是四个字所化。
<br/>　　文字成形，在古籍之中并非是没有记载，据说是曾经有着一尊强者，被称为画圣。
<br/>　　此人以比作画，笔墨未干，其中画中的图像，就会浮现，凝聚身形。
<br/>　　一些古画，也有画卷成妖的记载。
<br/>　　没想到这古籍之中的文字，也可以凝聚成形，刚刚跑出来的那一尊雄狮，已经是快要接近成妖的地步了。
<br/>　　楚风眠惊讶之余，目光也向着古籍看了过去。
<br/>　　这四个字凝聚的雄狮被楚风眠打散，便是失去了力量，落入到了古籍之中，顿时显现了出来。
<br/>　　狮霸无疆四个大字。
<br/>　　在这古籍的第一页，浮现出来。
<br/>　　这四个字，是九域之中无比古老的文字，距今至少是数十万年的时间。
<br/>　　幸好是楚风眠熟读了九域古籍，才了解这种文字。
<br/>　　“狮霸无疆……这里难道是霸狮武祖的洞府？”
<br/>　　楚风眠脑海之中回忆着，思考这四个字，突然脑海之中显现出了一个名字。
<br/>　　霸狮武祖。
<br/>　　曾经九域妖神之中的一位传奇，这霸狮武祖，据说曾经只是一尊寻常的妖兽狮祖。
<br/>　　后来以惊天领悟，创造出狮祖武道，堪称武祖。
<br/>　　武祖这个称呼，本是唯有武道大成的武者，才有资格被成为武祖的。
<br/>　　而妖神，本不属于此列，但是霸狮武祖创造而出的武道，登峰造极，甚至是超越了无数武道至尊。
<br/>　　所以他称武祖，无人敢反驳，也是妖神之中，唯一一位武祖。
<br/>　　“狮霸无疆，这其中记载的，果然是霸狮武祖的武道！”
<br/>　　楚风眠翻开这这一本古籍，其中记载的种种，跟楚风眠猜想的没有错，是霸狮武祖的武道。
<br/>　　霸狮武祖，虽然是一尊妖神，却没有族群，独自一人，后来他的武道，虽然传承下来的一部分，但是并不完整。
<br/>　　以至于后来越来越没落。
<br/>　　没想到在这洞府内，楚风眠居然是找到了霸狮武祖完整的武道。
<br/>　　这一本古籍，如果被一尊武者得到，绝对是足以一飞冲天，纵然是一个小人物，也足以成就一方绝代强者。
<br/>　　曾经的霸狮武祖，可也是一尊天人，这一本古籍之中记载了霸狮武祖一生的武道领悟。
<br/>　　让一位九劫古帝得到，足以是让一位九劫古帝的实力，提升一倍都有可能。
<br/>　　现在大罗天星域，四大势力之中记载武道的无数武技之中，只怕都没有任何一本，可以跟眼前这一本狮霸无疆所媲美。
<br/>　　狮霸无疆，这四个字，也代表着当时霸狮武祖，在那个时代之中的所向无敌。
<br/>　　只是……
<br/>　　楚风眠叹息了一声。
<br/>　　这一本古籍，对他却是没有任何的用处，楚风眠要修行的，不是任何一人的武道。
<br/>　　甚至是剑道之主的剑道，楚风眠都没有想要修行，他要做的，是开辟属于自己的剑道。
<br/>　　超越天下一切的剑道。
<br/>　　这狮霸无疆，对楚风眠来说，却是意义不大，因为楚风眠不可能抛弃剑道，去学习霸狮武祖的武道。
<br/>　　不过天下万道，皆属同源，这一本狮霸无疆记载了霸狮武祖的武道，也记载了曾经霸狮武祖无敌的气势。
<br/>　　只要是楚风眠细细参悟，也足以领悟其中几分意境，也足以融入到楚风眠的剑术之中。
<br/>　　甚至是足以依次，创造出新的剑术来，就像是楚风眠以九玄仙术，创造出九玄剑术一样。
<br/>　　不过这想要参悟，不是一朝一夕的事情。
<br/>　　楚风眠将这一本狮霸无疆，打上了几道禁制，封锁了上面的力量之后，便是身形闪烁，向着独桥深处再度飞去。
<br/>　　知晓这洞府的来历，楚风眠脸上便是更加的喜悦，这霸狮武祖，可也是曾经称霸九域的人物。
<br/>　　而且比起一方势力，这霸狮武祖所留下的宝藏，可能更多。
<br/>　　毕竟像是云羽天宫，还有天元商会，其中遗迹之中的宝物虽然不少，可比起他们的辉煌，却是要少得多。
<br/>　　因为一方势力，会培养无数弟子，收敛的许多圣药，圣丹，都会分发下去，而不会一直藏起来。
<br/>　　虽然宝物众多，但是也消耗无数，所以遗留下来的，并不算多。
<br/>　　但是这九玄仙尊洞府，还有眼前这霸狮武祖的洞府，却是不同。
<br/>　　他们都是独自一人的存在，没有势力，所收敛的所有宝藏，在他们陨落之后，都会遗留在洞府之中。
<br/>　　这霸狮武祖洞府之中的宝藏，只怕是比起一个势力遗迹之中的宝藏，还要多。
<br/>　　建木树枝，紫星石，还有着古籍狮霸无疆，都只是其中的冰山一角。
<br/>　　真正的宝藏还遗留在其中。
<br/>　　楚风眠身形连连闪烁，不过在到了五千里之后，楚风眠的遁光，变的更慢了下来。
<br/>　　现在楚风眠身上承受的压力，几乎是让八劫巅峰古帝，都动弹不得，就连上前一步，都难以做到。
<br/>　　就算是九劫古帝，在这样的压力之下，都无法催动遁光，只能靠脚行走。
